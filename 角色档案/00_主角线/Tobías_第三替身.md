# Tobías · "El Hijo Olvidado"

> 别的替身在装。我是。
> ——他对镜子里的自己说过的唯一一句话。

> ⚠️ **首周目玩家不应看到任何关于此角色的暗示。**
> 写到他时，所有可见文本要确保 (a) 不出现"私生子"线索；(b) 不出现"Báez 还有别的孩子"暗示；(c) 不在主目录任何角色卡里反向引用他。

## 二、声音卡

### 与 Saulito 的根本差异

Saulito：他想活着撑过去。
Tobías：**他想做一些 Saulito 做不到的事**——撕掉法令、看着每个人脸色变化、在内阁会议上故意让某个细节走漏。

### 公开场合（"装父亲"的版本）

```
体裁：他对真总统的演说语言**比 Saulito 学得快**——他从小听着关于父亲的传言长大。
节奏：比真总统更**阴**——他能模仿到表面，但**他的眼神里有恨**。
词汇：他用得起政治术语，但他用得**像一个没受过完整教育的乡下青年突然学会说官话**——
       这种生硬其实是他独有的"被框死了的演员风格"。
危险：他**会故意露出破绽**——这是 Saulito 永远不会做的。
```

### 内心独白（**第三人称、过去式、克制**）

```
人称：他 / 他的（**第三人称**——这是他与 Saulito 最大的视角差异）
节奏：长句多过短句。**像是有人在替他写传记**——
       他从小听着关于父亲的传言长大，他思考自己的方式
       已经被那些故事的语法格式化了。
词汇：克制、文学化。允许使用乡村俚语作为"突然回到自己"的破口。
情绪锚点：母亲、舅舅家的木桌、Veranillo 的某条河、某次在电视上看见父亲的瞬间。
```

**示范片段**：
> 他签下那份名字时手没有抖。这让他自己感到失望。
> 他从小幻想过这一刻——他以为自己会在签的瞬间想起母亲。
> 他没有。他想起的是舅舅家厨房里的那块灶台。

### 标志性动作 / 习惯

- **签字前不读稿**——这是他与真总统最致命的反差。**他读不下去**——一旦看到父亲签过的同一种字段，他会停 2 秒。**这是他的破绽，也是他的存在感**。
- 不戴 Seiko——他的手腕没习惯过表的重量。**Susana 给他戴上又取下三次，他始终戴不正**。
- 在内阁会议中**会偶尔露一个微笑**——一个 Báez 从不做的微笑。**这是他在故意试探**。

### 不会说的话

- 任何"我父亲" — 他**只能说"上一任总统"**。提到"父亲"二字会触发立即崩溃。
- 任何对母亲的公开提及——这是他**唯一禁区**。寡头若在此试探，他会**砸杯子**（Cordura −大）。

## 三、与替身位置的关系

### 戏剧张力的来源
- 他是**唯一一个真的能"代行"真总统的人**——血缘合法性最高。
- 他也是**最危险的**——他**对父亲有真实的恨**，他在某些场合会**故意做一些 Saulito 做不到的事**。
- 这是**反英雄式的觉悟弧线**——但他的"觉悟"不是为了人民，是为了**报复一个抽象的父亲**。
- **本作政治学最尖锐的视角**就在他这条线：当一个被遗弃的儿子坐到那个位置上，他做的事是政治还是私愤？两者能分清吗？

### 专属机制
- **"擅自动作"系统**：事件中会出现 Saulito 没有的"反向选项"——例如"撕掉一份父亲签过的法令"、"在内阁会议上当面问 Castaño 一个旧政变细节"。每次擅自动作大幅推进剧情，但显著拉爆 *Verosimilitud* 与 Cordura。
- **DNA 卡**：他不怕任何家族 DNA 测试——这是他相对 Saulito 与 Catalina 的独特优势。但他怕的是**人格层面的小细节**（左手疤、表的握法）。

## 四、重大事件

### 解锁场景（多周目某个偏门支线终点）
- 一份家族档案 / 一段 Sofía Reinoso 寄来的研究 / 一次乡村巡视——指向**Veranillo 的一户人家**。
- Vega 第一次知道他存在。
- **Susana 早就知道**——她当年抱过他母亲怀里的一个婴儿（这条线参 `99_秘密角色/_阅读警告.md`，Susana 知情人面更深一层）。

### Acto I（"我从一开始就是他"）
- 第一次内阁会议——他比 Saulito 更稳。Castaño 反而更早怀疑（"今天的总统怎么这么安静"）。
- 第一次签字——他**故意停顿了一秒**。Susana 在旁边屏住呼吸。

### Acto II（与 Mariana 的镜像）
- Mariana 在国宴上对他说："您和您父亲，气质很不一样。"
- 他不答。**他笑了一下**。这是 Mariana 第一次**对替身真心害怕**的瞬间——她见过太多"演父亲"的人，但**没见过"恨父亲的人正在演他"**。

### Acto III（Aurora 的特殊反应）
- 在山区——Aurora 这次**没看穿他**。**她看穿了别的**：她看见了**这个男孩身上有真总统的味道，但味道里多了一层她不熟悉的东西**。
- Aurora 给他煮汤时手在抖。**她可能知道他是私生子**。

### Acto IV（联姻强制 vs. 报复欲）
- 寡头提"娶 Mariana"。Tobías 路线**完全不同**——他可能**接受**，原因不是钱，是"看着 Doña Isabel 把女儿送给 Báez 的儿子，却不知道是哪个儿子"。**这是本作最冷的笑话之一**。

### Acto V（北方使馆"找到他的母亲档案"）
- Whitfield 茶会递的不是地址纸条——是**他母亲的死亡医院记录**。
- "我们替您保管了。"——这一句话在他这条线里的杀伤力 ≠ Saulito 路线。

### Acto VI（撕法令）
- 一份父亲 1995 年签过的法令到他桌上需要续期。**他撕了**。
- 这一动作触发整个游戏中**最大的一次派系连锁反应**——但也触发 *El Cuestionario Final* 的一个特殊分支。

### 终章
- 他可能会在某个场景里**主动公开自己的身份**。这是本作最罕见的"Patrón 自爆"路径——绝大多数玩家直到第三周目才会想到这一选项。

## 五、写作红线

1. **首周目锁死**：在主目录任何文档中**不得**暗示"Báez 还有别的孩子"。包括但不限于：医院档案、寡头家族族谱、北方使馆电报、Sofía Reinoso 研究目录——**绝不出现"私生子"或"乡下" + "Veranillo" + "21 岁"的组合**。
2. **第三人称内心独白**：他的内心叙事**严格用第三人称过去式**——这是与 Saulito（第二人称现在式）/ Catalina（第一人称广播稿）的核心区分。
3. **"父亲"是禁区**：他不能在公开场合说"我父亲"——他**只能说"上一任总统"**。这一规则连续违反 3 次会触发立即剧情崩塌（设计中）。
4. **不要写他成为英雄**：他**不是**。他擅自动作是为了报复，不是为了正义。
5. **他的解锁不属于第一周目**：所有他的事件、对白、UI 元素**首周目玩家不应该看到任何线索**——这是本作叙事节奏的关键。
6. **Aurora 看不穿他**这件事**只在他的路线下成立**——其他周目里 Aurora 看得穿。**这条要严格控制写作**。
7. **他的母亲不出场**——她已经死了。**任何"母亲幽灵般出现"的魔幻现实主义场景都要克制**（参 `06_主题与调性` 红线 2）。
