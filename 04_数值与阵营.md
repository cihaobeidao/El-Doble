# 04 · 数值与阵营系统

> 四层数值结构。每层为同一个戏剧问题服务——你这个不该在的人，能在这个房间里待多久？
> 派系角色与小传详见 01_世界观与派系小传.md。

---

## 一、Tier 1 · 国家指标（公共，玩家可见）

| 指标 | 初始值（首周目） | 含义 | 跌破阈值 |
|---|---|---|---|
| **Economía**（经济） | −30 | 通胀、就业、外汇储备 | < −60 → 触发"恶性通胀总崩溃"事件链 |
| **Orden**（秩序） | −15 | 治安、暴力、罢工 | < −60 → 触发"全国紧急状态" |
| **Legitimidad**（合法性） | 0 | 国际承认 + 国内宪政认知 | < −50 → 触发"使馆撤回支持"或"宪法法庭弹劾" |
| **Tesoro**（政治资本） | 数字资源（不是 −100~100） | 玩家发动行动消耗它（**与时辰独立**） | 跌至 0 → 几乎无法主动发起政策 |

> **重要：Tesoro 不等同于时辰**。Tesoro 是**政治资本**（发动一项议案、收买一个媒体、换得一次紧急会议），消耗它需要"花政治影响力"；时辰是**时间预算**，决定你今天能见几个人、做几件事。两者独立但相互制约——你可以有一上午的空闲时间但没有政治资本去推动议案。

四个国家指标的初始值由上一周目"知识传承"机制调整（详见 05_多周目机制.md）。

## 二、Tier 2 · 派系关系（玩家可见，态度文字标签）

| 派系 | UI 显示 | 含义 |
|---|---|---|
| **军方（La Junta）** | 铜星徽章 | 老派 / 青年派分别独立计分，UI 合并 |
| **十二姓氏（Los Doce）** | 白手套徽章 | 寡头集团对你的态度 |
| **圣灵集团（El Cártel）** | 白十字徽章 | 贩毒集团是否把你视作"可谈生意的人" |
| **两个大国（El Águila y el Toro）** | 鹰徽章 + 公牛徽章（**两个并列**） | 4a 北方使馆 / 4b 南方使馆 双子轴 |
| **街头与教会（La Calle y la Iglesia）** | 蜡烛徽章 | 民意 + 媒体 + 教会权威合并值 |

**显示语言**（替代精确数值）：
> 极度敌对 / 敌对 / 警惕 / 中立 / 试探合作 / 真实合作 / 信任 / 同盟

> **设计原因**：拉美政治不是数字，是态度。让玩家感受**关系的氛围**而非分数。

### 两个大国的特殊机制

- **互相挤压**：一方上升 ≥ 25，另一方触发"被冷落"事件链（数值轻微下降 + 派出关键 NPC 试探你）。
- **中间派甜蜜区**：双方维持在"试探合作"上下时，每周给玩家 +1 Tesoro 隐性补贴。
- **极端化锁死**：任一子轴 ≥ 90 或 ≤ −60 永久锁死另一子轴（被动倒退，再也回不来）。

### 原住民群体（不计入数值轴）

*Los Pueblos del Toro* 不进入派系数值系统，但有专属隐性标记（详见 §四）。详见 01 文档"次要群体"小节。

## 三、Tier 3 · 替身个人状态（玩家可见，关键资源）

| 指标 | 范围 | 含义 |
|---|---|---|
| **Verosimilitud**（伪装度） | 0–100 | 你"看起来像总统"的可信度。决定大多数事件的难度 |
| **Cordura**（神智 / 精神） | 0–100 | 精神状态。睡眠、酒精、孤独、恐惧都会影响。**0 触发发疯事件 *La Locura***（详见 05 文档） |
| **Sospecha**（怀疑度） | 0–100（每个派系独立） | 该派系当前怀疑你不是真总统的程度。**任何一个 ≥ 80 立刻触发对应揭穿事件链** |

### 派系起始 Sospecha

各派系对替身的"起始怀疑度"差异极大——这反映他们各自的情报能力 + 与真总统的实际亲密度：

| 派系 | 起始 *Sospecha* | 缘由 |
|---|---|---|
| **北方使馆** | 70 | 他们对真总统医疗记录比谁都熟。从一开始就知道，问题不是"被识破"，而是"他们什么时候决定行动" |
| **南方使馆** | 30 | 情报水平不如北方，但亲昵风格易让你失言 |
| **十二姓氏** | 35 | Mariana 已识破但未上报母亲（详见 01 文档 Mariana 卡） |
| **军方青年派** | 0 | Vega 全知 |
| **军方老派** | 50 | 历史经验 + 怀疑一切 |
| **圣灵集团** | 20 | 他们更在乎你的生意态度，不太在乎你是谁 |
| **街头与教会** | 5 | 最不可能识破——但一旦识破，反扑最猛 |

### Verosimilitud × Sospecha 的关系

- *Verosimilitud* 高，可以**被动减缓**所有派系怀疑度的增长。
- 单次事件的 *Verosimilitud* 检定通常需要：基础 50 + 派系 *Sospecha* 修正 + 事件特殊难度。
- 一些选项要求 *Verosimilitud* ≥ 阈值才能选（例如"亲赴现场" ≥ 65、"对 Mariposa 说真总统的话" ≥ 80）。

## 四、Tier 4 · 隐性标记

非数值标记，类似 RPG 的 quest flag。

**标记的作用**：
1. **解锁灰色"档案选项"**——多周目玩家在事件中看到金色斜体选项，需要曾经获得过该信息（具体机制详见 05_多周目机制.md）。
2. **触发后续戏剧性回调**——某事件文本会因"上周或本周累积过的标记"完全改变（例：`MET_LUCIA_AT_PROTEST` 会改变后续 Lucía 出现的语气）。
3. **影响结局分支判定**——多个标记的特定组合是某些结局的进入条件。

**全部标记的集中索引**：详见 08_剧情/01_关键标记索引.md。

**标记的玩家可见性**：
- 隐性标记**默认对玩家不可见**——玩家不会在数值面板上看到一份"我做过的事"列表。
- 这是设计意图：玩家应该**记得自己做过什么**，而不是依赖系统帮他记。
- 例外：周目结束的总结页，会**集中展示**该周目获得的关键标记 + 错过的关键标记——这是知识传承的回看时刻。

## 五、数值之间的因果链（设计原则）

**核心因果**：

```
玩家选择 ──→ 时辰 / Tesoro 消耗（直接代价）
            ↓
        国家指标变化 ───→ 派系态度变化
            ↓                ↓
        Verosimilitud / Cordura ←── Sospecha 变化
            ↓
        触发新事件 / 锁死旧事件 ──→ 隐性标记积累
            ↓
        结局判定时调用所有数值与标记
```

**写作时常用的因果模板**：
- "强势的政策"（如军队镇压）：Orden ↑ / Calle ↓ / Junta-老派 ↑ / Cordura ↓
- "退让的政策"（如延后法案）：Tesoro ↓ / 该派系态度 ↑ / Legitimidad ↓
- "人性化的选择"（如对 Mariposa 弯腰）：Calle ↑ / Verosimilitud ↓ / Cordura ↑ / Sospecha-Doce ↑
- "操纵媒体"：Calle ↑（短期） / Cordura ↓ / Calle ↓（长期反弹）

**禁忌**：
- **不要让任何一个选项"全好"或"全坏"**——本作的数值哲学是**任何选择都至少有一项明显代价**。
- **不要让 *Verosimilitud* 单独决定一切**——它影响很多检定，但不应是"加点流"的核心目标。
- **不要让玩家通过刷数值绕开戏剧抉择**——这是"自动保存 + 单存档"机制存在的原因（详见 05）。
