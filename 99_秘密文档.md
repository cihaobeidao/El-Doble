## 一、*La Verdad Compuesta*（合成真相）

> **请慎重对待这一节**。任何对话、文档、甚至随口的"剧透警告"都可能让某些秘密被过早泄露。

### 1.1 一句话版

> **Augusto Báez Marín 这个人已经死了 28 年。Saulito 是第三任替身。维拉杜里亚联邦共和国从 1998 年起，由一系列模仿者治理——而真正的"主谋"是第一夫人 Gabriela 与化妆师 Susana。**

### 1.3 与 A/B/C/D 卷宗的对应关系

| 卷宗 | 表面叙事 | 真相 |
|---|---|---|
| **A · 在 El Norte 治病** | 真总统在北方某医院 | 第二任替身在 2024 年确实去过北方做检查——但这只是**幌子**，让人误以为他还活着 |
| **B · 在 Vasconia 友军基地** | 真总统在邻国军营 | 第一任替身的尸体（2010 年自然死亡后）被秘密迁葬到 Vasconia 边境一处友军基地——这是 Vargas 总统帮过 Gabriela 的一个忙 |
| **C · 已死** | Vega 日记里的"真总统已死" | **半真**——Mateo Vega 的日记记录的是 2010 年第一任替身的死，他**误以为**那才是 Báez 本人。他至今不知道 1998 年的事 |
| **D · 国内软禁** | 修道院水电单 + 反常购药 | **完全真**——但被软禁的是第二任替身，不是 Báez 本人 |

> **设计意图**：A/B/C/D 都是真的，但每一条都对应**不同的"Báez"**。玩家集齐 4 张后必须自己拼图——意识到"原来不只有一个 Báez"。

### 1.4 Saulito 与"演员血统"

- *Las Veladas* 这个区从 1990 年代起就是政治演员的孵化器——殖民时期那里曾有一座小剧院，关闭后演员散到街头。
- 第一任替身 Eladio Tobar 是这一脉的"前辈"。
- Saulito 自小被母亲送去街头模仿艺人那里学手艺——**Gabriela 与 Susana 多年来一直在这条街上"看着"他长大**。
- 这就是为什么 Vega 那天凌晨能精确地在那条街拦下他——**他们已经盯了十年**。

### 1.5 多周目终局结构

- **首周目**：玩家走 A/B/C/D 中的一张。结局千差万别，但**绝大多数玩家不会发现真相**。
- **二到四周目**：玩家解锁不同卷宗。每一卷被拼齐时，会有一个"震惊回放"——例如玩家走 D 线索拼齐"国内软禁"时，会看到一段 cutscene 揭示**那个修道院里的人不是 Báez 本人**。
- **第五周目（或集齐 4 张后）**：档案室中出现 *La Verdad Compuesta* 的占位符。玩家进入新周目后，事件中开始出现**金色高亮的"骨头线索"**——例如 Susana 看你的眼神有了一段新的内心独白注解（"她在你脸上看见了她从前抱过的婴儿"）。
- **最终结局触发条件**：玩家在某次特定事件中**主动说出**真相的某个关键片段（例如对 Susana 说"你认识我父亲，对吗？"）—— 这触发 *El Cuestionario Final*（最后的盘问），逐步把所有秘密铺开。
- **最终结局名为 "*Las Veladas*"**——以 Saulito 长大的街区命名。这是本作真正的"True Ending"。结局视觉是 Saulito 站在他童年的那条街上，黎明前的那种灰光——他是替身、也是儿子、也是国家的一个旁注。

### 1.6 关于这条真相的"道德重量"

> 这条真相之所以重要，不是因为它"震惊"，而是因为它说明了：**这个国家从 1998 年起就一直在被假象统治，而所有派系都参与了维持假象**。
>
> 所有玩家在第五周目意识到这一点的瞬间，应当感到的不是"哇好爽"，而是**冷**——因为这意味着：在他们以为自己在做选择的所有事件里，他们其实一直在帮一台早已运转了 28 年的机器再运转一天。
>
> **这是马尔克斯式的真相**——不是反转，是确认。这个国家从一开始就是被空椅子统治的。

## 二、真实人物防呆名单

> 写作时**绝不**让以下类别的真实人物姓名出现在游戏文本中。这不是"避免争议"，是**避免廉价指认**——一旦读者对上号，他们就停止思考普世性，开始对号入座。

### 2.1 完全禁止类

- **任何拉美在世国家元首 / 在任内阁高官**
- **任何 1973 年以来的拉美军事独裁者**（Pinochet, Videla, Stroessner, Banzer, Rios Montt, Trujillo, Somoza, etc.——一个都不要）
- **任何当代或近三十年的真实大毒枭**（Escobar, Guzmán, etc.）
- **任何当代或近三十年的真实游击领袖**（Marulanda, etc.）
- **任何被强制失踪 / 被暗杀的真实记者 / 工运领袖 / 学生领袖**——他们的尊严不属于我们
- **任何在世或近期去世的美国总统 / 国务卿 / 国安顾问 / 中情局局长**
- **任何拉美主要寡头家族真实姓氏**（Slim, Pinto, Edwards, Matte, Echeverría——别用 Echeverría 单独的姓氏 ✗ 我们用复合姓 Ríos-Echeverría 是边线，可接受但需复合）
- **任何当代或近期教皇**

### 2.2 禁止 + 替换原则

- **历史伟人（独立战争时期及之前）可以出现**，但仅作为**壁画 / 课本 / 街头雕像 / 饮料品牌名**——不发表观点。允许的例：Bolívar、San Martín、Sucre、Hidalgo、Martí。
- **历史人物可以借用其类型而非姓名**——例如"我们的 Castaño 家族军政府是 X 类型"可以，但不能让 Castaño 家族里有人叫"Augusto"或"Rafael"等明显对应。
- **真实战争 / 政变 / 经济危机事件**可以**借用结构**，不能借用细节（详见 06 文档红线 3）。
  - 可以：1973 年某次政变（数字与全球范围的若干事件相符是巧合）
  - 不行：把某次具体危机的死亡数字、文件名、地名直接照搬

### 2.3 名字检查清单

写作时若出现新角色，请按以下清单检查：

- [ ] 这个名字是否与任何真实在世政治人物的姓 / 名重合？
- [ ] 这个名字组合（姓 + 名）是否在 Google / 维基百科上对应到具体真实人物？
- [ ] 这个职位 + 这个名字 + 这个国家的组合，是否会让任何拉美读者自然联想到具体真人？

如果任何一项 yes，**改名**。

### 2.4 灰区（需个案讨论）

- **天主教教阶**（红衣主教、主教、神父）：可以给虚构名字，但要避免与近几任教廷高层重名。
- **足球球星 / 文化名人**：可以**作为背景音**提及（"广播里又在放 Maradona 的纪录片"——这种是允许的，因为是已经过去的、文化共识层面的）。
- **国际机构**：IMF / World Bank / OAS 等可以提名，但不要让具体在职官员出场。

### 2.5 自检的最高规则

> **写完一个角色后，问自己一个问题：如果一位与本作描写的国家最相像的拉美国家的公民读到这一段，他/她会觉得"这是我们的故事"，还是"这是在影射我们的某个具体的人"？**
> 
> 前者：通过。
> 后者：改写。

## 三、红蓝对照表 · 主版

> 这是 06 文档"红线 4 对称原则"的执行工具。本节定义"什么是左 / 什么是右"，并提供一份首位填写示例。

### 3.1 立场谱系定义（本作内部）

#### "左"端立场（在维拉杜里亚语境内）

- 资源国有化（特别是铜矿）
- 土地改革 / 重新分配
- 工人组织权 / 工会权力
- 原住民自治
- 反帝（特别是反北方共和国干预）
- 福利扩张 / 公共服务
- 限制军队 / 军费缩减
- 文化上的反殖民、反保守、反教会权威

#### "右"端立场（在维拉杜里亚语境内）

- 私有化 / 市场化
- 财产权神圣
- 法律与秩序 / 强警察
- 传统家庭 / 反堕胎 / 反离婚
- 与北方使馆深度合作
- 减税 / 财政紧缩
- 强军 / 军队权威
- 文化上的天主教保守、欧洲精英主义、反"红色"

#### "中间派 / 现实主义"立场

- IMF 路径下的有限改革
- 技术官僚治理
- 与两大国都搞好关系
- 渐进民主化
- 反腐（不分派系）
- 不站在任何一端的"专业治国"幻想

> **重要**：这不是真实政治学的精确定义——这是**本作内部的工作定义**。任何具体事件可能有多重立场维度（一个事件可以同时是"反帝（左）"和"传统家族保护（右）"）。**这正是我们要刺穿的复杂性**。
