# Doña Filomena · Saulito 的母亲

## 一、身份信息

| | |
|---|---|
| 全名 | Filomena Aguilar de Tobar |
| 称呼 | Doña Filomena / "妈" |
| 年龄 | 74 |
| 出身 | 中部省份 Veranillo 乡下，与真总统 Báez 同省份（**这是她在 99 真相中的位置坐标**——首周目玩家不应注意） |
| 现住地 | Veranillo 一座两房一厅的老房子，腿脚不好（关节炎）。 |
| 主要收入 | Saulito 每月寄 250 北元（黑市汇兑） |
| 婚史 | 与 Eladio Tobar 短暂同居（1979 年前后），Saulito 出生后不久 Eladio "消失"。她**至今以为他抛弃了她**。她从未再婚。 |
| 信仰 | 天主教，但更多是民间信仰（家里供圣母像 + 一只占卜瓷盘） |
| 标志物件 | 一条绣着 *S.T.* 字母（Saúl Tobar）的旧棉帕——她出门时必带 |
| 健康问题 | 双膝关节炎严重（一次更换需 ~30,000 cobre ≈ 800 北元）；高血压 |

## 二、声音卡

```
体裁：乡下口音，词汇朴素。她**从不用都市新词**。
节奏：慢。她讲话时常停顿——不是因为思考，是因为呼吸。
词汇：街市俚语 + 民间宗教用语。
       *Mijo*（"我的儿"——对 Saulito 的标准称呼，比 Saúl 还常用）
       *Ay, Virgen Santa*（"啊，圣母啊"——感叹时）
       *Eso no es de Dios*（"那不是上帝的事"——她否定某事时）
口头禅：
  - "*Mijo, ¿comiste?*"（"我的儿，你吃了吗？"——她每次见 Saulito 第一句）
  - "*Tu padre, que en paz descanse...*"（"你父亲，愿他安息……"
     —— **她以为他死了**。这是 99 秘密的伪装层。**写她时这句话要保留**——
     这是首周目玩家"Saulito 没有父亲"的认知锚）
```

### 标志性动作
- 见到儿子时**先摸他耳朵**（民间检查"你是不是真的"——这是个老人式的玩笑）。
- 听人讲新闻时**用棉帕擦眼镜**——擦的不是污渍，是紧张。

### 不会说的话
- 任何政治意见——她**不关心**。她对总统这个人没有特别的爱憎。
- 直接抱怨腿——她**只在儿子追问时说"还行"**。

## 三、与替身位置的关系

- **她不在 Saulito 路线主战场**——她住在 Veranillo 乡下，是远端的"私人重力"。
- **首次直接出场**最早于 Acto III 末或 Acto V——视玩家是否在 Whitfield 茶会"地址纸条"事件后选择"派人去看母亲"。
- **她不会上京**：Saulito 在 La Casa Cobre 时她不会主动来——她不想给儿子添麻烦。
- **她是"威胁权重"载体**：从序幕 Vega 复述她门牌起，她的存在就是 Saulito 不能逃跑的理由之一。

## 四、重大事件

### 序幕
- 不在场。**Vega 复述她门牌、Saulito 寄汇款金额** —— 这是序幕"威胁不需言明"的实体之一。

### Acto I 中段
- Saulito 第一次"私人时段"重看档案时，会写一封假装是 Augusto 风格的家书——但他偷偷在边角画了一只蝴蝶（这是给母亲的暗号，意为"我安全")。**这封信永远没寄出**——Susana 拦下了。

### Acto III 末（"地址纸条"事件后）
- Whitfield 茶会递上她家门牌地址。Saulito 选择：
  - **(a) 立即派 Vega 派人保护母亲** → 触发标记 `MOTHER_GUARD_REQUESTED`，但同时寡头 *Sospecha* +5。
  - **(b) 自己请假回乡探望** → 触发隐性事件链 *El Viaje a Veranillo*（仅 1 个时辰窗口可用，需周全规划）。
  - **(c) 什么都不做** → Cordura −15，同时标记 `MOTHER_LEFT_ALONE`（极后期周目影响）。

### Acto V（如选 b 路径解锁）
- *El Viaje a Veranillo* 真正的 Saulito 私人危机。她**没认出他**。
- 她见到他时第一句话："Mijo，你怎么穿成这个样子？" —— 这是一句把所有政治权力一秒打回原形的话。
- 她**摸了他的耳朵**。她**摸出他不是 Saulito**——她摸到了化妆师贴的耳后假皮。
- 她什么也没说。**她转身去煮汤**。
- 离开时她塞给他棉帕。"Mijo, *que la Virgen te cuide*."（"我的儿，愿圣母看顾你。"）

### Acto VI 与终章
- 她**不知道她儿子在做什么**——这是她从未改变的事。
- 终章里她在 Veranillo 的厨房里打开收音机，听到一段她**听不懂但感到熟悉**的官方公告。
- 这是 Saulito 终章的**最重感情锚**——但**绝不被旁白点破**。

## 五、写作红线

1. **她不是工具**：**禁止**把她写成"母亲死了 / 受伤"作为剧情推动力的素材。她是一个**真实的人**，她的存在意义不在于死。
2. **她不知道 Eladio 的真相**：她以为他抛弃了她。**写她提到丈夫时永远停留在这一层认知**——这是首周目"Saulito 没有父亲"的伪装层。
3. **她的乡下口音要真**：写她对话时**避免书面语**。允许语法不规范。
4. **她见到 Saulito 时摸耳朵**这个动作是**只属于她的细节**。**禁止**让其他角色重复这个动作。
5. **她不评价政治**——任何让她对总统、寡头、北方使馆发表意见的台词都写错了。
6. **腿不要写成苦情戏**——她有真实的痛，但她**不抱怨**。这是她底色的尊严。
7. **棉帕**这个物件是终章的伏笔——**不要在 Acto III 之前让玩家见到**。
