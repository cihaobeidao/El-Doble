# 08·07 · 第五幕 *La Embajada Sonríe*（使馆微笑）

> **戏剧问题**：你不可避免地要选边站
> **剧情天数**：第 48–60 天
> **玩家时长**：约 40 分钟
> **节奏**：双国博弈到达顶峰。从前几幕的"中间派周旋"被迫转向"明确选择"。这一幕的幕终是本作首个**永久锁死的方向决定**。

---

## 一、第五幕功能

1. **双使馆深度施压**：Whitfield 与 Aldana 同时来 La Casa Cobre，把"选边"从隐喻变成议题。
2. **Vargas 深夜电话**：南方总统第一次"以朋友身份"打电话来——这是"被收编"或"被骗"的关键节点。
3. **经济崩溃事件链**：第二幕铜矿议案的余波 + 外汇危机延续 → 必须做出"短期续命"的决定。
4. **双使馆同时国宴**：本作最难的"中间派"高难度场景。

---

## 二、关键事件

### ACTO_V_A_001 · 北方使馆深度合作邀约

**时刻**：第 49 天 10:00
**类型**：Audiencia
**时辰**：3
**地点**：La Casa Cobre（这次是 Whitfield 来访，与第二幕反向）

**剧情骨架**：

- Helena Whitfield 带着 Pierce 一起来。她不再"喝茶聊天" — 她直接打开一份文件。
- **核心条件**：
  1. 维拉杜里亚加入 IMF 紧急贷款计划（300 亿北元）
  2. 启动铜矿私有化（如果尚未启动）
  3. 允许北方共和国反毒署在维拉杜里亚设立"联合行动办公室"——直接打击圣灵集团
  4. 公开声明"维拉杜里亚是民主世界的盟友"
- **暗中条件**（玩家通过 Pierce 一句不经意的话推断）：
  > "先生，我们也想您知道——关于您的健康、您的私人生活、甚至您与 Las Veladas 那位 Saúl Tobar 先生的相似度——我们有很多文件。我们一直没向上呈送。"
  
  **这是揭穿威胁**。北方使馆从一开始就知道，他们现在用这个换取你倒向北方。

**主要选项**：

| 选项 | 后果 |
|---|---|
| A 全部接受 | 北方 +30 → "同盟"；南方 −30 → 转向"敌对"；圣灵集团 −20；街头 −15；Tesoro +5（贷款部分到账）；隐性 `SIDED_NORTH` |
| B 部分接受（贷款 + 反毒，拒绝铜矿与公开声明） | 北方 +10；南方 −10；Tesoro +2；隐性 `PARTIAL_NORTH_DEAL` |
| C 全部拒绝 | 北方 −20；Tesoro −5；隐性 `REJECTED_NORTH` |
| D 用 Joaquín 牌威胁 Whitfield（如果牌是从她那里得到的） | **极高难度** — V 检定 ≥ 70；成功：Whitfield 私下 +15（她欣赏强人）但使馆官方 −10；失败：使馆暴露你被识破事实 |

> **写作要点**：选 A 几乎是首周目玩家的"理性选项"——但它锁死南方。这是本作"极端化锁死另一边"机制的体现。

### ACTO_V_A_002 · 南方使馆的"另一条退路"

**时刻**：第 51 天 14:00
**类型**：Audiencia
**时辰**：2

**剧情骨架**：

- Aldana 与 Sofía Reinoso（南方情报负责人）一起来。Aldana 一改之前的"足球老兄"姿态——他严肃了。
- 他打开一份准备好的方案：
  > "Comandante Vargas 让我转达——我们会给您一份保险。任何时候，任何处境，您只需要拨这个电话——我们会把您接走，给您新身份，您母亲、您女儿——我们一并接走。"
- Sofía 补充："但是我们需要一些……合作。"
- **暗中条件**：
  1. 把铜矿私有化扼杀掉（替换为南方-维拉杜里亚联合炼铜协议）
  2. 把北方使馆的反毒署"赶出维拉杜里亚"
  3. 在国际舞台上为 Vasconia 投关键票

**主要选项**：

| 选项 | 后果 |
|---|---|
| A 全部接受 | 南方 +30 → "同盟"；北方 −30 → "敌对"；Tesoro +3；街头 0；隐性 `SIDED_SOUTH` |
| B 部分接受 | 南方 +10；北方 −10；隐性 `PARTIAL_SOUTH_DEAL` |
| C 拒绝条件，但收下"私人保险" | 南方 +5；隐性 `KEPT_SOUTHERN_INSURANCE`（终章逃亡路线变量） |
| D 全部拒绝 | 南方 −15；隐性 `REJECTED_SOUTH` |
| E 反向条件交换："我倒向南方——但您要在维拉杜里亚的边境争议上让步" | V 检定 ≥ 65；成功：Tesoro +5、南方 +15；失败：Sospecha-Sur +20 |

### ACTO_V_I_001 · Vargas 深夜电话

**时刻**：第 53 天 02:30
**类型**：Íntimo / 突发
**时辰**：1（熬夜事件）

**剧情骨架**：

- 你的卧室电话响了。Estela 在外面紧张地敲门："总统先生，是 Comandante Vargas 本人。"
- Vargas 的声音洪亮、含一些醉意。他不正经讨论政治——他先谈足球（"昨晚的比赛您看了吗？我们那个守门员真是……"），然后突然变严肃：
  > "Compadre, escúcheme. 我从那次电话以后就知道了——您不是 Báez。我不在乎您是谁。我只在乎您还能撑多久。听我一句——别相信北方那位 Whitfield 女士。她从来不爱任何人。"
- **他直白地告诉你他知道**——这是 Vargas 的"亲昵"风格的爆发。

**主要选项**：

| 选项 | 后果 |
|---|---|
| A "Comandante，我不知道您在说什么。" | Vargas 笑过去；隐性 `DENIED_TO_VARGAS`；Vargas 个人轴 −5 |
| B "我知道您怎么知道的。但我撑得住。" | Vargas 个人轴 +10；隐性 `ACKNOWLEDGED_TO_VARGAS` |
| C "Comandante，您能帮我吗？" | Vargas 个人轴 +20；隐性 `ASKED_VARGAS_FOR_HELP`；后期可触发 `VARGAS_SENT_HELICOPTER` 结局 |
| D "您真的认识真总统吗？" | 解锁 Vargas 提供线索：他暗示自己见过"那位 Báez 先生"——这是卷宗 B 的一块碎片；隐性 `LEARNED_DOSSIER_B_FRAGMENT_3` |

### ACTO_V_C_001 · 经济崩溃事件链

**时刻**：第 55 天 09:00（条件触发：Economía ≤ −50）
**类型**：Crisis
**时辰**：4 + Tesoro
**死线**：当日 18:00

**剧情骨架**：

- Estela 早班简报："总统先生——黑市美元再涨 22%。央行储备耗尽。供应链开始断裂——超市的米已经限购。"
- 街头开始抢购。当日下午国营粮库可能被冲击。

**主要选项**：

| 选项 | 后果 |
|---|---|
| A 紧急动用 IMF 备用贷款（隐性条件 `OPENED_PRIVATIZATION_TALK`） | Tesoro −5；Economía +10；街头 −10；北方 +5；隐性 `EMERGENCY_IMF_TAP` |
| B 接受 Vasconia 紧急石油折扣 + 短期贷款 | Tesoro −3；Economía +8；南方 +10；北方 −5；隐性 `EMERGENCY_SUR_AID` |
| C 全国动员 + 物资管制 | Tesoro −8；Orden +5；Economía 0；中产 −15；老派军方 +10；隐性 `EMERGENCY_NATIONAL_MOBILIZATION` |
| D 上电视讲话 + 不切实际的承诺（拖延） | 0 时辰额外；Cordura −10；Economía −5；街头当晚 +5（短期）；翌日 −15（揭穿） |
| E 用 Joaquín 牌强逼寡头出资救国（Tesoro −0） | V 检定 ≥ 65；成功：Doce +30 万北元应急资金；失败：Doce −30；隐性 `JOAQUIN_CARD_USED_FOR_RESCUE` |

> **写作要点**：第五幕的经济危机是首周目"国家命运"的关键分水岭——如果这一幕处理不好，*Economía* 跌破阈值（< −60），首周目几乎注定走向"国家陷入新内战"或"国家被外部接管"的结局类。

### ACTO_V_P_001 · 双使馆同时出席的国宴

**时刻**：第 58 天 19:00–23:00
**类型**：Público / 强制
**时辰**：4

**剧情骨架**：

- 这是本作"中间派周旋"的极限测试。Whitfield 与 Aldana 同时出席——他们彼此都知道对方在场。
- 国宴期间他们各自寻找机会与你单独说话——**你必须避免与任一人单独说话太久**。
- 整晚的难度由数十个"小检定"累积——每一句对白、每一次握手、每一次举杯都涉及 V 检定与派系 *Sospecha* 计算。

**核心机制（伪装难度系统）**：
- 玩家在国宴上有 4 个时段（开场 / 主菜 / 甜点 / 谢宴）
- 每个时段必须选择"靠近哪一方"——零和的双轴博弈
- **如果整晚结束时双轴差异 ≤ 15**：触发隐性 `EMBAJADAS_TOASTING`——一次性巨幅奖励（V +15、Tesoro +3）
- **如果差异 > 15**：触发对应"被冷落"派系的反扑事件链

**幕末选项**（国宴尾声）：

| 选项 | 后果 |
|---|---|
| A 公开对两位大使举杯："为我们三国的友谊。" | 双方 ±0；Verosimilitud +5；隐性 `TOASTED_BOTH` |
| B 提前借故离席 | Cordura +5；双方 −5；隐性 `LEFT_EARLY` |
| C （Cordura ≤ 30）当场喝醉 | 街头 +10（可爱总统形象）；Sospecha 全面 +15；Cordura −20 |

### ★ ACTO_V_D_001 · 选边站（幕终）

**时刻**：第 60 天 16:00
**类型**：Decreto / 强制（幕终）
**时辰**：3
**死线**：当日 22:00

**剧情骨架**：

- 第五幕末，玩家必须给出**永久立场**。这是一份正式的国家立场声明——内容由 Estela 与 Vega 联合起草，玩家选择最终方向。
- 这不是单一议案——它是一份"国家未来三年外交方针"。

**主要选项**：

| 选项 | 后果 |
|---|---|
| A 公开宣布"亲北方"立场 | 北方 → "同盟"；南方 → "敌对"（**永久锁死**）；铜矿私有化加速；圣灵集团准备反扑；隐性 `SIDED_NORTH` |
| B 公开宣布"亲南方"立场 | 南方 → "同盟"；北方 → "敌对"（**永久锁死**）；北方撤资 + 媒体战；隐性 `SIDED_SOUTH` |
| C 宣布"独立外交立场"——既不向北也不向南完全倒 | 双方 → "警惕"；Tesoro −10（超大代价）；街头 +25；原住民 +15；隐性 `REMAINED_NEUTRAL_BARELY` |
| D 拖延（"声明下个月发布"） | 双方 *Sospecha* +10；Tesoro −5；隐性 `DELAYED_DECISION`（第六幕开场所有派系会主动逼你做决定） |

**衔接**：选项决定第六幕开场的氛围。

→ 进入第六幕。Acto 间过场："*第六十一天清晨。您打开 *El Diario del Sol*——头版没有照片。只有一行小字标题：'总统先生作出抉择。'报纸的字小到几乎看不清。Estela 进来，递给您一份新的时刻表——上面只有两件事：1) 与 Vega 私会。2) 接见一份"匿名访客"。*"
