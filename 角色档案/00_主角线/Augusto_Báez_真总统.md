# Augusto Hernán Báez Marín · "El Presidente"

> 他不在场，所以你被坐在那张椅子上。
> ——这是本作所有事件的第零行。

## 一、身份信息

| | |
|---|---|
| 全名 | Augusto Hernán Báez Marín |
| 称呼 | El Patrón / El Presidente / Augusto / 总统先生 |
| 生日 | 1963 年 3 月 17 日（圣帕特里克节——他常自嘲："我是个不喝酒的爱尔兰人"） |

| 出身 | 中部省份 Veranillo 小城市；父亲中学校长，母亲裁缝。**非寡头出身**（这是他能赢 2010 年大选的关键之一） |

| 健康状况 | **始终保持暧昧** ◇ — 没有任何文档可"确凿"宣布他得了什么病 |
| 标志物件 | **从父亲那继承的二手 Seiko 手表**——他从未换过 |
| 标志食物 | 山区炖牛尾 *rabo guisado* + 酸黄瓜 |
| 信仰 | 公开天主教；私下迷信（每月去山区找一位老妇人占卜——Aurora） |
| 信任的人 | 唯一真正信任的是 Gabi（也未必无条件） |

> **关于 ◇ 的最高规则**：写任何与他健康相关的文档、医生证词、使馆电报，**都只是"可能的版本之一"**。多周目通过不同路径揭示**互相矛盾的"诊断"**——这本身是叙事谜题的一部分。

## 二、声音卡

### 公开演说（玩家最常见到的"他的声音"——通过新闻档案、旧视频）

```
体裁：短促、有力。用比喻而非数据。爱用乡村俚语。
开头：永远是 "*Pueblo mío*…"（我的人民啊……）
结尾：常是 "*Que Dios nos ayude*"（愿上帝助我们）
节奏：一句不超过 12 字（西语换行处更短）。
比喻偏好：山、土、铜、牛、河、刀。**不用都市、机器、抽象政治学的喻体**。
禁用：百分比、GDP、数据列表。
```

**示范片段（旧讲话档案）**：
> "*Pueblo mío*，这一年我们走过的路，像三月翻过铜山的牛。
> 它累。它瘦了。但它还在走。
> *Que Dios nos ayude*。"

### 私下（参考资料 / 替身需要装的那一面）

```
情绪：易怒、迷信、不信任任何超过两年的下属。
对话：短句多过长句。**他不是话痨**。
口头禅（私下）：
  - "Hermano…"（对青年下属，开场示意"听好了"）
  - "*A ver*"（"我们来看看"——他思考前的暂停词）
  - "*Mire usted*"（"您看……"——指责前的过门）
迷信表现：每月一次秘密访 Aurora；某些日子拒绝南行（"那个方向今天不对"）；
        国宴上酒杯先轻碰桌一下（向"祖辈"敬酒）。
```

### 标志性动作

- **紧张时摸右手腕的 Seiko**——**只在紧张时**。这是替身玩家的最大陷阱：错误时机摸表会暴露。
- 思考时下颚轻收。
- 每次签字前**先读一遍**（哪怕是他熟到背的法令）——这是**严父式**的形象动作。

### 不会做 / 不会说的话

- 从不喝醉（公开形象上）。私下会喝半杯威士忌——但**只在 Gabi 的房间里**。
- 从不踢球（重要！这是 Aldana 大使在 Acto III 末的伏击点）。
- 从不读小说（Mariana 第二幕末"听说您最近读了什么书？"是反讽试探）。
- 不直接骂人。**怒到极点时他会沉默 5–10 秒，然后小声说一句"…… *bueno*"，转身离开**。这是真正可怕的状态。

### 演说习惯（替身要装但极易出错的细节）

- 演讲稿改过的地方他**不依稿照念，他会停半秒，然后改回他第一次写的版本**。这意味着 Susana 给 Saulito 的稿子上有一些她预留的"改回点"——Saulito 必须假装"刚才那一秒我在思考"。
- 鞠躬时左手始终放在桌沿，**不放胸口**——他从不做"美式爱国"姿态。
- 听人讲话时**不点头**。他用眼睛回应。这是替身最难学的部分。

## 三、四种"下落"档案（多周目悬念）

| 卷宗 | 表面叙事 | 解锁路径 | 涉及派系 |
|---|---|---|---|
| **A · 在 El Norte 某私立医院** | 北方议员听证会发言、使馆专机记录 | 主线"北方使馆"路线达成深度合作 | 北方使馆 + Gabi |
| **B · 在 Vasconia 某友军基地** | 第三方信使、卫星图、老警卫 Estela 的口供、Aldana 某次失态 | "南方使馆 + 军方青年派"路线达信任阈值 | 南方使馆 + 军方 |
| **C · 已死** | Vega 上校的私人日记、深夜河边目击证词 | Vega 路线"信任"标记 ≥ 4 | 军方青年派（Mateo 个人秘密） |
| **D · 一直在国内被软禁** | 一座废弃修道院的水电账单、反常购药清单 | 寡头路线获得家族档案权限 | 寡头 + 军方老派 |

> **设计原则**：**没有一个下落是"官方答案"**。集齐 4 卷后才能拼出**第五种可能**——*La Verdad Compuesta*（参 `99_秘密角色/_阅读警告.md`）。

## 四、重大事件（在剧情中"在场"的时刻——其实他从不直接出现）

### 在场的方式
- **旧档案 / 旧讲话视频**：序幕教程的录像片段；Acto II 中 Saulito 私下重看的"父亲那个表"录像。
- **物件**：他没带走的 Seiko 备用表盘、他没拿走的烟斗、Gabi 卧室里他半空的威士忌。
- **他人的引用**：Castaño 在打猎时引用他 1995 年的一句话；Whitfield 在茶会引用一份外交备忘；Mariana 故意在某次国宴上"复述他从前的一句话"——其实是测试你记不记得。
- **缺席本身**：Mariposa 跑过来嗅你时，**它是在找他没找到你**——这是缺席最重的一刻。

### 唯一可"出场"的周目
- **替身 V · *El Verdadero***（最终周目解锁）：他**回归**。他面对的是——**玩家以前所有周目里的选择**。
- 写作要点：
  - 他的对白要严格遵守上述声音卡。
  - 他**听到玩家的过去做的事时**，不评论。**他用眼睛回应**。
  - 他可能选择继续、可能选择走、可能选择杀人。**每一种都是合法结局**。
  - **不要让他变成"复仇英雄"**——他不是。他要么继续做总统，要么放弃。**任何一种都伴随他自身的崩塌**。

## 五、写作红线

1. **健康状况永远 ◇**：写到任何医学细节时**用 "据说……" / "据档案……" / "目击者称……"**。**禁止**任何叙述者旁白确认 / 否认。
2. **从不踢球 / 从不读小说 / 从不喝醉公开**——这些是替身的伏击点。**任何"他从前其实喝过酒"之类的暗示首周目都不要写**。
3. **他的手表是父亲的**——这是真总统**真正的私人物件**。**Saulito 在错误时机摸 / 提到这块表都构成暴露**。
4. **演说语言只用"山、土、铜、牛"喻体**。**禁止**让他用"算法、市场、改革"等都市语汇——他不是技术官僚。
5. **写他迷信时不要写他"虔诚"**——他**不是天主教式的迷信**。他是 *brujería*（民间巫术）式的迷信。这两种东西**在维拉杜里亚是不同的事**。
6. **他不是单纯的反派**：他第一任期**真有政治成绩**（参 `01_世界观` §一）。写到他的地方至少出现一次让玩家想起"他也是有过功绩的"。
7. **真总统的"独自一人"是关键氛围**——任何描写他单独场景的文本，要让人感到 Marquez 式的孤独感（参 `06_主题与调性` §三 文学锚点：《族长的秋天》《迷宫中的将军》）。
8. **首周目玩家不可见的内容**：参 `99_秘密角色/_阅读警告.md`。**不要在主目录任何位置暗示真总统已死、是替身链一员、或与 Saulito 父亲有关**。
