## 二、声音卡

### 总规则
**Saulito 是本作唯一拥有"内心独白"的角色**——第二人称、斜体。其余时刻他作为台面上的总统讲话。这意味着同一秒里他会有**两套语言**：
- **舞台上**（场景叙述者描写他的对白）：他在尝试模仿真总统的"严父中间派"语气，但**总会漏出街头底色**。
- **舞台下**（斜体、第二人称内心独白）：**真正的 Saulito**——口语、短促、偶尔粗口、带朗姆味。

### 内心独白（斜体、第二人称）— 唯一规则手册

```
长度：常态 1–3 句。极端时刻可拉到 4–5 句，但绝不超过 6。
人称：你 / 你的（永远第二人称）。
节奏：短句。喘气感。允许"。…。"中间留白。
词汇：街头西语 + Las Veladas 黑话 + 偶尔粗口（Carajo / Hostia / Mierda 谨慎用）。
禁词：民粹主义 / 寻租 / 财政紧缩 / 治理结构 — 任何政治学术语 ✗
       除非：他从别人嘴里刚听到，正在脑里复述（这是允许的"半懂不懂"）。
情绪锚点：困、饿、想喝、记得 Lucía、想起母亲膝盖、想起狗。
```

**示范片段**：
> *他们在等你说话。Vega 告诉过你，签备忘录的时候要皱眉、不能笑。你皱眉了，但你皱得太久——那个穿白手套的女人，Doña Isabel，她正盯着你看。*

> *你想喝一杯。你不能喝。你想喝一杯。*

> *四万北元。他妈的四万北元。这是 Lucía 一个学位的钱。这是你妈那只膝盖换两个的钱。*

### 舞台上的对白（场景叙述者笔下）

```
风格：努力模仿"严父中间派"，**短句、用比喻而非数据、爱乡村俚语**。
偶尔出错：在重要瞬间街头底色会冒出来——一个无端的笑、一句"Hermano"
         脱口而出、一个本应该是总统的人不会做的小动作。
口头禅：开头  "*Pueblo mío*…"（**他记得这个，他会用，但用得太勤**）
        结尾  "*Que Dios nos ayude*"（紧张时会忘，会用 "*Que Dios nos cuide*"
              代替，那是 Las Veladas 妈妈们说的版本——这是**易暴露细节**）
        Saulito 自己的口头禅：**"Mira"**（"看……"）— 真总统从不用，
                              这是他跑题前的过门词
```

### 标志性动作

- **真总统**：紧张时摸右手腕的 Seiko 表。
- **Saulito（自己）**：撒谎时摸鼻梁。**这是"他自己"暴露的最大破绽**。
- 错误时机摸表 / 不该摸鼻梁却摸了 → *Verosimilitud* 风险。

### 不会说的话（首周目硬规则）

- 任何政治学术语原创使用（除非是他刚被科普后复述）。
- "我父亲那块表"——这是**真总统的台词**，Saulito 没父亲（他父亲在他婴儿时"消失"了）。在错误场合脱口而出会造成 ✗ 暴露。
- "去他妈的"系列直白脏话——**他会想，但不会说出口**。台面上他憋。憋的状态本身可以写。
- 任何"我相信民主 / 自由 / 改革"式抽象政治宣言——他不信这些。**他的政治直觉是街头义气**，不是理念。

## 三、与替身位置的关系

- **不是英雄主义入局**：他答应是因为 (1) 四万北元押金 ≈ Lucía 念完护理 + 母亲膝盖钱（参 `01_世界观` 物价表）；(2) **演技虚荣**——一辈子在演总统，没想过有一天能在 La Casa Cobre 演。
- **不是反英雄入局**：他没在反抗什么。**他没有政治觉悟**。他只是被夹住了。
- **后期反讽**：玩家会逐渐发现 Saulito **比真总统更像一个好的总统**——这是本作的重要节奏。**写作时不要提前点破**。
- 道德底色：街头义气型。不主动作恶，但容忍很多事。**会被场景刺到**——孩子被踩死、母亲被赶出医院——做出超出剧本的"Saulito 本能"分支。

## 五、写作红线

1. **三种声音不混**：场景叙述者写他的"舞台动作"+ 第二人称斜体写他的"舞台下"。**不要让外部叙述者直接评论 Saulito 的内心**。
2. **不要在玩家可见文本里出现"父亲"线索的任何暗示**——首周目 Saulito 自己不知道他父亲是第一任替身（参 `99_秘密角色/Eladio_Tobar_第一替身.md`）。"我父亲"在他口中**只能是一个空白 / 抽象 / 街头神话化的形象**，不能具体。
3. **黑色幽默来自情境**：他在国宴上面无表情地说"是啊，我亲自选的酒"——而玩家知道那酒值他半年房租。**不要旁白点破"多么讽刺啊"**。
4. **酗酒不是搞笑桥段**：Cordura 的酗酒动力学是机制（参 `04_数值与阵营.md`）。Saulito 喝醉是**痛**，不是滑稽。
5. **不要让他过早觉醒**：他不会在 Acto II 突然有政治理想。觉悟来自具体事件的"刺"，**不来自演讲**。
6. **数字要有体感**：他每次想到钱，参照点必须是物价表（鸡蛋 / 排队人数 / 母亲膝盖 / Lucía 学费）。
7. **不要把他写成马里奥式底层小聪明**——他可以会演、会哄、会转身找退路，但他**也会哭、会怕、会突然憋住不说话**。
