# 角色生成清单

> **每个角色先生成 canonical 头像（C00X_bust_v01）**，确认调性后作为 cref / IPAdapter 输入跑变体 + 场景。
> **风格 anchor** 全部引用自 `../_风格指南.md`——本文件每条 prompt 已内嵌一份，复制即用。
> 改风格 = 改 `_风格指南.md`，再用 find-replace 把每条的"=== STYLE ANCHOR ==="段同步过来。

---

## 0. 通用模板（每条 prompt 的结构）

```
=== STYLE ANCHOR ===     # 来自风格指南 §十，全角色共享
=== SUBJECT ===          # 角色识别——年龄/族裔/面部/身体/服装/道具
=== SETTING ===          # 场景 + 光线
=== COMPOSITION ===      # 镜头 + 角度
=== MOOD ===             # 一句话情绪锚
=== TECH ===             # 比例 + 工具后缀
=== NEGATIVE ===         # 来自风格指南 §六
```

每条 prompt 末尾的 `[NEGATIVE]` 用以下文段统一替换:

```
NEGATIVE: anime, manga, cel shading, comic book, cartoon, chibi,
3D render, CGI, video game cinematic, ultra-clean digital,
smooth plastic skin, flawless skin, modern beauty filter,
social media gloss, glamour photography, fashion editorial,
high saturation, neon, vibrant colors, HDR, oversaturated,
extreme bokeh, Western fantasy, sci-fi, futuristic,
text, watermark, signature, logo, frame, multiple faces,
duplicated limbs, distorted hands.
```
