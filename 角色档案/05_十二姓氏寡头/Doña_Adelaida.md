# Doña Adelaida Ríos-Echeverría

> 她不上电视。
> 但你今晚在每一台电视里看到的镜头，是她剪的。

| | |
|---|---|
| 全名 | Adelaida del Pilar Ríos-Echeverría |
| 年龄 | 51 |

## 二、声音卡

```
体裁：**社交场上最让人放松的人**——直到你意识到她在记每一句你说的话。
节奏：极有节奏感——她**让对话流动**。她不是讲话最多的人，但她让别人讲话最多。
词汇：艺术词汇 + 八卦 + 间接政治评论；
       她**不直接讨论政治**——她讨论"那场展览的开幕"、"那个导演的太太昨晚的礼服"——
       但每一段都嵌着一条政治资讯。
口头禅：
  - "*Pero cuéntame...*"（"再讲讲……"——她让人继续讲话的咒语）
  - "*Qué encantador.*"（"多迷人啊。"——她的标准反应——
     **它不携带任何评价**，这是她的工具）
  - "*Mira tú, Augusto...*"（"看你，Augusto……"——她对真总统的标准开场，
     比 Doña Isabel 的"*Mi querido*"更随意）
```

### 标志性动作
- 在派对上**不喝完酒**——她端着杯子半小时，只啜两次。
- 听别人讲话时**轻轻倾身**——这是她让对方放松的招牌动作。
- 离场时**总是先碰对方手肘**——一个隐性的"我们还会再见"信号。

### 不会说的话
- 任何对姐姐 Doña Isabel 的批评——她**不公开反对家族**。
- 任何宗教用语——她比 Don Federico 更**优雅地**远离宗教（她不是反信仰，是不在乎）。
- 任何关于第一段婚姻的事——这是她最深的伤（医生丈夫被发现婚外情后她离婚——而那段婚外情对方是 Doña Isabel 的某位亲信。这是家族内部的**未爆弹**）。

## 三、与替身位置的关系

### 她不直接施压
- 她**从不**在政治会议上提议什么——她让 Doña Isabel 提。
- 她**通过媒体**间接影响——某些新闻被压、某些新闻被放大。
- **她不揭穿替身**——但**她可能在某次社交场合的小对话中无意识递出一个测试**。

### 她真正在意什么
- **媒体的延续** —— 她家族的电视台是她**真正的私人项目**。
- **儿子 Tomás 的前途**——她不像 Doña Isabel 那样把 Mariana 作政治资产，但她**也希望** Tomás 在家族银行外有更多话语权。
- 她对 Mariana 与 Tomás 的暧昧——**她知道，她沉默**。这是家族内部"半公开的秘密"。

## 四、重大事件

### Acto II 国宴
- 她坐在 Saulito 不远处。她**没主动找 Saulito 讲话**——她在**观察**。
- 她对 Mariana 短促私语——这是 Mariana 对替身首次"识破"的旁观者证据（多周目深度路径上有用）。

### Acto III 中（媒体调度首次显形）
- 一个对寡头不利的小新闻**没上电视**——Saulito 在 Casa Cobre 看新闻时发现"昨天 Cumbre Mayor 矿区抗议没报道"。
- Susana 在档案中加 ◇ 注解："Adelaida Ríos-Echeverría 控制的某频道编辑会议昨晚开会两小时。"
- 玩家选择是否**亲自联系 Adelaida 商讨媒体合作** → 这是 Adelaida 路线的入口。

### Acto IV 中（与 Adelaida 的私下茶会）
- 玩家如选了路径——Adelaida 在自己的画廊接待 Saulito。
- 她**讲了一整段画展开幕的故事**——其中嵌入：(1) 北方使馆某位副官最近换岗；(2) Vasconia 文化中心要在 Las Veladas 办活动；(3) "我母亲那一代女人都不容易，对吗？"
- **每一句都是测试**。**答错 → 派系 *Sospecha* +5**。
- 她私下提议"媒体合作"——表面是"为总统形象做包装"，**实际**是把 La Casa Cobre 的内部资讯卖给寡头电视台。

### Acto V（与 Mocosos 的代理战）
- 一场公开抗议被直播——Mocosos 圈在 TikTok 上把 Saulito 的某个表情变成 meme。
- **Adelaida 主动找 Saulito**："我可以让那个 meme 消失。" 这是她的"善意"——但代价是 Saulito 必须**让步**某条媒体监管法案。

### Acto VI / 终章
- 视路径：
  - Saulito 接受 Adelaida 媒体合作 → 终章中她的电视台**保护** Saulito 一段时间
  - Saulito 拒绝 → 她不报复，**她切断**——她对 Saulito 的所有协助消失
  - True Ending Route → 她**在最后一刻揭穿** Saulito（或**故意**没揭穿）—— 她的选择是**多周目玩家**才能解读的复杂层

## 五、写作红线

1. **她不是反派姐妹**——她**比 Doña Isabel 更"现代"**，但**也更危险**——她**不假装信任何东西**。
2. **她从不在政治会议上提议**——她**始终**通过媒体 / 私人茶会 / 社交场。**禁止**让她在内阁会议公开发言。
3. **她的第一段婚姻禁区**：参不会说的话。**禁止**让其他角色公开提及。多周目深度路径可揭示。
4. **她对 Mariana 与 Tomás 的暧昧的沉默**：**写作时让她每次出现 Mariana + Tomás 同框时**轻微改变眼神方向**——这是她"知情但沉默"的真实质感。
5. **她的耳环 / 不戴白手套** vs. **Doña Isabel 戴白手套**：这两位姐妹的视觉对比是**身份语言的核心**——任何写两人同框的场景都要保留这种对比。
6. **她不揭穿替身**——首周目锁死。**多周目深度路径**才决定她最终选择揭穿 / 沉默。
