## 设计原则（提前定，不需要等）

1. **不用 emoji 装饰玩家可见 UI**（参 `CLAUDE.md` §8 反模式）。
2. **数值变动不直接显示数字**——Tier 1 是文字标签，不是 "+5 Legitimidad"（参 `04_数值与阵营.md`）。
3. **三种声音必须视觉可区分**——这是渲染层的核心 ROI。
4. **风格与角色 / 场景图同源**——参 `../_风格指南.md`，UI 框 / 字体 / 色板与图片美学一致（哑金 / 铜锈红 / 灰白石灰 / 炭黑）。
5. **字体**:正文中文用衬线（思源宋体 / Source Han Serif），西语 flavor 词用 Garamond 或 Bembo 类衬线，新闻广播段落用模拟打字机字体。**禁止**圆体、卡通字。
6. **UI 元素动画**:克制——不滑入、不弹跳。**淡入 / 淡出 + 短停顿**。
