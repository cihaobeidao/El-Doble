# Iván Báez · 长子

## 一、身份信息

| | |
|---|---|
| 全名 | Iván Augusto Báez Marqués |
| 年龄 | 28 |
| 出身 | 在 La Casa Cobre 长大；私立寄宿学校 → 北方某商学院辍学 |
| 公开身份 | 名义"家族投资公司"高管，实际是花花公子 |
| 财富 | 父亲名下的若干资产代持人；个人月开销 ~80,000 北元 |
| 与圣灵集团的牵连 | 通过夜店关系认识 *Renata Salazar*（El Padrino 的女儿）；曾与圣灵集团边缘人物在 Puerto Sangre 一起被警察拍到过——记录被销毁 |
| 与父母关系 | 与母亲 Gabriela 仍亲，**与父亲多年疏远**（最后一次正式见面是 2024 年圣诞）|
| 标志 | 一辆改装过的银色 Maserati；手腕上戴一条镶嵌字母 "I.B." 的金链 |

## 二、声音卡

```
体裁：富二代 + 半受过教育的拉美男青年口语。
节奏：散漫，常**跳话题**。他是那种"开会迟到一小时但 90 秒内能让所有人笑"的人。
词汇：英语词夹杂多（**business / deal / weekend / cool**）—— 这与
       街头 Spanglish 不同，他的英语是**北方留学**带回来的。
口头禅：
  - "*Eh, viejo...*"（"嗨，老头……"——他对所有上了年纪的男人用这个，包括他父亲）
  - "*Tranqui.*"（"放松。"——他想化解尴尬时）
  - "*Es lo que hay.*"（"事情就这样。"——他逃避深谈时）
```

### 标志性动作
- 谈话时**反复看手表**（不是 Seiko——是一只 Patek Philippe）。
- 笑时露上排牙齿（漂亮，**他知道**——他用过这个表情拿过赞助）。

### 不会说的话
- 任何对父亲的怨言——他**不愿在外人面前说**。但**他与母亲单独时会说**。
- 任何具体的圣灵集团相关——他**装作没听过这个名字**。

## 三、与替身位置的关系

- **他不见替身**——他几乎不在 La Casa Cobre 出现。但他会在某些场合（葬礼 / 国宴 / 慈善活动）短暂出现。
- 他**不识破**——他与父亲多年疏远，他对父亲的**身体细节**没有印象。
- 他的危险性来自**圣灵集团**——他与 Renata 的关系可能被集团用作"我们家人"的杠杆。

## 四、重大事件

### Acto II 初（首次见面）
- 一场公开慈善活动 —— Iván 短暂出席。他对"父亲"伸出手："*Eh, viejo...*"
- Saulito 选择回应方式：
  - **(a) 短促点头不回话** → 中性
  - **(b) 用 Augusto 的"严父"姿态训斥几句** → 寡头 +3（保守派欣赏），Iván 内心反感
  - **(c) 像 Saulito 自己那样，**回拍他的肩膀** → Iván 短暂愣住——**他从未被父亲这样拍过肩**。Cordura −5（暴露风险），但**触发隐性轴 IVAN_NOTICED_SOMETHING**

### Acto III（圣灵集团第十天送礼前）
- Iván 私下来到 La Casa Cobre 找"父亲"借钱（他欠了 Renata 一笔）。
- 玩家选择：
  - 给 → 圣灵集团 +5（这等于通过 Iván 间接接受了集团），但 Cordura −5
  - 拒绝并训斥 → 寡头 +3 / Iván 离场 + 街头流言（"总统连儿子都不养"）
  - 让 Susana 处理 → Iván 路线 Sospecha −5

### Acto V（母亲的电话）
- Gabi 远程电话："Augusto，Iván 周末去 Puerto Sangre 了。" 这是 Gabi 在用 Iván 测试 Saulito 是否会做"父亲该做的事"。
- 选择反应决定 Gabi 与替身关系的走向。

### Acto VI / 终章
- 最坏路径：Iván 因卷入圣灵集团一次内部纠纷被砸头送医。**他不死**（参政策：本作不轻易杀子女）。
- 最好路径：Iván 在终章某场景里，**第一次单独对"父亲"说一句话**："*Viejo, te ves cansado.*"（"老头，您看起来累了。"）—— 这是他能给出的最大善意。

## 五、写作红线

1. **他不识破**——首周目锁死。
2. **他与父亲疏远**：写他时**永远保留情感距离**——他不主动靠近"父亲"。
3. **他不被简化为"反派儿子"**：他有真实的人性瑕疵和真实的脆弱。**不要让他成为单纯的麻烦制造者**。
4. **圣灵集团关系含糊**：**禁止**让他直接出现在 El Padrino 的庄园场景——他只是**边缘人物**。
5. **他与 Renata Salazar 的关系**：可以暗示，**不写情色细节**。
6. **他与 Mercedes（妹妹）**：他们关系还行，但**很久不联系**——这是他孤独的来源。
7. **他不死**——参 Mariposa 的政策原则。
