## 二、声音卡

### 与 Saulito 的根本差异
- Saulito：**从零学起**。她**懂**政治。
- Saulito：街头底色掩盖学识匮乏。她：**学识之上有职业控场感**。
- Saulito 内心独白第二人称口语；**Catalina 的内心独白用第一人称、广播式半正式短句**——像她在脑里给自己写一段不会播的稿。

### 公开场合（"假装是 Gabriela"的版本）
```
体裁：女性政治家公开形象——温暖、清晰、有节制。
节奏：每句结尾会**轻轻下沉**——这是主播本能。
词汇：避免学术化政治词，用 "我们" / "这片土地" / "孩子们的明天"。
危险：**她讲话太流利**。Gabriela 真人的对外讲话有一种刻意的"留白"——
       Catalina 不留白。这是她最容易被识破的细节。
```

### 内心独白（**第一人称、广播稿式**）

```
人称：我 / 我们
节奏：完整的句子。逗号与破折号多。
词汇：政治学术语允许（她懂），但带着"我可以这么说但不能这么播"的距离感。
情绪锚点：父亲、麦克风、那个北方记者前男友、
        某个她曾在节目里访谈过、第二天就被失踪的工人。
```

**示范片段**：
> "他在看我。他不像 Saulito 那样会逃我的眼睛——他在看的是我袖口的扣子，
> 那是 Gabriela 的扣子样式，不是我的。我应该笑一下。半秒。不能多。"

### 标志性动作 / 习惯

- **指间转笔**：在 ON-AIR 时是她的招牌；在替身位上她**会忍住**——但忍得太刻意时会变成另一种破绽。
- 戴老花镜并非真的近视——是节目人设。**她在内阁会议上摘下，戴上，又摘下**。
- 喝**冰水**而不是茶（Gabriela 是茶）。这是导播组要纠正的细节之一。

### 不会说 / 不会做的话（首周目锁死）

- **她不模仿 Augusto Báez**——她替的是 **Gabriela**。**她从不戴假胡子**（替身 IV Gabriela 自己反而会戴，那是仪式性玩笑——参 `Gabriela_Báez_第四替身.md`）。
- 不在公开场合提自己 Radio Aurora 的旧节目。

## 三、与替身位置的关系

- **解锁触发**：第一周目结局后（无论成败），玩家若解锁了 Vega 的"长期计划"档案，会发现**青年军官派早就准备好了第二位替身**——但这一次替身的是 Gabriela，而非总统本人。
- **公开剧本**："Augusto 总统因健康原因暂别公职，由第一夫人按宪法 X 条主持日常政务。" 街头反应未必激烈——Gabi 的国民熟悉度本就比真总统更高。
- **Catalina 的弱点**——她的脸太多人见过。Radio Aurora 老粉一眼能听出她的声音底色。**她的容错率极低**。
- **专属悬念**：她为什么愿意接？**她真的愿意吗？她是不是为了 Báez 政权倒台后的某个位置？**
- 她的解锁档案优势：**媒体网络**（自己经营了 6 年的电台）、**女性政治圈**、**与 Cardenal Soto 的旧交情**（她青年时在教会做过义工）。

## 四、重大事件

> Catalina 路线的具体场景仍以 `08_剧情/*` 为准——本节列出**专属于她的"声音/媒体"型事件**。

### 序幕（Catalina 路线，与 Saulito 序幕不同）
- **她不是被绑架来的**——她是被 Vega 当面**邀请**的。地点在某个深夜的 Radio Aurora 录音间，Vega 进来时她正在审第二天的稿子。
- **三个"选项"通向同一结局，记录姿态**：A 立即接受 / B 要求先见 Gabriela / C 谈判保留 Radio Aurora 编辑独立。
  - 选 B 的玩家：第一幕开端有一个"Gabi 第一次见替身"的私密镜头——这是 Saulito 路线没有的。
  - 选 C 的玩家：第二幕有 Whitfield 试探她"Radio Aurora 最近换风格了"的事件链。

### Acto I（媒体—权力转换的痛感）
- **第一次广播改稿**：Susana 给她的稿子里有一句明显是 Gabi 的口吻。她改了。**改了之后就出事了**——Gabi 远程电话进来。这是她与 Gabriela 的第一次"声音对声音"。
- 内阁会议上她**与 Castaño 第一次正面**——Castaño 看不起女人主政，但他更怕 Gabriela。

### Acto III（Cardenal Soto 旧交情触发）
- **教堂私聊**：Cardenal 不揭穿她，但他问的是 "*Hija, todavía rezas?*"（"孩子，您还祈祷吗？"）——这一句话是 Catalina 路线的 Cordura 危机。
- 她的弟弟（一位中学教师）从外省打电话进来——他没认出她不是 Gabriela，**但他听出"姐姐你嗓子怎么了"**。

### Acto IV（与 Mariana 的镜像戏）
- 寡头不再提"离婚 Gabi 改娶 Mariana"——他们提的是"为了国家形象，您应当与某位寡头家族男性订下私下协议"。**联姻线在 Catalina 路线被改写**。
- Mariana 第一次直接说："我以为您是 Gabi 的电视粉丝。" — 她从一开始就**知道**。

### Acto V（两国大使同时露面 / 媒体战）
- Aldana 提议"Vasconia 投资 Radio Aurora 的姊妹台"——表面是文化合作，实际是把 Catalina 的媒体根据地买走。
- Whitfield 不递地址纸条——她递的是**一段录音**，那是 Catalina 三年前一次未播的访谈。"我们替您保管了。"

### Acto VI（声音的最终代价）
- 一场公开演说前，**她的嗓子嘶哑了**——这是机制：长期高强度模仿 Gabriela 的语调对她真实声带的损伤。她必须选 (a) 用药硬撑 / (b) 取消 / (c) 用"自己的声音"演讲（这是 *La Voz* 的核心抉择）。
- 选 (c) 触发独立结局分支 *La Voz Real*（"真正的声音"）。

## 五、写作红线

1. **她不是 Gabi**：写她公开讲话时，**不要让她和 Gabi 完全一致**。**留 0.3 秒的偏差**——这是她身份的真实空间。
2. **Radio Aurora 旧节目内容**写到时**只能从听众的回忆中出现**——**不要写她自己在替身位上回忆 Radio Aurora 的具体节目**，那会让她"出戏"。
3. **她的内心独白第一人称**——绝不与 Saulito 的第二人称混用。**写作时要清楚自己在哪个替身的视角下**。
4. **"Hija, todavía rezas?" 这种线**只能由 Cardenal Soto 触发——**不要让其他角色重复这个语气**。教士的语气和别人不一样。
5. **她与前男友（北方记者）的关系**：可以提，但**不要写情色细节**。这是政治性的关系（一个北方记者 + 一个本国电台台长），其复杂性高于浪漫戏。
6. **不要让她变成"女权胜利"叙事**——本作不写"小人物逆袭"，对女性政治家也不写"她们被压迫但终将获胜"的爽文。Catalina 的弧线是**她和这个国家一起耗损**。
