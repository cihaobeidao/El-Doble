# Mariposa · 那只脏狗

> 它认你。
> 你穿西装它也认。
> 你换头发它也认。
> 你说官话它就盯着你。

## 一、身份信息

| | |
|---|---|
| 名字 | Mariposa（西语"蝴蝶"——Saulito 喝醉后给的名字，与狗本身毫无关系） |
| 品种 | 街头杂种，体型中小，毛色深棕带白胸 |
| 年龄 | 约 6 岁（Saulito 一直说"差不多" — 街头狗没有明确生日） |
| 状态 | 序幕被留在酒吧门口；第一幕中段街区有人收养；第四幕末**在某场公开活动中突然出现在人群中** |
| 标志 | 左耳缺一小块（被 Las Veladas 街头一辆摩托擦过）；尾尖是白的 |
| 习性 | 不咬人；怕雷；喜欢咖啡渣；从不进教堂（Las Veladas 老说法："狗不进圣母的院子") |

## 二、声音卡（动物的"声音"——其实是它如何被书写）

> 它不说话。它的"对白"是动物行为描写——而**这些描写要克制、不拟人化**。

```
描写规则：
  - 不写它"想"什么（不要内心独白化）。
  - 写它**做**什么——闻、退、转头、盯着、低吼、舔。
  - 写**周围人对它做什么**——一个孩子拍它、一个警卫赶它。
  - 永远**不让它"开口"**（即使是想象的）。
禁忌：
  - 不写"它好像知道似的"——这是廉价拟人。
  - 不让它救主——本作不写"狗英雄叙事"。
  - 不让它死——它至少活到游戏结束（参 99 写作政策：动物不死）。
        如确实需要"消失"，让它走进巷子里再没出来——克制且无解。
```

### 标志性出场方式
- 第四幕末公开活动——**它从人群里走出来**。它不冲，它不叫。它**闻、停、坐下**。
- 它的视线**不离开 Saulito**。它**不接近**——它只是确认。

## 三、与替身位置的关系

### 它为什么是 *Verosimilitud* 危机
- 真总统从不养狗，从未与 Mariposa 接触。**Mariposa 来自 Saulito 的街区**。
- 它认 Saulito。它**不会被化妆 / 假发 / 颊植入骗到**。它认**气味**（Las Veladas 的某个面包店、酒、某种汗腺特征）。
- 它出现 = **对所有在场寡头与使馆人员"为什么这条狗对总统这么熟"的活质疑**。

### 它不会做的事
- **它不会咬假总统**——它不愤怒。它**爱 Saulito**，它**好奇**。
- 它不会跑过来扑——这是俗套电影桥段。它**只会安静地观察**——这才是真正的 *Verosimilitud* 危机：一个让所有人不舒服但又说不出口的细节。

## 四、重大事件

### 序幕（第 0 天 04:17）
- Saulito 走出 *El Gallo Borracho* 时它在门口等。Saulito 走进黑色雪佛兰前**摸了它一下**。
- "*Cuídate, gordito.*"（"自己照顾自己，胖小子。"——他的别名）
- **它跟着车跑了几步，停下，坐在路灯下**。

### Acto I 中段（远处的消息）
- Las Veladas 的某个小贩告诉 Susana 的某个采购员："*Saulito 那条狗，Pancho 收了。*"——Susana 在档案里加了一条 ◇ 注解：街区有人收养了它。
- Saulito 看到这条 ◇ 时长出一口气。**这是 Cordura +5 的小阳光**。

### Acto III（背景音）
- 某次出席 Las Veladas 慈善活动——人群里有一个孩子领着一条狗。**Saulito 不能确定那是不是 Mariposa**。
- 这是**伏笔**——首周目玩家可能不会意识到。

### Acto IV 末（最棘手的危机）
- 公开活动（露天音乐节 / 露天弥撒 / 街头节庆）——**Mariposa 从人群里走出来**。
- 它坐下，盯着 Saulito。
- **场上的反应**：
  - Doña Isabel：眯眼。
  - Mariana：**笑了一下**——但只有 Saulito 能看见。
  - Vega：手按枪柄。
  - Susana：不动声色地从侧门走出去——她是**唯一一个能解决这件事的人**。
- 玩家选择：
  - **(a) 让 Vega 处理** → 警卫赶走 Mariposa，但寡头 *Sospecha* +5（"为什么要驱赶一条无害的狗？"）
  - **(b) 走到它面前蹲下** → *Verosimilitud* −10 但 Cordura +20（"他像是认识这条狗"）
  - **(c) 完全无视，继续讲话** → 中性数值，但 **Cordura −15**（"他在心里问自己什么时候变成不蹲下的人")
- 这是本作前 5 幕中**最微妙的事件**之一。**写作时要让玩家感到"哪个选择都不对"**。

### 终章（True Ending）
- Saulito 站在 Las Veladas 黎明前的灰光里——**Mariposa 走过来，坐下**。
- 它不再确认——它**知道**。

## 五、写作红线

1. **不拟人化**：参声音卡描写规则。**它的内心从不被叙述**。
2. **它不死**——这是设计政策。如需"消失"用克制方式（走进巷子）。
3. **它不咬人 / 不扑人 / 不英雄式救主**——它只是**确认**。
4. **它的名字 *Mariposa* 是反讽**——一只脏狗叫"蝴蝶"。**不要点破这个反讽**。
5. **它出现在公开活动的那场戏**：**严禁**让它"狂吠揭穿"——它**安静**才是真正的危机。
6. **它的标志细节（左耳缺块、尾尖白）**：写到时**至少二选一**让玩家在画面里看到——这是首周目玩家在 Acto IV 末"哦这是那条狗"的认知锚。
7. **它不属于任何派系**——任何派系想"利用"它的剧情**禁止**。它是 Saulito 的私人重力，仅此而已。
