# Gabriela "La Gabi" Báez · "La Primera Dama"

> 她不模仿任何人。她以"Gabi 代理执政"的合法性公开露面——
> 戴上总统的假胡子只是一场仪式性的姿态。
> 这是黑色幽默感最重的一条线。

## 一、身份信息

| | |
|---|---|
| 全名 | Gabriela Soledad Báez（婚前姓 Marqués） |
| 昵称 | "La Gabi"（国民几乎都这么叫她，比叫"夫人"亲）  |
| 年龄 | 49 |
| 出身 | 沿海港口 Puerto Sangre 中下层移民家庭。父亲是码头工人，母亲洗衣。 |
| 上行轨迹 | 1996 年小姐冠军 → 晚间新闻主持十一年 → 1999 年与 Augusto Báez 结婚 |
| 公开形象 | "国民第一夫人"；笑容温柔；天主教徒；常做慈善 |
| 真实底色 | **她比她丈夫聪明，但她从未以"聪明"作为公开形象** |
| 子女 | Iván（28，长子）/ Mercedes（24，长女） |
| 标志 | 一只 1996 年小姐冠军的胸针——她从不戴，但放在化妆台上 |
| 信仰 | 公开虔诚天主教；私下：**她相信权力**；她对 Cardenal Soto 既尊敬又**保持距离** |

## 二、声音卡

### 公开（电视、慈善、葬礼上）
```
体裁：温暖、节制、清晰。**她讲话有"留白"**——这是她与 Catalina 的核心区别。
节奏：她**敢停顿**。一段话里有 1–2 个 1.5 秒以上的沉默。
词汇："我们" / "孩子们" / "这片土地" / "信仰" / "家"。**少用政治术语**。
开头：常以"*Buenas tardes, mi gente*"（"下午好，我的人们"）开场。
结尾：常以一句无主语的祈祷收尾——*"Que descansen los nuestros"*（"愿我们的人安息"）。
```

### 私下（与 Susana / Vega / 真总统）
```
体裁：**短促、利落、毫无废话**。她不浪费私人时间在客套上。
节奏：她在私人场合说话**比公开场合快 30%**。
词汇：允许出现政治学术语（她**真懂**），允许偶尔粗口（极少，但有）。
口头禅（私下）：
  - "*A ver, querido*..."（"我们来看看，亲爱的……"——
     在她要纠正你时的第一句）
  - "*Mira, no.*"（"听着，不行。"——她不商量某事时）
```

### 内心独白（第四替身周目）

```
人称：第一人称单数与"我们"交替——这是她思考权力时的语法。
       她思考个人时用"我"，思考国家时用"我们"。
节奏：完整、有结构。**她不会失控**——这是她与其他替身最大的区别。
词汇：政治学术语 + 文学化比喻 + 偶尔回到 Puerto Sangre 童年的口音词。
情绪锚点：父亲的码头、Augusto 第一次见面时的傻笑、
         Mercedes 寄回的最后一封信、Iván 不接她电话的某个晚上。
```

**示范片段（第四替身周目）**：
> 我系上他的领带。这件事我做了二十七年。
> 这个国家只有三个人知道我比 Augusto 更能让这个领带打得直——
> Susana、Mateo Vega、还有他自己。
> 他没有反对过。他从来没有。

### 标志性动作 / 习惯

- **整理别人的领带**——这是她的招牌动作（媒体记录中她在国宴上为 Augusto 整领带 N 次）。在替身周目里**她会为 Saulito / Tobías / Catalina 整领带**——这是她的"我接管了"无声宣告。
- **不戴婚戒**（自 2024 年起）—— 只有几个人注意到。这是她的私人语言。
- 慢饮咖啡。**她从不端起杯子立刻喝**——她端起后会先看一眼桌面再喝。

### 不会说 / 不会做的话

- **她从不在公开场合反驳 Augusto**——这是 27 年的政治表演。
- 她不会笑出声。**她最多微笑**——但**笑得让你以为她笑出声了**。
- 她不主动提她的父亲（已故）—— 她父亲生前从未来过 La Casa Cobre。这是她最深的心结。

## 三、与替身位置的关系（首周目 NPC 视角）

### 她在剧情中作为 NPC 的份量
- **远程电话**：她 90% 时间在外（医院 / 出访 / 慈善活动），通过电话与"总统"对话。这是 Saulito 路线最大的"她不在场"机制——她不见 Saulito，所以她不需要被骗到。
- **首次见面**：被她递延到 Acto III 末或 Acto IV 初——具体时机由玩家选择决定。
- **她的怀疑**：她从第二次电话起就**听出 Saulito 不是 Augusto**——她比所有派系都先识破。她**不揭穿**——参下面"她为什么不揭穿"。

### 她为什么不揭穿（首周目逐步揭示，多周目深化）
- 首周目层面：她可能怀疑这是 Vega 与 Susana 的安排，她在等他们解释。
- 多周目层面：参 `99_秘密角色/_阅读警告.md`——**她是这台机器的设计者**。

### 第四替身周目（极后期解锁）
- **解锁条件**：玩家在多个周目中识破 Gabriela 在不同时机的多次插手。
- **视角差异**：她**不模仿任何人**。她以"Gabi 代理执政"的合法性公开露面——戴上 Augusto 的假胡子是仪式性玩笑（一段冷到底的黑色幽默）。
- **派系关系全部重置**：所有派系对她都是已知的、有自己历史的人。Doña Isabel 与她有 25 年的"互相敬重的敌意"。Cardenal Soto 见过她童年（Puerto Sangre 教会）。
- **专属机制**：她的"伪装度"被替换成"权威度"——她的难度不是"装得像不像"，是"她敢做多少 Augusto 不敢做的事"。

## 四、重大事件

### 首周目 NPC 时段（Saulito 路线）
- **第一次电话**（Acto I 中段）：她问 Saulito 关于 Mercedes 一封新邮件的细节——Saulito 答不上来。她**没追**。她转话题问"你今天吃饭了吗"——这一句让 Saulito 第一次不知道该用哪个嗓子。
- **第二次电话**（Acto II 末）：她说"Augusto，我希望你记得 Lucía 的生日"——她叫的是 Saulito **女儿**的名字。她在送一个无声讯号：**我知道你是谁**。
- **首次见面**（Acto III 末 / Acto IV 初）：Casa Cobre 私人客厅。她不锁门。她让仆人退出。她坐下，**她整他的领带**。她什么也不说。然后她说"你比他善良。这是好事，也是坏事。"——这句话是 Saulito 路线的 Cordura 大爆点。
- **Acto V 国宴**：她在场。两国大使同时露面那场——**她替 Saulito 撑住一半**。哪一半，由玩家在 Acto IV 与她建立的关系类型决定。

### 多周目深度揭示（首周目玩家不应看到的层级）
- 参 `99_秘密角色/_阅读警告.md` —— 她是真正的主谋。

### 第四替身周目（极后期）
- 序幕（她路线）：她坐在化妆台前。Susana 进来。她说"今天我自己来"。
- Acto I：她戴着假胡子开第一次内阁会议。**全场没有人笑**——这是黑色幽默的最高时刻。
- Acto II：Doña Isabel 第一次在私下与她单独喝茶——25 年的对话压在 30 分钟里。
- Acto VI：她**主动公开身份**（她路线特有结局之一）。
- 终章：她可能选择继续 / 让位 / 退场。每一种都伴随她自身的崩塌。

## 五、写作红线

1. **她不是"善良的妻子"**：本作不写她为爱情牺牲的剧本。**她有自己的政治路径**——只是这个国家允许女人公开掌权的窗口太窄，她选择了"夫人"这个标签。
2. **她不会在首周目被识破**——她比所有派系都先识破替身，但她的"识破"通过**侧面提示**（叫错女儿名字、整领带、留白沉默）传递。**禁止**她在首周目台词中直接说"我知道你不是 Augusto"。
3. **她整领带这个动作**严禁在主目录其他角色卡中重复使用——这是 Gabriela 的招牌。**Susana 不整领带，她贴假发**。
4. **她的童年（Puerto Sangre）**只能由她自己提及——其他角色提到时**带着对她出身的某种隐含瞧不起**（寡头会用"她毕竟来自港口"这种半句话来贬损）。这是她政治情商的来源之一。
5. **Cardenal Soto 与她的关系**：互相尊敬但保持距离。**不要让他们有过深的私人交情**——那会破坏 Cardenal 的"道德外部性"地位。
6. **首周目玩家可见的所有 Gabi 文本**：写作时务必反复自检，**不让任何一段提示她是策划者**。她是 NPC，不是反转。
7. **第四替身周目假胡子**：这是**仪式性玩笑**——黑色幽默的最高密度。**不要写成滑稽**。**写成肃穆**。这是她在告诉这个国家"你们的总统从来不需要是男人"。
8. **她不会成为"女权符号"**——她是**一个具体的女人**。她有真实的局限、真实的妥协、真实的冷酷。
