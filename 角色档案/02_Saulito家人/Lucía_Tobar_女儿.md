# Lucía Tobar · Saulito 的女儿

## 一、身份信息

| | |
|---|---|
| 全名 | Lucía Esperanza Tobar |
| 年龄 | 19 |
| 出身 | 首都 Las Veladas（Saulito 与第二任前妻所生）；从 14 岁起独自打工 + 念书 |
| 现职 | 国立大学护理学院二年级；每天 5:00 起床做兼职贴补学费 |
| 学费缺口 | 还差 4 学期 ≈ 12 万 cobre（约 3,160 北元）—— **这就是 Saulito 答应做替身的实体理由之一** |
| 与父亲关系 | "不算亲，但她不恨他"。她见过父亲所有的失败和有限的努力。 |
| 政治倾向 | 她**对 Mocosos 圈子有兴趣**但不是积极成员；她支持 Radio Aurora（**Catalina 的电台**）。这条**首周目玩家不应注意**。 |
| 标志 | 一个旧的医学专业书包（缝补处用红线，是她妈妈缝的）；左眉上方一颗小痣 |

## 二、声音卡

```
体裁：年轻人 + 受过部分高等教育的城市青年口语。
节奏：快。她**讲话比她父亲快**——这是她那一代的特征。
词汇：医学术语开始进入她日常语言（她**会用"心率" / "血压" / "高烧"**——这些是她父亲不熟的词）。
       街头西语保留——她出身那条街。
口头禅：
  - "Pa..."（"爸……"——比 *Papá* 更口语，她叫父亲的方式）
  - "*¿Qué onda?*"（"怎么了？"——青年 Mocosos 圈口语）
  - "*Estoy bien, en serio.*"（"我没事，真的。"——她被父亲追问时的标准回答）
```

### 标志性动作
- 听人讲话时**右手会无意识检查自己脉搏**（医学生职业习惯）。
- 紧张时咬左下唇内侧（无人察觉，包括她自己）。

### 不会说的话
- 任何"我恨你"——她**没有走到这一步**。她的怨意是更复杂的"我希望你别让我担心"。
- 任何浪漫线索——本作不写她的恋爱。她有同学，她有友谊，**她不被简化为感情戏的素材**。

## 三、与替身位置的关系

- **她是 Saulito 路线最大的"内部反向重力"** —— 每次她出场，Saulito 都会想"如果她知道我现在在做什么"。
- **她**不直接威胁替身位置——她是间接 *Verosimilitud* 危机：她可能在某个公开场合**看见"父亲"**，并意识到那个人虽然像他父亲但**有一些她不熟的细节**。
- **她不知道她父亲是替身**——首周目锁死。

## 四、重大事件

### 序幕之前
- Vega 复述她考试时间——这是序幕威胁的实体之一。

### Acto II 末（首次场景）
- 一次抗议行动里医学生急救队的镜头扫到她——电视新闻的画面。Saulito 在 La Casa Cobre 看到。**他不能反应**——内阁会议正在进行。这是 Cordura −10 的硬触发。

### Acto III 中（"接待室"事件）
- Lucía 因学校手续问题来 Casa Cobre 接待室办事（普通市民程序）—— **她不知道她父亲在场**。
- 玩家选择：
  - **(a) 借总统身份避见她** → Cordura −15，标记 `AVOIDED_LUCIA`
  - **(b) 冒险与女儿见面**（她会发现父亲在做什么） → 触发 *La Hija* 事件链，Cordura 大爆 + 寡头 *Sospecha* +10，但解锁 *La Verdad Doméstica* 隐性轴上的关键节点
  - **(c) 派 Susana 出面以"行政人员"身份接待她** → Cordura −5，但 Susana 路线上 +1 信任

### Acto V（间接出场 / 公共事件）
- 一次大学示威，警察暴力镇压——Lucía 在医学生急救队。**Saulito 不能干预**——干预 = 暴露。
- 这是 Saulito 路线"良知与现实的反复磨损"主题最尖锐的一段。

### Acto VI / 终章
- 多周目深度路径：Lucía **在某个夜晚收到一封匿名信**——信里有 Mariposa 的照片（参 `Mariposa_狗.md`）。她**看出来这条狗是她父亲的**。
- True Ending 路径上，Saulito 在最后一刻可以选择**给她写一封信**——这封信永远不寄出，但**它的存在改变结局画面**。

## 五、写作红线

1. **她不构成专属支线**——避免主线被亲情戏拉走。她出场的事件都是**短、痛、快**。
2. **不写她浪漫**——本作不需要她的恋爱戏。
3. **她不知道她父亲是替身**——首周目锁死。**任何让她"识破"的事件都要推到多周目深度路径**。
4. **她讲话用医学术语**——这是她身份真实质感的来源。**不要让她突然变回"普通女儿"**。
5. **Saulito 与她见面的所有事件都要 Cordura 大爆点**——这是机制设计。**不要写"温馨重逢"**——他们的所有见面都是**带刺的爱**。
6. **她的母亲已离婚 / 不在场**：本作不写 Lucía 母亲——避免 Saulito 的"前妻线"成为又一条情感戏副线。
7. **她使用 Radio Aurora**这条线：在 **Catalina 周目**时会有镜像戏（Catalina 在录音间，Lucía 在听）。**首周目不要打这张牌**。
