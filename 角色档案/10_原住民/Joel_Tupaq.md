# Joel Tupaq-Mamani

> 他是 Aurora 的孙子。
> 他不同意他的祖母。
> 他在尝试做一些她明确拒绝的事。

## 一、身份信息

| | |
|---|---|
| 全名 | Joel Tupaq-Mamani |
| 年龄 | 28 |
| 出身 | Cumbre Mayor 山区——Aurora 的孙子 |
| 教育 | Universidad Nacional 人类学系（在读博士第二年） |
| 现职 | 学生 + 兼职 NGO 研究员（一家欧洲资金的"原住民发展项目"基金会） |

## 二、声音卡

```
体裁：受过最高教育的拉美原住民青年——西语 + 本族语流利。
节奏：**比 Aurora 快得多**——他**有焦虑感**——他知道时间不多。
词汇：
  - 学术西语 + 人类学术语
  - 本族语夹杂（**他用本族语 cussing 但用西语进行政治讨论**——
     这是他自己内部的语言分裂）
  - **不**用 Mocosos 网络黑话——他比他们大半圈
口头禅：
  - "*Mira, primo...*"（"看着，表哥……"——他对 Mocosos 圈用）
  - "*Mi abuela dice que no, pero...*"（"我祖母说不行，但是……"——
     他对祖母说法的开场——他**永远说"但是"**）
  - "*¿Cuántos más?*"（"还有多少？"——
     他面对统计数据时的反讽——
     "失踪人数、被强占的土地、消失的语言"等）
```

### 标志性动作
- 谈话时**双手抱胸** —— 这是他**警惕**的姿势。
- **从不**祈祷——他离开了天主教—— 但他**也没有**回到本族信仰——他在两者之间空着。
- 笑时**短促**——他**急**——他**不容许自己放松太久**。

### 不会说 / 不会做的话
- 任何对祖母 Aurora 的公开批评——他**敬重她**——尽管他不同意她。
- 任何对原住民的浪漫化——他**反对**那种"我们是高贵的野蛮人"叙事。
- 任何与圣灵集团的暧昧——他**鄙视**他们——尽管集团在山区有部分影响力。

## 三、与替身位置的关系

### 他不直接接触替身
- 他**通过抗议、专栏、学术报告**间接施压。
- 他**怀疑** Saulito 是替身——但**他没有把这怀疑公开化**——他认为揭穿替身**对原住民议题没有直接帮助**。

### 他想要什么
- **铜矿土地权**——具体到几块矿区的传统土地认证。
- **本族语言进入小学课程**——Cumbre Mayor 至少。
- **承认问题** —— 维拉杜里亚宪法**至今没有承认**原住民的"民族"地位—— 他想要这件事。

### 他与祖母的代际矛盾
- Aurora："不要参与那些 *patrón* 们的游戏，他们最终会吃掉你。"
- Joel："不参与是另一种被吃掉。"
- 两人都**爱**对方——这种代际矛盾**没有解决方案**——这是 06 红线 4 的执行。

## 四、重大事件

### Acto III 末或 Acto V 一次抗议
- 一次大学抗议——Joel 在场——他**作为研究员观察**——但他**也参与**。
- 警察暴力镇压——Joel **被短暂带走**（拘留 6 小时后释放）。
- Saulito 在 La Casa Cobre **看到这则新闻** —— 玩家选择：
  - 不干预 → 中性，原住民 −5（他们**不期待**总统帮——但每一次"不干预"都累积）
  - 派人保释 → 原住民 +10，但寡头 *Sospecha* +5
  - 公开声援 → 原住民 +大，街头 +10，但寡头 −大

### Acto V（铜矿土地议题）
- Joel 在 *Voz Veladuria* 发布一篇调查文章——指出 *Norhem Copper* 在 Cumbre Mayor 的某矿区**正在通过"环境评估"程序非法扩张**到原住民传统土地。
- 这与 Don Federico 的铜矿议程**直接冲突**。
- Saulito 选择是否将文章纳入内阁讨论——决定 Joel 路线深度。

### Acto VI（**滑坡事件后续**）
- 如 Saulito 路线触发了山区滑坡（Aurora 那场预言）—— Joel **公开质问** Saulito。
- 他**不在 Saulito 面前**——他在一次新闻发布会上：
  - "*Los pueblos del Toro avisaron. Nadie escuchó. ¿Cuántos más?*"（"公牛之民提前通知过。没人听。还有多少？"）
- 这是 Joel 在剧情中的最高音量瞬间——**他唯一一次**公开提高音量。
- 隐性标记 `JOEL_FOUND_PRESS_HEARING` 触发——影响事件语气。

### 终章
- 视玩家路径：
  - 与 Joel 合作 → 终章里他可能成为下一届某种公共议程的青年代表
  - 拒绝 / 镇压 → 他**不死**——他**消失** —— 不公开但 Mocosos 圈知道（去欧洲做 PhD 第三年）—— 这是另一种"政治死亡"
  - True Ending Route → 他在最后一刻**点头一次**——Saulito 路过抗议人群时——这是 Joel 唯一一次承认 Saulito 这个具体的人

## 五、写作红线

1. **他不是浪漫化的"觉醒原住民青年"** —— 他**有焦虑、有犬儒、有自我怀疑**。**严守**。
2. **他与祖母的矛盾不解决** —— 写作时让两人**互相爱但各自坚持**——这是 06 红线 4。
3. **他不接圣灵集团** —— **禁止**让他与 El Padrino 的女儿 / 中间人有任何合作。
4. **他不公开自称政治领袖** —— 这是他对 Aurora 教导的妥协——**严守**。
5. **他与 Mocosos 圈的"边缘"关系** —— 他**不是 Mocosos**——他**比他们大、比他们政治化、比他们悲伤**。**严守这种张力**。
6. **本族语 cussing 但西语政治讨论** —— 这是他内部分裂的语言体现。**写作时偶尔保留**这种切换。
7. **他不死** —— 参动物 / 子女政策。**他可能消失**（去欧洲读 PhD）—— 这是政治死亡的克制版本。
