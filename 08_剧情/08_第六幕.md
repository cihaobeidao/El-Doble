# 08·08 · 第六幕 *El Espejo se Rompe*（镜碎）

> **戏剧问题**：至少一方已经识破了你
> **剧情天数**：第 61–73 天
> **玩家时长**：约 45 分钟
> **节奏**：所有伏笔开始爆。从前几幕的"暗中怀疑"被显式化——揭穿事件、政变倒计时、Susana 的告白、可能的 *La Locura* 发疯。这是本作戏剧密度最高的一幕。

---

## 一、第六幕功能

1. **揭穿事件链**：根据玩家累积的派系 *Sospecha*，**至少一方主动行动**——将"我们知道你是替身"摆上桌。
2. **历史伏笔回收**：Vega 的真实计划、Susana 的过去、Mariana 的纸条（多周目）、圣灵集团的"善意提醒"。
3. **政变倒计时**：青年派与老派军方互相试探发动政变，玩家成为筹码。
4. **可能触发**：Cordura 0 → *La Locura* 发疯事件（详见 [05_多周目机制.md](../05_多周目机制.md) §四）。
5. **第七天选择终局路线**：玩家此时决定如何收场。

---

## 二、关键事件

### ACTO_VI_A_001 · 揭穿事件链入口

**时刻**：第 61–63 天（视前面派系 *Sospecha* 谁先到 80）
**类型**：Audiencia / 强制
**时辰**：视事件

**触发逻辑**：
- 任一派系 *Sospecha* ≥ 80 → 该派系发起"揭穿"事件
- 没有派系到 80 → 触发"北方使馆主动揭穿"（默认路线，因他们起始 70）

**典型变体**：

#### 变体 A：北方使馆"善意的揭穿"

**地点**：北方使馆茶会（Whitfield 邀请）

**剧情骨架**：

- Whitfield 把一只档案夹推到桌上。
- "先生，我想让您看一份东西。这是我们的人三个月前写的报告——里面有 Saúl Tobar 先生的所有公开记录、医疗记录、甚至他在 2014 年某次婚礼上演讲的录音。我们一直没向上呈送。"
- 她递给你一杯茶："我相信我们可以以朋友的方式继续合作。"

**玩家选项**：

| 选项 | 后果 |
|---|---|
| A 全盘配合（"您要我做什么？"） | 北方 → "同盟"（如果之前不是）；Cordura −20；隐性 `BECAME_NORTH_PUPPET` |
| B 反咬（如果有 Joaquín 牌或类似筹码） | V 检定 ≥ 65；成功：Whitfield 退让一步；失败：使馆官方揭穿你 |
| C "您不会公开。我赌您不会。" | **极高难度** — V 检定 ≥ 75；成功：Whitfield 第一次失态；失败：游戏即将结束 |
| D （**`KEPT_SOUTHERN_INSURANCE` 已设置**）"我有别的退路。" | Whitfield 沉默；事件结束；触发南方接应路线 |

#### 变体 B：寡头家族"强行接管"

**触发**：Doce *Sospecha* ≥ 80（多次拒绝联姻 + 触发 Joaquín 牌后）

**剧情骨架**：

- Doña Isabel 不预约就来 La Casa Cobre。她不戴手套——这是宣战的姿态。
- "先生——或者我现在该叫您 Saúl。我们的家族**已经知道**。我们也知道 Vega 是怎么把您送进来的。"
- 她的条件比北方使馆更直接——**完全交出执政权**，让寡头集团代理"过渡内阁"，玩家继续"挂名"做总统。

#### 变体 C：圣灵集团回收魔鬼合同

**触发**：`DEMONIC_PACT_CARTEL` 已设置

**剧情骨架**：

- *Don Marcial* 第三次出现。这次他不带礼物。他递给你一张纸——上面是一个名字 + 一个地址。
- "Patrón，我们当初帮过您。今晚 21:00 之前，我们需要这个人**消失**。地址在这里。我们都是兄弟。"
- 名字可能是：
  - Beatriz Solano（如果玩家通过她获得 Joaquín 牌）
  - 一位北方反毒署线人
  - 一位青年派军官（圣灵集团的报复对象）
  - **或** 一位玩家完全不认识的"一般人"——这是最重的版本

**玩家选项**：

| 选项 | 后果 |
|---|---|
| A 配合 | 此人死亡；圣灵集团 → "同盟"；街头 −20 永久；Cordura −40；隐性 `EXECUTED_CARTEL_ORDER` |
| B 拒绝 | 圣灵集团 −30；他们威胁公开你的身份；隐性 `BROKE_CARTEL_PACT` |
| C 通过 Vega 反扑（**`KNOWS_VEGA_DIARY` 已设置**） | V 检定 ≥ 70；成功：Vega 帮你扼杀圣灵集团这条命令；失败：双方都视你为敌 |
| D 警告对方（用 Estela 私下递信） | Cordura +10（你保住了灵魂）；Estela 的渠道被烧毁（隐性 `BURNED_ESTELA_CHANNEL`） |

#### 变体 D：综合揭穿

**触发**：多个派系 *Sospecha* ≥ 70 但都未到 80

**剧情骨架**：

- 一份联合通牒——通过 Vega 转交。所有派系都"知道"了，但都不愿先动手。
- 玩家此时面对的是 **"协调式接管"**——所有派系联合提出："总统先生，您现在退位、或者我们公开揭穿。"

### ACTO_VI_I_001 · Mateo Vega 的崩溃

**时刻**：第 65 天 23:30（深夜事件）
**类型**：Íntimo
**时辰**：1（熬夜）

**剧情骨架**：

- Vega 来你卧室。他从未在这个时间出现过。他坐下，倒了一杯朗姆。
- "Patrón——Saúl——我得告诉您。我从一开始就在用您。我以为这次行动能让国家重新清醒——但事情走到现在，我意识到我不过是另一个 Castaño。"
- 他给你看一份文件——他的"2026 备忘录"政变草案。它已经被起草到第 47 页。
- "明天早上我可以发动它。我可以把全国军权拿在手里。但我不知道我该不该做。"

**玩家选项**：

| 选项 | 后果 |
|---|---|
| A 鼓励他："您是这个国家的希望。" | 隐性 `ENCOURAGED_VEGA_COUP`；后期可能触发"军政府接管"结局 |
| B 阻止他："您不能成为另一个 Castaño。" | 隐性 `STOPPED_VEGA_COUP`；Vega 隐性轴 +30；他撕碎备忘录 |
| C 直接说："我不在乎您怎么做。我只想三个月后离开。" | Vega 失望地走开；隐性 `INDIFFERENT_TO_VEGA`；Vega 后续与你疏远 |
| D （**`MENTIONED_LUCIA_TO_VEGA` 已设置**）"Vega，您还记得序幕那晚我跟您说我女儿吗？我现在请您——不要让国家变成她不能去的地方。" | 隐性 `INVOKED_LUCIA_TO_VEGA`；Vega 沉默 5 分钟；他撕碎备忘录；Vega 隐性轴 +50；终章 Vega 路线大幅推进 |

> **写作要点**：选项 D 是序幕标记 `MENTIONED_LUCIA_TO_VEGA` 的"回收"——一个看似无关紧要的开场标记在 60 天后变成关键变量。**这是本作"隐性标记 = 戏剧记忆"的最佳示范**。

### ACTO_VI_I_002 · Susana 的告白

**时刻**：第 67 天 凌晨 02:00
**类型**：Íntimo / 强制
**时辰**：1（熬夜）
**地点**：La Casa Cobre 化妆室

**剧情骨架**：

- 你不睡觉。你去化妆室——Susana 在。她比你还早。
- 化妆台上摆着一张老照片——一张黑白照片，照片里是一个抱着婴儿的年轻男人。背景是 Las Veladas 的某条街。
- Susana 不解释。她只说："Patrón——Saúl——这张是 1979 年。这个抱着婴儿的男人，叫 Eladio Tobar。这个婴儿是您。"
- **她告诉你你的父亲是谁**——你父亲是第一任替身（详见 99 文档）。
- "我那时候 16 岁。我抱过您。我听过您母亲哭。"

**玩家选项**：

| 选项 | 后果 |
|---|---|
| A 沉默——你不知道说什么 | 隐性 `LEARNED_FATHER_FROM_SUSANA`；Cordura −15（信息冲击）；下一周目自动获得 `KNOWS_SAUL_FATHER_WAS_DOUBLE` |
| B "为什么是现在告诉我？" | Susana：*"因为再不告诉，您就要变成他了。"*；隐性 + Cordura −10 |
| C "我父亲死了？怎么死的？" | Susana 摇头；"不该在今晚说"；隐性 `WANTS_TO_KNOW_FATHER_DEATH`（多周目可解锁更多） |
| D （**`KNOWS_GABRIELA_PUPPETMASTER` 已设置**）"Gabriela 也参与了，对吗？" | Susana 第一次惊讶；隐性 `EXPOSED_GABRIELA_TO_SUSANA`；多周目第五真相揭示加速 |

> **首周目大多数玩家会停在选项 A——这本身已经是震撼**。多周目玩家通过选 D 推进真相揭示。

### ACTO_VI_C_001 · 政变倒计时

**时刻**：第 70 天 06:00（如果 Vega 路线 → 启动；如果选 ENCOURAGED_VEGA_COUP）
**类型**：Crisis
**时辰**：8（一日全天 + 跨夜）

**剧情骨架**：

- 如果触发：清晨 Vega 的青年派军官在首都各关键节点部署。Castaño 老派同时反应——双方在 La Casa Cobre 周围对峙。
- 玩家在风暴中心——是被卷入军政府接管，还是亲赴前线协调？

**主要选项**：

| 选项 | 后果 |
|---|---|
| A 默许 Vega 接管 | 触发"军政府"结局变体 |
| B 上电视讲话试图调停 | 玩家伪装度大考；成功率视前几幕的 *Verosimilitud* + 街头好感；失败：流血政变 |
| C 通过北方 / 南方使馆请求外部介入 | 触发"外部接管"结局变体 |
| D 直接逃亡（如果 `KEPT_SOUTHERN_INSURANCE` 设置） | 触发 Vasconia 庇护结局 |

### SPECIAL_LOCURA · La Locura 发疯事件（可能触发）

**触发**：Cordura 跌至 0
**类型**：Crisis（最高优先级）

**详细机制见** [05_多周目机制.md](../05_多周目机制.md) §四 + [02_序幕.md](02_序幕.md) 中关于内心独白系统的设计。

**第六幕 *La Locura* 发疯地点（视玩家正在做什么）**：

- **内阁会议中**：你站起来。所有人停下。
- **国宴上**：你拿起酒杯，往天花板上扔。
- **卧室独自**：你看着镜子，**镜子里的人不是你**。
- **演讲台上**：麦克风开着。你说出第一句话。
- **化妆室与 Susana**：你看着她，叫出她的名字——但你叫的是 19 岁那个 Susana 的名字（隐性多周目变体）

**三种选择**（详见 05 文档）：
- **A 把它说出来——全部** → 触发 *El Discurso Loco* 结局
- **B 把它咽回去——用一个荒诞的笑话遮掩** → 隐性 `SWALLOWED_THE_TRUTH`、Cordura 强制重置 15、Verosimilitud −30
- **C 不说话——离开房间** → 隐性 `SUSANA_KNOWS_THE_BREAK`、Cordura 强制重置 25、Verosimilitud −10

### ★ ACTO_VI_D_001 · 选择终局路线（幕终）

**时刻**：第 73 天 18:00
**类型**：Decreto / 强制（幕终）
**时辰**：4

**剧情骨架**：

- 第六幕末，玩家面临**终章前的最后选择**。Vega 与 Susana 一起来到办公室。Susana 把一份合同放在桌上——这是当初序幕里的"三个月后接您离开"的计划。
- "Patrón，三个月到了。我们可以按原计划接您出去。但是，您也可以选择另一条路。"

**主要选项**（这是终章入口）：

| 选项 | 终章路线 |
|---|---|
| A 按原计划离开（"撤幕"路线） | 玩家试图与母亲、女儿一起去南方。**成败视前置标记**——如果 `BURNED_ESTELA_CHANNEL` 或 `SIDED_SOUTH` 强烈，路线被破坏 |
| B 留下来，公开身份（"广场"路线） | 玩家在 75 天的全国直播上自首。结果视街头与教会、Cardenal Soto 隐性轴 |
| C 留下来，继续演（"演出继续"路线） | 玩家选择让谎言延续。视前置真相标记决定多周目结局变体 |
| D 公开真相但不公开身份（"半揭露"路线） | 玩家在 74 天某场演讲中"暗示"——但不直接承认；这条路触发的结局取决于多个派系的反应 |
| E （**多周目 + `LA_VERDAD_SPOKEN` 满足条件**）走"找到真相"路线 | 触发 *Las Veladas* 真结局——详见 [09_终章与结局.md](09_终章与结局.md) |

**衔接**：选项决定终章具体路径。

→ 进入终章。Acto 间过场："*第七十四天，黎明。Susana 把您的西装挂在化妆室门后。她不说话。您看着窗外——天还没亮，但有一只鸟开始叫。*"
