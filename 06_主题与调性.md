# 06 · 主题与调性

> 这份文档不是规则——是**创作的内核**。任何一个事件、一句台词、一个数值变化，回头都可以问一句："这与我们的主题相干吗？"

---

## 一、本作想让玩家思考的命题

### 核心命题（贯穿全作的一句话）

> **当一个体制需要任何人都可以替代，那么它真正在替代的，是什么？**

### 三个支线命题

1. **"演员与位置"**——一个人坐到那个位子上，究竟是因为他是他，还是因为他穿着那身西装？这是 Saulito 全部内心戏的根。
2. **"小人物的良心"**——当你没有受过政治训练、没有理论、没有理想，仅凭街头本能去做决定，你会做出比专业政客更坏的决定，还是更好的？这是本作对"政治素人"的辩证态度。
3. **"国家是否值得"**——拉美黑色幽默的最深问题。维拉杜里亚是值得拯救的吗？如果不值得，玩家此刻还在做什么？

### 我们**不**想做的命题

- 不做"民主 vs 独裁"的二元判断剧。本作里没有任何派系是"对的"。
- 不做"小人物逆袭成英雄"的爽文。Saulito 的弧线不是英雄之路，是**良知与现实的反复磨损**。
- 不做"政治阴谋揭露真相后世界变好了"的天真叙事。**揭露真相在本作里几乎从来不会让事情更好**。

## 二、叙事原则（写作时的硬规则）

### 原则 1 · 三种声音的分层

每段文本必须明确属于以下三种声音之一：

| 声音 | 视角 | 写作风格 | 出现场景 |
|---|---|---|---|
| **场景叙述者**（第三人称） | 客观、冷静、镜头感 | 短句、画面优先、不评判 | 事件描述主体、转场 |
| **Saulito 内心独白**（第二人称、斜体） | 主观、私密 | 短促、口语、偶尔粗口、避免政治术语 | 关键抉择前、夜晚私人时段 |
| **新闻 / 报纸 / 广播**（拟文体） | 公共话语 | 模仿真实媒体语气、可以是寡头报纸的温文尔雅，也可以是 Radio Aurora 的尖锐 | 章节序场、事件背景 |

**禁止把三种声音混在同一段里。**

### 原则 2 · 拉美式的"魔幻现实主义"用得克制

- 允许：占卜婆婆的预言准了一次；圣母像在大雨中无故掉头；某位将军临终前看见小时候的猫坐在床尾。
- 禁止：直接的超自然事件直接改写情节（圣母不能让数值变化，预言也不能解锁结局）。
- **魔幻现实主义在本作里是质感，不是机制。**

### 原则 3 · 黑色幽默不能"贴脸笑"

- 黑色幽默来自**情境本身的荒谬**，不是来自俏皮的旁白。
- 例（好）：替身在国宴上被问"总统先生，您今晚的酒选得真好"，他面无表情地说"是啊，我亲自选的"，而玩家知道——这酒是他从总统的一个秘密酒柜里随便拿的，一杯就值他过去半年的房租。
- 例（坏）：旁白突然冒一句"哈哈，多么讽刺啊！"——**永远不要这样写**。

### 原则 4 · 派系的"道德平面"必须是真的复杂

每个主要 NPC 至少要有：
- 一个让玩家**真心想认同**他的瞬间。
- 一个让玩家**真心讨厌**他的瞬间。
- 一个让玩家**意识到自己看错了**的瞬间。

> 即使是 Doña Isabel（虔诚天主教 + 寡头发言人）这样在政治立场上"站在玩家对立面"的人——也要写出她在某一刻真心心碎的样子。

### 原则 5 · 反复让玩家在"不知道"中做决定

- 大多数关键决策时，玩家**不应该有完整信息**。这与多周目档案系统呼应——你越是渴望知道更多，你就越要再玩一次。
- 这是 Suzerain / 苏丹的游戏 / 布兰特爵士共同的"压迫式信息匮乏"风格。

### 原则 6 · 死亡、暴力、性的处理

- **死亡**：可以发生，但镜头是侧面的。**不写直接的尸体特写**。一个母亲坐在派出所门口的水泥地上是死亡描写；尸体上的伤口不是。
- **暴力**：节制。一场暴动可以暗示数百人受伤，但不要把镜头钉在血上。
- **性**：本作中是政治性的（权力关系）远多于身体性的。第一夫人 Gabi 与某使馆高官的旧交、Saulito 与一个老情人的回忆——可以涉及，但**不写情色细节**。
- **与儿童相关的暴力**：完全不写。可以暗示（如某次空袭中"那条小学的旗杆倒了"），不直接描述。

### 原则 7 · 拉美口语的真实感

- 适度使用西语词（Patrón / Compadre / Hermano / Mira / Carajo / Hostia）。
- 中文翻译时**保留西语原词**作为氛围词，旁加翻译，不全译。
- 慎用网络梗、Z 世代用语（除非是 *Los Mocosos* 视角）。
- 街头人物的语言要"脏一点 + 真一点"，寡头的语言要"洁净 + 假一点"——用语言区分阶级。

## 三、调性参考清单

### 文学（核心质感来源）

- **加西亚·马尔克斯**《族长的秋天》《迷宫中的将军》— 衰老独裁者、孤独的权力、魔幻现实主义。**这是本作美学第一锚点**。
- **巴尔加斯·略萨**《公羊的节日》— 多视角叙事 + 多米尼加独裁者特鲁希略时代。
- **罗贝托·波拉尼奥**《2666》— 暴力、不可知、官僚气息、街头与精英的隔阂。
- **胡安·鲁尔福**《佩德罗·巴拉莫》— 死人与活人的边界。"失踪总统"线索的氛围参考。
- **博尔赫斯**《阿莱夫》《通天塔图书馆》— 档案、镜子、替身——这些母题的源头。
- **石黑一雄**《长日将尽》— 一个一辈子在替别人服务的人，开始反思自己的服务对象是谁。Saulito 后期的内心结构。

### 电影 / 电视（视听风格）

- **《巴西》(Brazil, 1985)** — 官僚的荒诞 + 黑色幽默。
- **《大独裁者》(1940)** — 替身设定的祖先。**本作的元设定致敬**。
- **《地下》(Underground, 1995)** — 国家在战争里溃烂的喧嚣感。
- **《Roma》(2018)** — 拉美城市质感、寡头家庭的内部肌理。
- **《Narcos》（剧集）** — 贩毒集团的复杂呈现。
- **《The Death of Stalin》(2017)** — 黑色幽默 + 政治荒诞**的喜剧上限**——本作可以借鉴的台词节奏。
- **《Tropical Malady》(2004)** — 热带魔幻的边缘氛围。

### 游戏（机制 + 叙事可参考）

- **Suzerain (2020)** — 数值轴 + 派系关系 + 政治抉择的工程范本。**结构最直接的参考**。
- **The Life and Suffering of Sir Brante (2021)** — 章节式叙事 + 隐性标记 + "你以为自己在做选择"的压抑感。
- **The Sultan's Game**（苏丹的游戏）— 资源管理 + 短局 + 多周目暗线的节奏。
- **Pentiment (2022)** — 历史质感 + 多周目 + 对 NPC 的复杂书写。**"角色复杂性"的参考标杆**。
- **Disco Elysium** — 内心独白系统、多重人格化的内心声音。**Saulito 的内心独白系统是从这里学的**。
- **Citizen Sleeper (2022)** — 资源/状态/事件混合，节奏紧凑。
- **Reigns / Reigns: Her Majesty** — 派系平衡的极简形式。**反例之一**——本作不要这种黑白二选式抉择。

### 调性禁区（**反例**）

- **不要"沙漠 + 仙人掌 + 墨西哥草帽"刻板印象**。维拉杜里亚是雨林、铜山、加勒比海港。
- **不要把贩毒集团写成简单的反派恶徒**。他们必须有真正的社会基础。
- **不要把美国式干预写成单一的"邪恶帝国"**。北方共和国内部要有真正的分歧。
- **不要让宗教只是"背景元素"**。天主教在本作里是有政治权重的真实力量。
- **不要让"穷人"成为统一的同情符号**。穷人也有内部派系。

## 四、写作工艺细节

### 章节标题的命名风格

- 西语为主，括号附中文。例：*El Espejo*（镜子）、*La Embajada Sonríe*（使馆微笑）。
- 标题应是隐喻、不剧透。

### 事件标题的命名风格

- 短句 + 名词为主。例："白手套"、"凌晨四点的电话"、"那条狗为什么不叫"。
- 避免"决定 X"、"选择 Y"这种功能性命名。

### 数值变化的呈现

- UI 上不显示精确数字（除 Tesoro 之外）。
- 用文字反馈："工人对你失望了。" / "Vega 上校点了点头。" / "Doña Isabel 戴着白手套的手离开了你的手。"
- 玩家在数据屏（按 Tab 调出）能看到精确数值。**主叙事流中只看氛围**。

### 复盘 / 结局的处理

- 结局后的档案室用**冷峻、客观、报告式**的语言。
- 不写"恭喜你完成了！"——写"维拉杜里亚联邦共和国在 X 月 Y 日进入下一个阶段。以下是 75 天后的几张照片。"

## 五、致敬段落（设计灵感地图）

> 提醒哪些是核心借鉴、哪些是必须守住的差异化。

| 来源 | 借鉴 | 我们要避免的 |
|---|---|---|
| Suzerain | 数值 + 派系 + 长会议场面 | 它过于"严肃"，我们要更黑色幽默 |
| Sir Brante | 章节制 + 隐性标记 + 多结局 | 它有家族多代叙事，我们是单代多替身 |
| 苏丹的游戏 | 紧凑单局 + 资源管理 + 多周目暗线 | 它的随机性偏大，我们要更"剧本化" |
| 马尔克斯 | 拉美质感 + 衰老的权力 + 魔幻现实主义 | 我们不能写成纯文学游戏，必须保有玩家互动的紧张 |
| Disco Elysium | 内心独白系统 | 它是单角色 RPG，我们是多替身的不同声音 |
| Pentiment | NPC 的复杂性 + 多周目设计 | 它是侦探剧，我们是政治剧 |

## 六、敏感话题红线

> 本作是讽刺剧 + 政治悲喜剧。讽刺需要敢踩，但**踩什么、不踩什么**得提前画线。

### 红线 1 · 真实国家不直接对应

- 维拉杜里亚是**复合体**——它的某段历史像智利、某段经济像阿根廷、某个总统像哥伦比亚某届、某个寡头家族像危地马拉某姓。**但任何单一对应都是错的**。
- **写作时的硬测试**：写完一段，问自己"如果一个智利读者来读，他会觉得这是在写智利吗？" 如果答案是"是"，请加一些**与智利完全不符的细节**进去。
- 北方共和国 *El Norte* 与南方共和国 *Vasconia* 同理——**不要让任何一国可以被一对一识别**。

### 红线 2 · 真实人物不直接出现

- 任何在世或近现代的真实政治人物，**不出现姓名**。
- 历史人物（如 Bolívar、Allende、Castro）可以**作为壁画 / 课本 / 街头绰号**间接出现，**但不发表观点**。
- **绝对禁止**：将虚构 NPC 的某段台词或行为，写成一句已知真实人物名言的拙劣模仿。

### 红线 3 · 真实当代事件不直接复刻

- "结构借鉴"可以——例如某次债务危机、某次毒品战争、某次外国干预的**叙事结构**可以借。
- "细节复刻"不行——具体的人名、地名、年份、伤亡数字不能照搬。
- **写作时的硬测试**：维基百科一段话能不能直接对应你这段剧情？如果能，**改细节**。

### 红线 4 · 政治立场对称原则

> 本作的讽刺指向**所有**政治派别。任何一边都不能被写成单纯的英雄或单纯的反派。

- 寡头家族里要有真心心碎的瞬间（如 Mariana）。
- 军方里要有真心想救国家的人（如 Vega）。
- 贩毒集团里要有真实的社会基础（如圣灵集团给孤儿院发面包是真的）。
- 北方使馆里要有真心相信"我们在帮"的人（如 Pierce）。
- 街头民众里要有**愚蠢且暴力的瞬间**（不要让"人民"变成天然正确的符号）。
- **检验法**：让一位虚构的"自由派友人"和一位虚构的"保守派友人"分别看你的剧本——他们都应该觉得"这游戏对我那边也太狠了点"。如果只有一边觉得不舒服，剧本就偏了。

#### 红蓝对照表（内部工具）

每个 Acto 写完后须填一份**红蓝对照表**——逐章列出对左 / 右两端的批评是否平衡。

**模板**：

```
─────────────────────────────────────────
 Acto: 第 X 幕《XXX》
 写作完成日期: YYYY-MM-DD
 主笔: ___
─────────────────────────────────────────
 对"左"端立场的批评（事件 / 台词 / 情节）：
   1. ___________________
   2. ___________________
   3. ___________________

 对"右"端立场的批评：
   1. ___________________
   2. ___________________
   3. ___________________

 对"中间派 / 现实主义" 的批评：
   1. ___________________

 平衡评估（主笔自评）：
   □ 大致平衡 — 两端都被刺到
   □ 偏左 — 对右端不够狠
   □ 偏右 — 对左端不够狠
   □ 真空 — 这一幕没碰到政治派别（也是合法的）

 主笔的反思（200 字以内）：
   ___________________
─────────────────────────────────────────
```

**工艺纪律**：
- 每写完一个 Acto，主笔自查一次。
- 每写完两个 Acto，**主笔互查**——彼此看对方的对照表。
- 每写完一个完整的"周目剧情"，整体过一次"全作总账"。
- **填表的过程比表本身重要**——它强迫主笔重新审视自己刚写下的字。

**实际执行参考**：详见 99_秘密文档.md 的"红蓝对照表 · 主版"——含左 / 右 / 中间派的具体定义与首位填写示例。

> **注意**：这份对照表是**内部审稿工具**，**永远不出现在游戏内**。

### 红线 5 · 种族 / 宗教 / 性别处理

- **原住民**（*los pueblos del Toro*）不应被写成纯粹的受害者符号——他们有自己的政治、有自己的派系、有自己的内部分歧。**任何"高贵的野蛮人"式呈现都要重写**。详细派系小传见 01 文档"次要群体"小节，含 Aurora / Joel / Maestro Quispe 三位 NPC。
- **非裔后裔**（沿岸港口居民）同理——避免落入"音乐家 + 罪犯"这类双重刻板印象。
- **天主教会**是有政治权重的真实力量——不要把所有神职人员写成虚伪的反派。Cardenal Soto 的良心拷问要让玩家**真的不舒服**。
- **女性 NPC** 不能被写成只服务于"男主角的成长"。Mariana、Gabriela、Catalina、Doña Isabel、Susana——每个人都要有自己独立的目标。
- **性少数**：不在主线戏剧机器里，但若出现（次要 NPC），**不写成奇观**，按普通人写。
- **儿童**：可以出现在背景中，**不直接呈现暴力对儿童的施加**。

### 红线 6 · 黑色幽默的"笑话边界"

> 黑色幽默来自**情境本身**，不来自**对受害者的嘲笑**。

- **可以笑**：体制的荒谬、寡头的虚伪、使馆的偏执、媒体的浅薄、Saulito 自己的窘迫、政治术语的陈词滥调。
- **不能笑**：具体死者的痛苦、特定群体的苦难、性暴力、宗教虔诚本身（可以笑虚伪的虔诚，不能笑真心的虔诚）、儿童任何形式的受害。
- **检验法**：把笑话的最后一句单独拎出来读——它是在笑"系统"还是在笑"人"？笑系统过关，笑人重写。

### 红线 7 · 关于影射"现实玩家熟悉的"政治记忆

- 玩家会**自然地**把维拉杜里亚映射到他们熟悉的某个真实国家。**这不是 bug 是 feature**——但我们不主动诱导。
- 写作时**不主动制造"这就是 X 国"的指认快感**。我们不要那种"哈哈，作者懂！"的廉价快感——它会让游戏退化成一个迷因合集。
- 优先做"普世性的政治剧"——一个**任何拉美国家、任何后殖民国家、任何小国**的玩家都会觉得"这就是我们的故事"。
