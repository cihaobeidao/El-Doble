# 08·06 · 第四幕 *El Padrino Toca a la Puerta*（教父敲门）

> **戏剧问题**：圣灵集团送来"礼物" + 寡头联姻线推开
> **剧情天数**：第 33–47 天
> **玩家时长**：约 40 分钟
> **节奏**：从街头危机回归精英压榨。所有派系开始**真实地**索取——不再只是试探，而是要东西。这一幕的幕终（国庆游行 + Mariposa）是第一次"灵魂代价"的高峰。

---

## 一、第四幕功能

1. **圣灵集团正式登场**：白西装中间人 + 现金礼物 + 魔鬼合同的开端。
2. **联姻线引爆**：Doña Isabel 正式提议"总统离婚 Gabriela、改娶 Mariana"。
3. **Joaquín 牌时机**：玩家此时通常已收集到至少一条获取路径，是打牌的最佳节点。
4. **Mariposa 重逢**：第四幕末国庆游行——本作最重的"灵魂代价"瞬间。

---

## 二、关键事件

### ACTO_IV_A_001 · 白西装中间人送礼

**时刻**：第 34 天 22:30（深夜熬夜事件）
**类型**：Audiencia
**时辰**：1（如果在卧室接待）/ 0（如果不见）

**剧情骨架**：

- 一位穿白西装、戴白礼帽的中年男人 — *Don Marcial* — 通过一条"非官方但畅通无阻"的渠道进入 La Casa Cobre。
- Estela 没拦下他（圣灵集团的人脉渗透到了她）。
- Marcial 把一只手提箱放在你桌上，里面是 5,000 北元现金（≈ Saulito 三年总收入）+ 一封手写信。
- 信的内容简短："Patrón，我们都是兄弟。— *El Padrino*"

**主要选项**：

| 选项 | 后果 |
|---|---|
| A 收下现金 | 圣灵集团 +15；隐性 `ACCEPTED_CARTEL_GIFT`；多周目记录"礼物清单" |
| B 退还现金，留下信 | 圣灵集团 −5（小损失）；隐性 `REJECTED_CARTEL_GIFT_KEPT_LETTER` |
| C 全部退还 | 圣灵集团 −15；隐性 `REJECTED_CARTEL_GIFT` |
| D 收下现金，立刻让 Estela 把它捐到 Lucía 在的护理学院 | 圣灵集团 +5（他们欣赏"漂亮"的处理）；街头 +10；隐性 `LAUNDERED_CARTEL_GIFT_FOR_LUCIA` |

> **写作要点**：D 选项是 Saulito 的人物魅力点 — 他懂街头规则。圣灵集团送钱是"建立关系"的姿态，玩家把钱"洗"成对女儿的爱也是一种姿态。**他们会笑——他们没生气**。

### ★ ACTO_IV_A_002 · 联姻提议

**时刻**：第 38 天 11:00
**类型**：Audiencia / 强制
**时辰**：3
**地点**：La Casa Cobre 总统办公室

**剧情骨架**：

- Doña Isabel 来访。这次她不戴白手套——她戴着一对祖母绿耳坠（与 Mariana 的同款，**家族标识**）。
- 她直接提议："总统先生——为了国家的稳定，您应该考虑离婚 Gabriela，改娶 Mariana。"
- 她列出三个理由：
  1. Gabriela 健康欠佳（**多次缺席公开活动**——她在为玩家辩解 Gabi 的缺席）
  2. 维拉杜里亚保守阶层需要"看到一个新的家庭形象"
  3. Mariana 已经同意（**这是真的，但 Mariana 同意的动机是另一回事——详见 99 文档**）

**玩家选项**：

| 选项 | 后果 |
|---|---|
| A 答应 | Doce +30；街头 −25；Cordura −20；隐性 `MARRIAGE_ACCEPTED`；Mariana 第二次见面（`ACTO_IV_I_001`）变成"婚约谈判"而非"私人交流" |
| B 拒绝 | Doce −20；Mariana 个人轴 +20（她欣赏你拒绝）；街头 +10；隐性 `MARRIAGE_REJECTED` |
| C 推迟（"我需要时间想想"） | Tesoro −2；Doce 0；隐性 `MARRIAGE_DELAYED`；下一幕重提 |
| D 用 **Joaquín 牌**（如果已获得） | Doce 噤声；联姻线推迟一幕；Doce *Sospecha* +30；隐性 `USED_JOAQUIN_CARD` + `MARRIAGE_DELAYED_BY_BLACKMAIL` |
| E （**Mariana 隐性轴 ≥ 20**）"我先与 Mariana 单独谈谈" | Doce 0；Mariana 个人轴 +10；解锁 `ACTO_IV_I_001` 变体（两个被推上去的人在房间里） |

> **写作要点**：选项 A 看似"政治正确的妥协"，但代价极重 — Cordura −20、街头 −25。它是表面解决方案，灵魂代价巨大。玩家有时会以为 A 是"安全选项"，**实际它是本作里最危险的选项之一**。

### ACTO_IV_A_003 · 圣灵集团的魔鬼合同

**时刻**：第 40 天 14:00（条件触发：玩家正在通过路径 C 获取 Joaquín 牌）
**类型**：Audiencia
**时辰**：2

**剧情骨架**：

- 如果玩家选择通过圣灵集团获取 Joaquín 车祸丑闻档案——*Don Marcial* 再次出现，这次直接把档案放在桌上。
- "Patrón，我们办事的人也是有良心的。这份档案给您。我们什么都不要。"
- 但他在离开前说："Patrón，您记得，我们以后会回来感谢您的。"

**玩家选项**：

| 选项 | 后果 |
|---|---|
| A 接受档案 | 获得 Joaquín 牌；隐性 `DEMONIC_PACT_CARTEL`；圣灵集团 *Sospecha* −5（他们建立信任）；**第六幕集团会"善意提醒"——必触发** |
| B 拒绝接受 | Joaquín 牌从此线断绝；圣灵集团 −10；隐性 `REJECTED_CARTEL_DOSSIER` |
| C 接受档案，但当场把它撕掉给他看："谢谢，但我不需要这个。" | **极高难度** — V 检定 ≥ 70；成功：圣灵集团 +20（他们欣赏强人姿态）；失败：圣灵集团 −20、Sospecha +20 |

### ACTO_IV_AR_001 · Joaquín 牌的使用时机

**时刻**：玩家可主动选（约第 38–47 天间）
**类型**：Archivo / 玩家行动

详细机制见 [01_世界观与派系小传.md](../01_世界观与派系小传.md) "Joaquín 的车祸丑闻"小节。

**第四幕能打的点**：
- 联姻提议事件（`ACTO_IV_A_002` 选项 D）
- 铜矿议案二阶段（如果第二幕推迟过，第四幕重新提）
- 寡头家族的某场私下宴会
- 终章前的"自保"使用（推迟到第六幕也可）

### ACTO_IV_I_001 · 与 Mariana 的第二次见面

**时刻**：第 42 天 19:30（条件触发：选 `ACTO_IV_A_002` 选项 A 或 E）
**类型**：Íntimo
**时辰**：3
**地点**：La Casa Cobre 后花园 / 私人会客厅（视前置选项）

**剧情骨架（变体 1：选 A 联姻已答应）**：

- Mariana 来谈"婚事细节"。**她带着一份准备好的协议**——她已经预想过这一切。
- 她不假装高兴，也不假装难过。她递给你一支笔："总统先生，请您签——但请您先读完。"
- 协议里有一条：**婚后 Mariana 保留她的所有公开活动权 + 拒绝怀孕权**。她不是来当贤妻的，她是来执政的。

**玩家选项**：

| 选项 | 后果 |
|---|---|
| A 签字 | Mariana 个人轴 +30；Cordura +5（她让你松了口气——这不是真婚姻）；隐性 `READ_MARIANA_TERMS` |
| B 拒绝："这不是真婚姻——我不能签这个。" | Mariana 个人轴 +15；Cordura −10；隐性 `REJECTED_MARIANA_TERMS` |
| C （**Mariana 隐性轴 ≥ 25**）放下笔："Mariana, ¿quién es usted en realidad?" | 解锁特殊对话：**她第一次主动告诉你，她也是被推上去的**；隐性 `MARIANA_TOLD_TRUTH`；Mariana 个人轴 +50 |

**剧情骨架（变体 2：选 E 单独见面，未答应联姻）**：

- 你在后花园等她。她比预约时间早 10 分钟到。她不带笔，不带文件。
- 她直接说："您和我都不该在这里。我母亲推我出来，是因为她以为我会乖乖嫁过来。但我也只是个工具。"
- **这是 Mariana 路线最深的对话节点**。她第一次以"她自己"的身份与你说话。

> **写作要点**：变体 2 的对白要让玩家**感到震动**——这一刻 Mariana 不是"反派的女儿"或"政治资产"，她是一个被同样的系统困住的人。Saulito 此时的内心独白可以很简短：*"原来我不是这房间里唯一被推上来的人。"*

## 三、★ ACTO_IV_P_007 · 国庆游行 / *La Perra de Las Veladas*（幕终）

> **完整剧本范本见 [07_风格样本.md](../07_风格样本.md) 样本 3**。

**时刻**：第 47 天 14:30（游行进入 Las Veladas 边缘）
**类型**：Público / Íntimo 混合 / 强制
**时辰**：6
**反应窗口**：30 秒（玩家选择窗口）

**剧情骨架**：

- 国庆游行队伍从 Plaza Bolívar 出发，沿 Avenida del Cobre 行进。坐敞篷礼车的玩家身边是 Vega（右）与 Doña Isabel（左）。
- 队伍进入 Las Veladas 边缘——Saulito 看见自己的酒吧、自己的公寓窗户。
- 一只狗从人群中冲出来——**Mariposa**。她长大了，左耳缺一块——是去年她跟 Doña Pilar 的猫打架。**她的眼睛认得你**。
- 一个穿白衬衫的少年（她现在的主人）追在后面：*"¡Mari! ¡Mari, vuelve!"*

**玩家选项（30 秒内必选）**：

| 选项 | 你做什么 | 后果 |
|---|---|---|
| A 装作完全不认识 | *Verosimilitud* +15；*Cordura* −20；隐性 `DENIED_MARIPOSA` |
| B 弯腰，对她说一句 | *Verosimilitud* −25；Doce *Sospecha* +20；Mariana 个人轴 +30；街头 +25；隐性 `KEPT_MARIPOSA` |
| C 让保安抱走她（嫌弃语气） | *Verosimilitud* +5；*Cordura* −30；街头 −10；隐性 `BETRAYED_MARIPOSA`（**永久心债**） |
| D （**V ≥ 80**）用真总统的嫌恶语气 | *Verosimilitud* +25；*Cordura* −40；圣灵集团 +10；街头 −20；隐性 `BECAME_HIM_FOR_A_SECOND` |

**衔接**：游行结束后玩家在化妆室卸妆。Susana 不说话——但她看见了你眼角的那滴东西。隐性 `SUSANA_SAW_YOUR_TEAR`。

→ 进入第五幕。Acto 间过场："*您今晚没有吃晚饭。Estela 端来一碗山区炖牛尾——真总统的标志性食物。您看着碗里的肉，半天没动叉子。*"
