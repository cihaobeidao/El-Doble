# 08·02 · 序幕 *La Noche del Cobre*（铜屋之夜）

> **戏剧问题**：你为什么答应？
> **剧情天数**：第 0 天（凌晨 4:17 至次日 8:00）
> **玩家时长**：约 20 分钟（含基础卡片教程）
> **核心机制**：**没有真选择**——序幕的所有"选项"通向同一结局，差别只在玩家以何姿态接受。
> **时辰系统**：序幕特例，**不消耗时辰**——时辰系统从第 1 天 8:00 起算。

---

## 一、序幕功能

序幕同时承担三个功能：

1. **角色介入**：让玩家成为 Saulito —— 不是数据卡片，是一个具体的、酗酒的、有母亲与女儿的中年男人。
2. **世界引入**：通过 Vega 与 Susana 的对话，把维拉杜里亚当下的危机端上桌。
3. **机制教学**：*El Expediente Báez* 双阶段档案系统的第一阶段（基础卡片 + 5 题考核）。

---

## 二、事件清单

### ★ PROL_C_001 · *La Acera del Cobre*（铜街人行道）

> 完整剧本范本见 [07_风格样本.md](../07_风格样本.md) 样本 1。

**时刻**：第 0 天 04:17
**地点**：Las Veladas / Avenida del Cobre
**类型**：Crisis（无真选项）
**强制**：是

**剧情骨架**：

- Saúl 走出 *El Gallo Borracho* 酒吧，刚演完三场没人多给小费的总统模仿秀。
- 黑色雪佛兰停在他面前。Coronel Mateo Vega 与化妆师 Susana Quintero 在车上。
- Vega 提供四万北元押金 + 三个月扮演真总统 Augusto Báez Marín 的合同。
- 当 Saúl 问"如果我说不"时，Vega 平静地复述他母亲门牌、女儿考试时间、汇款金额—— **威胁不需要言明**。

**三个"选项"（实际同一结局）**：

| 选项 | 表面意思 | 隐性后果 |
|---|---|---|
| A "把钱拿回去。我自己上车的。" | 保留尊严 | Cordura +5；标记 `WALKED_IN_DIGNIFIED` |
| B "……我女儿下周末过生日。" | 确认家人安全 | Cordura −5；标记 `MENTIONED_LUCIA_TO_VEGA` |
| C "合同里加一条——三个月后接我们走的人不是您。" | 谈判式接受 | Vega 青年派起始信任 +5；标记 `NEGOTIATED_EXIT_CLAUSE` |

**衔接**：直接 cut 到化妆室 → 触发 `PROL_AR_001`。

---

### ★ PROL_AR_001 · *El Expediente Báez*（Báez 档案）— 教程

**时刻**：第 0 天 06:08–07:30
**地点**：La Casa Cobre 地下化妆室
**类型**：Archivo / Tutorial
**强制**：是

**剧情骨架**：

- Susana 把一只磨损的牛皮文件夹放在桌上："Patrón，先记这些。等会儿我考您。"
- **阶段一：基础卡片**（4–6 页）— 玩家可反复翻阅。包含：
  - 总统姓名 / 出身 / 三件标志性细节（手表、口头禅、签字习惯）
  - 妻子 + 两位成年子女的关键事实
  - 三位最常见 NPC 的"应对开关"（Doña Isabel = 戴白手套、保守、念经；Vega = 短句不废话；Aldana = *Compadre* 但不要逾矩）
  - 三句口头禅 + 演说开关句
  - 一份"目前不太确定"列表（健康状况：传言不一，本档案不下结论）
- **置信度三标签**：✓ 确认 / ◇ 不确定 / ✗ 暂缺。**没有故意错误**。

**5 道考核题**（题库 15–20 题随机抽 5）：

> "您每天早晨第一杯喝什么？"
> "您父亲那块表是什么牌子？哪一年的？"
> "您今晚拒绝吃什么？"
> "您 1996 年送给夫人的纪念礼物是什么？"
> "您 2014 年关停的那家电台叫什么？"

**考核成绩 → 起始数值**：
- 5/5 → *Verosimilitud* +20
- 3–4/5 → *Verosimilitud* +10
- 1–2/5 → *Verosimilitud* +0
- 0/5 → *Verosimilitud* −10 + "露馅风险"标记

**陷阱埋伏**：错答的具体细节会在后续第 1–4 幕的对应事件中**轻微 +5 派系 Sospecha**——玩家不会被显式提示"这是陷阱"，但会感到"刚才好像哪里出错了"。**周目结束时**汇总展示。

**衔接**：考核结束 → Susana 帮你化妆 → 走廊上 Vega 简短交代第一场内阁会议 → 触发 `PROL_P_001`。

---

### ★ PROL_P_001 · 第一次内阁会议（凌晨简版）

**时刻**：第 0 天 07:45
**地点**：La Casa Cobre 内阁会议室
**类型**：Público
**强制**：是

**剧情骨架**：

- 这是序幕的最后一场。**这场会议不展开实质政治议题**——Vega 已经事先用"总统身体不适"说服内阁延后正式会议。
- 玩家在场上要做两件事：
  1. **签字**一份装样子的文件（无内容代价；但玩家此时正学习"签字" — 真总统签的 *Augusto H. Báez M.* 与 Saulito 的字迹有差距，由 Susana 提前演示过一次）
  2. **回应国防部长 Castaño 的一句问候**："Patrón，您的咳嗽好些了吗？"

**回应选项**：

| 选项 | 反馈 | 隐性 |
|---|---|---|
| A "好多了。Rómulo，谢谢。"（用对方教名） | Castaño 微微一愣（真总统从不用教名） | Junta-老派 Sospecha +5 |
| B "还好。"（最短回答） | 真总统标准回答 | 无 |
| C "您也保重。"（反向关切） | 转移焦点 | Junta-老派 +2 |

> 这场是教学："**最低风险 = 最少话**"。玩家在序幕末就被默默告诉：在维拉杜里亚做总统，少说话最安全。

**衔接**：会议结束 → Vega 把第一天的时刻表交给 Saulito → 进入第 1 天 8:00 早班简报 → 跳转 [03_第一幕.md](03_第一幕.md)。

## 三、序幕的关键标记总结

序幕至多设置以下标记（互斥项除外可累计）：

- `WALKED_IN_DIGNIFIED` / `MENTIONED_LUCIA_TO_VEGA` / `NEGOTIATED_EXIT_CLAUSE`（互斥三选一）
- 错答陷阱标记（不可见，由考核成绩决定）

## 四、与第一幕的衔接

序幕结束时玩家：
- *Verosimilitud*：基础（修正后约 0–20）
- *Cordura*：约 75（基础 80 ± 序幕选项的微调）
- *Sospecha*：按各派系起始值（详见 04_数值与阵营.md §三）
- *Tesoro*：基础值
- 持有：基础卡片（可随时翻阅）；正式档案 *尚未* 拿到

**进入第一幕的状态变量**：
- `EXPEDIENTE_TUTORIAL_COMPLETE`：自动设为 true
- 序幕考核分数被记录为 `TUTORIAL_SCORE_X`，影响第一幕初期事件难度

## 五、写作要点回顾

1. 序幕是**情感引入**而非"教程引导"——避免"现在请按 X 键学习"这种语气。所有教学融入剧情。
2. Vega 的形象在此节定型：**疲惫的、相信自己在做正确事情的中年军人**。不是反派、不是英雄，是"职业道德下的执行者"。
3. Susana 在此节是无脸的：她不眨眼、不评判、动作精准。**这是为后期她的"她抱过初生的你"伏笔留余地**——见 99 文档第 1.4 节。
4. 玩家应当在序幕结束时感觉到：**"我现在被关在一栋没有出口的房子里。"**
