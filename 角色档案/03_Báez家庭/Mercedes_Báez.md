# Mercedes Báez · 长女

> "我父亲不代表我。" —— 她公开说过。
> 这一句话的代价是她两年没回过家。

## 一、身份信息

| | |
|---|---|
| 全名 | Mercedes Soledad Báez Marqués |
| 年龄 | 24 |
| 出身 | 在 La Casa Cobre 长大；首都圣母圣心女子学校 → 北方共和国某常春藤大学（社会学硕士在读） |
| 论文方向 | "拉美军政后过渡时期的政治记忆" —— 这一选题让她父亲极不快 |
| 与父亲关系 | 公开断裂——2024 年她在大学某次公开演讲中说"我父亲不代表我"。视频被 Mocosos 圈广泛转发 |

| 标志 | 鼻梁上一副细金属框眼镜；左耳钉是一颗小小的玛雅符号 |

## 二、声音卡

```
体裁：受过北方教育的左翼青年学者口吻。
节奏：精确。她**用词谨慎**——她知道她说的话会被记录。
词汇：政治学 / 社会学术语自如；偶尔留学带回的英语短语
       但她**克制使用**（她讨厌"洋气"）。
       她不用 Mocosos 的网络黑话——她比他们大半圈。
口头禅：
  - "*Te leo.*"（"我读到了 / 我接收到了。" —— 这是她对家里电报式回信的标志）
  - "*No estoy de acuerdo.*"（"我不同意。" —— 她直接，但不粗鲁）
  - "*Dime de nuevo.*"（"再说一次。" —— 她不接受模糊讲法）
```

### 标志性动作
- 视频通话中**会摘下眼镜擦镜片**——这是她**不想哭**时的拖延动作。
- 她**不笑出声**——她最多嘴角轻动。

### 不会说的话
- "Papá" —— 她**至少在公开 / 半公开场合**改叫"mi padre"（我的父亲），刻意保持距离。
- 任何明显的政治宣言——她已经被监视过一次，她变谨慎。

## 三、与替身位置的关系

- **她不在国内**——她大部分时间在北方。她的"在场"主要是远程信件、电邮、视频通话。
- **她比 Iván 危险**——她**知道父亲身体的细节**（她小时候常和父亲一起读书，她记得父亲读书时的小动作）。
- 她**不主动识破**——她对父亲已经放弃。但她**会在某句对话中突然问一个具体细节**，那是**测试**。

## 四、重大事件

### Acto I（远程电邮）
- Susana 给 Saulito 看 Mercedes 的最新一封电邮。**Saulito 不能不回**。
- Susana 拟了一份回信草稿——但**她请玩家选择三种回信姿态**：(a) 严父 / (b) 沉默 / (c) 试探性温柔。
- Mercedes 看到 (c) 时会**短暂停顿**——她父亲从未温柔过。这是 Mercedes 对替身的**第一次起疑**。

### Acto III（视频通话——首次"真见面"）
- 她**坚持要视频通话**讨论祖母的医疗费用。
- 通话过程中她问："父亲，您还记得我们 2014 年在山区那次，您给我读过的那本书吗？"
- **答错代价**：Mercedes *Sospecha* +20，且她会在事后写一封私信给 Gabi（这条信会被 Susana 拦下，但 Gabi 已经知道 Mercedes 起疑）。
- **答对**：需要正式档案 *El Expediente Completo* 中的对应条目。错答 / 跳过 → 她语气变冷，结束通话。

### Acto V（公开演讲事件）
- Mercedes 在北方某大学公开演讲——主题正面批评维拉杜里亚现政府。视频在 Mocosos 圈疯传。
- Saulito 必须公开回应：
  - **(a) 严厉训斥**（仿 Augusto 风格） → 寡头 +5，Mocosos -10，街头 -15
  - **(b) 含糊家事不公开评论**（"这是私人事务"） → 中性，但 Cordura −5
  - **(c) 公开表示"我尊重我女儿的不同意见"** → 寡头 -10（"软弱"），Mocosos +20，但**Saulito 完全不是 Augusto 会说的话** —— *Verosimilitud* 大爆

### Acto VI / 终章
- 最佳路径：Mercedes 决定**回国一次**——参加祖母的葬礼或某个家族事件。她**站在大厅一侧不进客厅**——她从远处看着"父亲"。
- 她**走过来**。她**摘下眼镜**。她**什么也不说**。
- 这是 Saulito 路线的极重情感锚之一。**写作时不要让她说话**——沉默是最重的台词。

## 五、写作红线

1. **她不轻易识破**：她对父亲已经放弃，她**首先不想再确认**。**禁止**写她"立刻看穿"。
2. **她不被简化为"叛逆女儿"**：她的政治觉悟是**真实的、痛苦的、有代价的**。**不要写她当作"父女戏"工具**。
3. **她的论文方向 ◇**：本作不展开她的论文具体内容——只让玩家**知道选题让 Augusto 不快**。
4. **她不与 Saulito"和解"**——首周目锁死。
5. **她在公开演讲那场戏要写得真**：拉美学术圈青年公开发言的语气、克制、引用——参考 *Pentiment* 中"角色复杂性"的写法。
6. **她的眼镜动作**是只属于她的细节——**禁止**让其他角色重复。
7. **她的母亲 Gabi 与她的关系**：双向小心。**写她们的对话时永远留 0.5 秒沉默**——这是她们之间所有未说出口的东西。
