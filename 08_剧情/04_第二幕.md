# 08·04 · 第二幕 *Los Doce*（十二个）

> **戏剧问题**：寡头开始试探你
> **剧情天数**：第 8–18 天
> **玩家时长**：约 35 分钟
> **节奏**：从"撑过第一周"的小胜到第一次直面寡头集团的智识压力。这一幕是 *Verosimilitud* 系统的真正第一次大考——尤其是国宴上的 Mariana。

---

## 一、第二幕功能

1. **十二姓氏正式登场**：Doña Isabel 从邮件 / 电话进入实体接触，伴随 Don Federico 与 Mariana。
2. **双使馆温和试探**：北方 Whitfield 茶会 + 南方 Aldana 足球邀请——两国此时尚未撕破脸，玩家可以**真正实践中间派周旋**。
3. **铜矿议案首次浮现**：本作主线政经核心议题之一，第二幕末玩家必须给出"近期立场"（不是终决）。
4. **国宴 = 风险与机会的会合点**：Mariana 出场，所有派系第一次在一个房间里。

---

## 二、关键事件

### ACTO_II_A_001 · Doña Isabel 觐见

**时刻**：第 9 天 11:00
**类型**：Audiencia
**时辰**：2

**剧情骨架**：

- Doña Isabel 来访。**她戴着白布手套**——本作的"派系标识"。她不递名片，她递一束 Cumbre Mayor 庄园的兰花。
- 议题表面是"关于农业部一些预算调整"，实际是来观察新总统。
- **她的核心试探**：问起"那次心脏病发作"。**真总统两年前有过一次小心脏病发作（档案记录）**——玩家若知道这个，能轻松回应。
- 同时她暗示家族下个月在 Cumbre Mayor 有"小型聚会"，希望您光临。

**主要选项**：

| 议题 | 选项 | 后果 |
|---|---|---|
| 心脏病发作话题 | A 顺过去（"已经好了"） | 安全；Sospecha 不变 |
| | B 详细描述（V 检定 ≥ 50） | 成功：Sospecha-Doce −5；失败：+10 |
| | C "我不想谈那个" | 隐性 `EVADED_HEALTH_TALK`；Doce 0；Cordura +5 |
| 庄园聚会邀请 | A "我会安排" | 隐性 `WILL_VISIT_CUMBRE`；联姻线提前一幕 |
| | B "等议会休会再说" | 推迟联姻线；Tesoro −1 |
| | C "您直接告诉我您要谈什么"（强势） | Doce 大幅 −5；Mariana 个人态度 +10（她欣赏正面对峙） |

**衔接**：会面结束后，Doña Isabel 的助手"无意"留下一份资料夹——里面是几份**关于铜矿私有化的"研究报告"**（实际是寡头的游说材料，Estela 标注："不必急于看"）。

### ACTO_II_A_002 · 北方使馆 Whitfield 茶会

**时刻**：第 11 天 16:00
**类型**：Audiencia
**时辰**：2
**地点**：北方使馆

**剧情骨架**：

- Helena Whitfield 邀请总统到使馆喝茶。会客厅有一架老钢琴，墙上挂着北方共和国前几任大使的肖像。
- 茶会议题：**IMF 贷款重启的可能性**。Whitfield 暗示——如果维拉杜里亚启动铜矿私有化，IMF 可能松口。
- 中段她"无意"提起："先生，我们的医生在维也纳的一家医院……如果您需要任何健康咨询……"

**核心选项**：

| 选项 | 后果 |
|---|---|
| A "我会与内阁讨论铜矿议案" | 北方使馆 +5；隐性 `OPENED_PRIVATIZATION_TALK` |
| B "私有化是个长期议题"（推迟） | 北方 0；Tesoro −1 |
| C "先重启贷款，再谈私有化"（条件交换） | V 检定 ≥ 60；成功：北方 +10、Tesoro +2；失败：北方 Sospecha +10 |
| D 关于医生："不必，我有自己的人。" | 北方 0；隐性 `REFUSED_FOREIGN_DOCTOR` |

> **设计意图**：Whitfield 永远不威胁——她的危险在于她的**关心是真心的**。玩家 Cordura 越低，越容易被她"哄"出去话。

### ACTO_II_A_003 · 南方使馆 Aldana 足球邀请

**时刻**：第 13 天 19:00
**类型**：Audiencia
**时辰**：3
**地点**：圣克里斯托瓦尔市中心球场（Vasconia 资助修建）的 VIP 包厢

**剧情骨架**：

- 国家队比赛，Aldana 邀请总统进 VIP 包厢观赛。气氛极为热闹——Aldana 大声唱跑调的 Vasconia 民歌，往你的酒杯里加冰，问你足球。
- **核心试探**：他问 *"Compadre，您年轻时是不是也踢过球？"*——真总统从不踢球，Saulito 踢过。
- 中场休息时 Aldana 把你拉到包厢角落，递给你一个小信封："这是 Vargas 总统让我转交您的。他说——他是您的朋友。"

**核心选项**：

| 议题 | 选项 | 后果 |
|---|---|---|
| 足球话题 | A "我从不踢，但喜欢看。" | 安全；Sospecha-Sur 0 |
| | B "我年轻时进球过几次。" | V 检定 ≥ 50；失败：Sospecha-Sur +10 |
| | C "Compadre，您觉得我踢过吗？" | 玩心眼；Aldana 笑过去；隐性 `PLAYED_DUMB_TO_ALDANA` |
| Vargas 信封 | A 拆开看 | 内容是一份"友情贷款"提议——南方使馆 +5 |
| | B 收下不拆 | Tesoro 0；隐性 `KEPT_SEALED_LETTER`（多周目可解锁特殊事件） |
| | C 退还："请告诉 Comandante 我的感谢，但我不能收。" | 南方 −5；隐性 `REFUSED_VARGAS_LETTER`；北方使馆某处会知道这件事，他们好感 +5 |

### ACTO_II_C_001 · 第一次外汇危机

**时刻**：第 15 天 08:30
**类型**：Crisis
**时辰**：3 + 死线
**死线**：当日 16:00

**剧情骨架**：

- 早班简报，Estela 紧张地报告："总统先生，黑市美元（北元）暴涨 18%。央行储备只够撑一周。"
- 突发——你必须当天给出回应。

**主要选项**：

| 选项 | 时辰 | 后果 |
|---|---|---|
| A 紧急向 IMF 求助（启动 Whitfield 路线） | 2 | 北方 +10；街头 −5；隐性 `IMF_OPENED_FORMAL` |
| B 紧急向 Vasconia 求助（石油折扣 + 短期贷款） | 2 | 南方 +10；北方 −10；隐性 `SUR_FORMAL_AID` |
| C 启动外汇管制 | 3 + Tesoro −3 | 经济短期 +5；街头 +5（黑市套利者愤怒，普通人受益）；老派 −5；中产 −10 |
| D 装作什么都没发生 | 0 | 经济 −15；Cordura −10；Estela 表情冷淡 |

> **设计意图**：第一次让玩家感受到"外交选边"是经济现实——你不是在抽象站队，你是在让人民下个月吃不吃得起鸡蛋。

## 三、★ ACTO_II_P_004 · 国宴 / *Los Guantes Blancos*（白手套）

> **完整剧本范本见 [07_风格样本.md](../07_风格样本.md) 样本 2**。

**时刻**：第 17 天 19:00–22:30
**类型**：Público / Audiencia 混合 / 强制
**时辰**：4

**关键人物在场**：
- Doña Isabel（右侧第二位）
- **Mariana**（首次出场，对面坐）
- Don Federico（与 Pierce 对话）
- Pierce（北方使馆）
- Aldana（南方使馆）
- Vega（玩家左后方两米）
- Susana（在远处假装调整镜子）
- Cardenal Soto（中央桌，沉默）

**Gabriela 缺席**——借口"偏头痛"。

**Mariana 的核心试探**（两层对白）：

1. *"听说您最近读了什么书？"* — 真总统不读书。
2. *"我父亲那块表是 Seiko Sportsmatic 1972 年的——两位老先生应该年纪差不多吧。"* — 真总统父亲那块是 1968 Seiko 5。她故意说错，测你会不会纠正。

**玩家选项**（详见样本 2）：A 纠正 / B 装傻 / C 转移话题 / D 多周目档案选项。

**幕终事件 + 隐藏机制**：
- 你最终对 Doña Isabel 说"我会让秘书安排"——这是真总统从不会用的句式。**Mariana 嘴角动了一下**。她已确认。
- 隐藏标记累计：`MARIANA_KNOWS_VERSION_X`（A/B/C/D）

## 四、第 18 天 — 铜矿议案首次抉择

### ACTO_II_D_001 · 铜矿议案（一阶段）

**时刻**：第 18 天 15:00
**类型**：Decreto / 强制（幕终）
**时辰**：2 + Tesoro 视选项
**死线**：当日 19:00

**议案内容**：北方与寡头联手推动的铜矿私有化第一步——把"国营矿业管理局"改组为股份公司，引入外资战略投资人。

**主要选项**：

| 选项 | 时辰 / Tesoro | 后果 |
|---|---|---|
| A 签字通过 | 1 / Tesoro −0 | Doce +15；北方 +15；街头 −15；原住民事件链触发；隐性 `SIGNED_COPPER_PRIVATIZATION` |
| B 退回议案要求修改 | 2 / Tesoro −2 | Doce −5；北方 −5；获得喘息一幕；隐性 `WITHDREW_COPPER_PRIVATIZATION` |
| C 公开宣布"暂缓讨论" | 2 / Tesoro −5 | Doce −15；北方 −15；南方 +10；街头 +20；原住民 +15；Tesoro 大幅消耗 |
| D 用 Joaquín 牌（如果已获得） | 1 / Tesoro 0 | Doce 噤声；议案被推迟；隐性 `USED_JOAQUIN_CARD` |

> **重点**：此议题在第五幕会再次回来。第二幕的选择是"短期立场"，不是"永久结局"——但**会改变玩家剩余资源 + 派系起始姿态**。
