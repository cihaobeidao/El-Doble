# Don Federico Ríos-Echeverría

> 他是 Doña Isabel 的小叔。
> 他在北方使馆有一把椅子。
> 在维拉杜里亚的国库报告里也有一把椅子。

## 一、身份信息

| | |
|---|---|
| 全名 | Federico Ignacio Ríos-Echeverría |
| 年龄 | 68 |
| 出身 | Ríos-Echeverría 第二代 Don Roderigo 的弟弟（家系图中表"叔"辈，实际与 Isabel 同辈共事）。 |
| 教育 | 圣克里斯托瓦尔陆军学院（短期）→ 北方共和国 *MIT Sloan* 商学院 MBA（1980 年代）|

| 婚姻 | 未婚，无子嗣；公开同性伴侣关系（一位北方建筑师，长居 La Casa de Federico 山区别墅） |

## 二、声音卡

```
体裁：双语精英企业家——他**比 Doña Isabel 更"国际"**，但**也更冷**。
节奏：极有效率。每句话都为"deal" 服务。
词汇：英语 / 西语自由切换；金融术语自如；
       他**不用**任何宗教用语——他**从不假装信**那一套。
口头禅：
  - "*Look, Augusto...*"（**他经常用英语开场**——这本身是阶级标志）
  - "*Es buen negocio para todos.*"（"对所有人都是好生意。"——
     **他每次提议都用这句话收尾**——这是他对世界的总论）
  - "*Tiempo es dinero.*"（"时间就是金钱。"——他不耐烦时）
```

### 标志性动作
- 谈话时**反复转动左手食指上的金戒**（家族纹章戒）。
- 听人讲话时**不看对方眼睛**——他看**对方说话时的嘴形**——这是一种他自学的"读人"技巧。

### 不会说的话
- 任何宗教情感话语——他**不假装信**。
- 任何对工人 / 原住民利益的口头同情——他**不浪费**这种修辞。
- 任何家族内部"团结"的辞令——他**懒得**装家庭温情。

## 三、与替身位置的关系

### 他想要什么
- **铜矿私有化的最大受益者候选**——他代表 Norhem Copper 进场。
- 他要的是**一份签字** —— 总统签字让铜矿走完私有化程序。
- 他**不在乎总统是谁**——他要的是签字。

### 他的危险性
- 他**最快**意识到 Saulito 是替身——**他从签字方式上看出来**（他对 Augusto 签字习惯有过详细研究，因为他要预判每一份合同会被怎样签）。
- 但他**不揭穿**——一个"软"的总统对他更好用。
- 他与 Pierce 的私交意味着——**他可能比 Whitfield 更早从北方使馆获得 Saulito 替身的"分析师报告"**。

## 四、重大事件

### Acto I 末（铜矿私有化议案首次端上桌）
- 内阁会议——Federico 简洁陈述。"Look, Augusto, IMF 在催，议案不能再拖。"
- 玩家选择：
  - 表态支持 → 寡头 +20，原住民 −20，街头 −15
  - 表态反对 → 寡头 *Sospecha* +20，但街头 +10
  - 推迟 → 中性，但 Federico 私下"提醒" Saulito（参 Acto II）

### Acto II 初（Federico 首次单独约见）
- 山区别墅 La Casa de Federico —— Saulito 第一次离开 La Casa Cobre 单独被某位寡头带去。
- Federico 不施压——他**给** Saulito 一杯非常贵的威士忌（25 年 Macallan，市价 ~3,500 北元——**Saulito 一年的房租**），讲一个"我们一起做了多年朋友"的故事——故事 90% 真，10% 是他在测 Saulito 是否记得"我们"。
- 答错 → Federico 心里"识破"提前。

### Acto III 中（与 Pierce 的同台）
- 一次北方使馆的私人晚宴——Federico 与 Pierce 同桌。**他们互称名字**——这是亲密的标志。
- Saulito 第一次意识到——**寡头 + 北方使馆的同盟比他想象的更紧**。

### Acto IV（联姻议题与铜矿议题的捆绑）
- Doña Isabel 推联姻 + Federico 推铜矿——**他们不公开协调**。但 Saulito 同意联姻 = Federico 立即推动铜矿议案到下一步。
- **打 Joaquín 牌 → Federico 的反应**：他**比 Doña Isabel 冷**——他说一句"*Mal momento, Augusto. Muy mal momento.*"（"时机不好，Augusto。非常不好。"）—— 他不哭，他**记下来**。

### Acto V（Federico 与 Pierce 共谋 / 背叛 Whitfield）
- 多周目深度路径——玩家可发现 Federico 与 Pierce 在 Whitfield 不知情的情况下推动了某些事。这是**北方使馆"鹰派内 vs 温和派内"分裂**的关键节点。

### Acto VI / 终章
- 视玩家路径：
  - 铜矿议案签字 → Federico 在终章是"得胜者"——但他**没有公开庆祝**。这是他**真正可怕的部分**：他的胜利是**安静的**。
  - 议案被否 → 他不立即报复——**他等**。多周目玩家会发现：他的报复在下一届政府才到。

## 五、写作红线

1. **他从不假装信**——任何让他用宗教用语 / 家族温情辞令的台词都写错了。
2. **他与 Pierce 的友谊是真的**——这是寡头 + 北方使馆同盟的私人锚。**写作时不要把他们写成纯粹利益关系**——他们**真的喝过几杯威士忌**。
3. **他无子嗣的同性伴侣**：可以提，但**不情色化**。这是他的私人生活，**不是政治筹码**——任何让其他派系"以同性恋为把柄"的剧情**禁止**——这违反 06 红线 4 的复杂性原则（他的私人生活不是漏洞）。
4. **他签字识破**这件事——多周目深度揭示。**禁止**让他在首周目台词中暗示。
5. **他的英语**：写他时**至少每三段保留一个英语词**——这是他阶级身份的真实质感。
6. **他的报复是安静的**——他**不喊**。**禁止**让他暴怒。
