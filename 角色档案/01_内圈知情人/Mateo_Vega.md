# Coronel Mateo Vega

> 你以为他是来救国家的清流。
> 他真的这么以为。
> 这是问题最大的部分。

## 一、身份信息

| | |
|---|---|
| 全名 | Mateo Enrique Vega Ríos |
| 军衔 | Coronel（上校） |
| 年龄 | 41 |
| 出身 | 首都贫民窟（**不是 Las Veladas，是相邻的 Barrio del Río**） |
| 教育 | 自己一路考上军校 → 北方某防务学院短期访学（与 Whitfield 一脉认识由此而起） |
| 现职 | 总统警卫队队长 + 青年军官派核心领袖 |
| 父亲 | Coronel Enrique Vega（已故，老警卫——参 `99_秘密角色/Coronel_Enrique_Vega.md`） |
| 婚史 | 已婚，妻子 Patricia 是首都某中学西语教师，育有一子（11 岁） |
| 标志 | 永远擦得发亮的军靴；左胸口袋里有一本黑色小本（《2026 备忘录》草案的一部分） |
| 政治立场 | "清流救国"——他真心相信"让一个真正干净的人来管国家"；他不信任老派 Castaño，也不真信北方使馆 |

## 二、声音卡

### 总规则
- **军人的话术：短、清、不绕**。
- 但他**不是简单的硬汉**——他读过书（短期防务学院时学过国际关系），会用相对学术的表达，但他**克制使用**，因为他怕被同袍嘲笑"留洋归来"。
- 他的语气**有一种不易察觉的悲剧感**——他**真的相信他做的是对的**，这让他成为本作"政治悲剧"最尖锐的化身。

### 公开 / 半公开
```
体裁：军人式简报。陈述句多过修饰句。
节奏：每句不超过 10 字（西语换行处更短）。
词汇：军事术语 + 法律术语 + 偶尔出现的"我们这一代"修辞。
口头禅：
  - "*Entendido, señor.*"（"明白，先生。"——他对替身的标准回应）
  - "*A tiempo.*"（"按时。"——他确认时间表时）
  - "*Hermano.*"（私下对青年下属——这是他与父辈军人最大的语调差异）
```

### 私下（与 Susana / 替身 / Whitfield 单独）
```
体裁：仍然短句，但允许出现一两句**慢一点**的"我以为……"
节奏：偶尔停 1.5 秒——他在脑里推一遍。
情绪锚点：父亲（已故）、那本黑色小本、北方使馆某次会议后他在停车场抽的那根烟、
        他妻子说过的某句话（"你想救的国家是真的国家吗？"）。
```

### 标志性动作 / 习惯

- 摘下军帽时**用大拇指内侧抹一下帽檐**——一个无意识的小动作。Saulito 在第二周目可识别这是 Vega（哪怕背影看不到脸）。
- **从不喝酒**——这与父亲那一代的军人形成强烈反差。
- **走路有左脚轻微靠后**的步态（一次旧伤，2018 年镇压示威时被人投石砸到）。

### 不会说 / 不会做的话

- 任何美化 Castaño 老派的话——他**真心鄙视他们**。
- 任何 1973 年事件的辩护——他对父辈军人那一代有**真实的不齿**。
- 在公开场合喊"民主"——他认为那是北方使馆的词。**他用的是"清廉" / "秩序" / "真正的国家"**。

## 三、与替身位置的关系

### 表面
- **他是把替身计划牵线的关键人物**。他需要"一个看起来是总统、但不是 Báez、也不是 Castaño"的人，争取喘息时间。
- 他与 Saulito 的关系是"指挥官与下属"——他下达明确指令、检查执行、不浪费情绪。
- 他**在序幕黑色雪佛兰后座**那一刻是 Saulito 整段经历的"招募者"——但他从来不与 Saulito 共情。

### 危险性
- 高。**他的"清流"信念是双刃剑**——他可能在某一刻判断"这个替身已经没用了"，**用决断的方式处理**。
- 他不是反派——他是**真心想干净的人**。这就是他可怕的地方。

### 他知道什么 / 不知道什么
- **他知道**：第二任替身存在；第二任替身被软禁；Saulito 是第三任。
- **他不知道**：1998 年发生的事；他的父亲 Enrique 也参与；Saulito 的父亲是第一任替身。
- 这种"半知"使他成为多周目里**容易被反派化也容易被同情化的人**——参 `99_秘密角色/Coronel_Enrique_Vega.md`。

## 四、重大事件

### 序幕
- **PROL_C_001**：黑色雪佛兰后座。三个选项 + 复述母亲门牌、Lucía 考试时间、汇款金额——**威胁不需要言明**。
- 他给 Saulito 的"四万北元押金 + 三个月扮演"合同——他**预设的是真心给的**。**他不是骗 Saulito**——他认为这是公平交易。

### Acto I（指令与检查）
- 第一次内阁会议前——走廊上简短交代。"Patrón，会议要点三条：(1) 军费议案不要表态；(2) Doña Isabel 提到农业补贴时点头一次；(3) 散会后立即回 La Casa Cobre。"
- Castaño 提议打猎——Vega **不能阻止**。这是他第一次感到"我控制不了所有变量"的瞬间。

### Acto II（与 Whitfield 的张力）
- Whitfield 在使馆茶会不动声色地问："Vega 上校最近气色不太好。" 这是北方使馆首次对青年派施压的暗示。**Pierce 实际上是 Vega 的对接人**，但 Pierce 与 Vega 的关系比表面紧张——Pierce 想推 Vega 走更激进的路线，Vega 不愿。
- Vega 在 Casa Cobre 内部找到一份**Castaño 私下与某南方使馆联络人接触的记录**——这是 Acto IV "南方收编军方中下层"线索的种子。

### Acto III（信任阈值开始）
- Vega 在 Saulito 巡视山区前夜，**单独向他交代某些政变备忘录的细节**——这是 *Vega 信任标记 +1* 的开关事件。
- 这之后玩家积累的"Vega 信任"标记决定 C 卷宗（"真总统已死"路径）的解锁。

### Acto IV（青年派 vs 寡头）
- 联姻提议浮出——Vega 私下找 Saulito 说："Patrón，您可以拒绝。我支持您。"——这是他与寡头第一次正面冲突的信号。
- Vega 此时**还没意识到**寡头能反咬青年派。他低估了 Doña Isabel。

### Acto V（北方使馆 vs 青年派的紧张）
- Pierce 在使馆派对上对 Vega 说："你的人在跑。" 这是使馆"鹰派"对 Vega 的施压点。
- Vega 在 Casa Cobre 自己的办公室里**第一次对着窗子站了 5 分钟**——一个他之前从不允许自己做的姿势。

### Acto VI（C 卷宗或反扑）
- 玩家积累的 Vega 信任 ≥ 4 → 解锁 C 卷宗：Vega 的私人日记被打开，记载"真总统已死"的版本。**Vega 自己以为 C 是真相**——但 C 是半真（参 99）。
- 反向：Vega 信任 < 阈值 → Acto VI 末尾 Vega 可能**亲手处理 Saulito**（最严酷分支）。

### 终章
- Vega 几乎没有"个人快乐结局"。最好的结局：他**带着妻儿离开维拉杜里亚**——这是玩家在 Acto V 让他活下来 + Acto VI 主动让他走的奖励。
- 最坏的结局：他在政变中被 Castaño 老派反扑，**死在他自己策划的"清流"里**。

## 五、写作红线

1. **他真心相信**：写他时**永远不要把他写成投机分子**。**他是悲剧人物**——这是本作"对左翼某些版本"的批评（参 `99_秘密文档.md` §3.2）。
2. **他不知道 1998**：所有他相关的台词、文件、内心暗示**严禁**透露 Susana / Gabriela / 1998 / 父亲也在场。**这是他作为"半知人物"的设计核心**。
3. **他的父亲是 Enrique Vega**——这件事 Vega 自己**知道父亲是已故老警卫**，但他**不知道父亲也参与了 1998**。**首周目内绝不暗示**。
4. **他不喝酒**——参考其父辈军人对比强烈。**写他时哪怕在悲剧时刻也不要让他喝酒**——这是他与父亲那一代的边界。
5. **他对 Whitfield 的态度**：互相利用 + 互相不信任。**不要写成纯粹的合作**。
6. **他的妻子 Patricia 不出场**——她最多通过电话、书信间接出现。**不要让她变成情感线工具**。
7. **他叫 Saulito"Patrón"——从不叫真名**。**叫真名的特权属于 Susana**。
8. **他在替身周目（Catalina / Tobías / Gabi）中态度不同**：
   - Catalina 路线：他**更尊敬她**——她是他的同代人。
   - Tobías 路线：他**更怀疑**——这个青年身上有一种他读不懂的恨。
   - Gabi 路线：他**实际上是她的下属**（参 99 真相）——但他自己以为是合作者。这是 Gabi 路线最 Marquez 式的反讽。
