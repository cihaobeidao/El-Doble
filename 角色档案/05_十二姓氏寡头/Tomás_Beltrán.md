# Tomás Beltrán-Ríos

## 一、身份信息

| | |
|---|---|
| 全名 | Tomás Eduardo Beltrán-Ríos |
| 年龄 | 30 |
| 出身 | 父亲为另一姓氏家族（Beltrán 家族，老牌银行家族）；母亲是 Doña Adelaida（**他与 Mariana 是表兄妹**） |
| 教育 | 圣克里斯托瓦尔法语中学 → 北方共和国 *Wharton* 商学院 MBA |
| 现职 | 家族银行 *Banco del Centro* 副总裁 |
| 婚姻 | 单身。多年间被家族安排过几次相亲，全都不了了之 |
| 与 Mariana 的关系 | **关系暧昧——这是家族内部公开的秘密** |
| 标志 | 干净修剪的胡子；一只 1980 年代复古 Omega Seamaster（**他祖父的**——比真总统的 Seiko 更高档但同样有"传家"味） |

## 二、声音卡

```
体裁：受过最高教育的拉美年轻金融家——克制、礼貌、有距离。
节奏：完整的句子，**几乎不用感叹号**。
词汇：金融术语 + 古典文学引用（他偶尔引用 Borges）；
       他**不夹杂英语**——这是他与 Joaquín / Iván 的区别（他更"老派"）。
口头禅：
  - "*Permítame...*"（"请允许我……"——他打断时的礼貌前置）
  - "*En principio...*"（"原则上……"——他保留意见时）
  - "*Con permiso.*"（"请允许我先告退。"——他离场前永远用这个）
```

### 标志性动作
- **不与人长时间对视**——他的眼神很快**滑开**，回到桌面或自己的酒杯。这与他对 Mariana 形成的对比是关键——他对 Mariana **会对视**。
- 听人讲话时**手指轻击桌面节拍**——这是他在脑里推算什么的标志。

### 不会说的话
- 任何关于他与 Mariana 的事——他**绝口不提**。
- 任何对 Joaquín 表弟的辩护——他**默默瞧不起** Joaquín。
- 任何党派立场——他**保持表面中立**——但金融上他与北方使馆走得近。

## 三、与替身位置的关系

### 他与 Saulito 几乎无直接关系
- Tomás 不参与政治——他在银行里。
- 他**不识破**——他对父辈层面的事不关心。
- 但他是**Mariana 联姻线的暗影**——如果 Mariana 嫁给"总统"，Tomás 的反应是 Mariana 路线的关键私人代价。

### 他真正在意什么
- **Mariana** —— 他对她的感情**比家族内部传言的"暧昧"更深**——但他也**清楚他们不可能**。
- **家族银行的稳定**——他不是寡头议程的推动者，他是**维护者**。

## 四、重大事件

### Acto III 中（家族晚宴首次出场）
- 一场寡头家族非正式晚宴——Saulito 与 Mariana 同桌，Tomás 也在。
- **Tomás 与 Mariana 单独对视一次**——Saulito 看到。这是替身路线上 Saulito 的第一次"私人嫉妒感"——**但他没有资格嫉妒**——他不是 Mariana 的丈夫，他不是任何人的什么。

### Acto IV 联姻提议时（Tomás 的私下反应）
- 联姻议程公开后—— 一次 Saulito 在某画廊偶然撞见 Tomás 单独站在阳台上。
- Tomás 转身。"Presidente." —— 短，礼貌。
- 玩家选择：
  - 离开 → 中性
  - 主动搭话提到 Mariana → Tomás 沉默 3 秒。说一句："*Cuídela.*"（"请您照顾她。"）—— 这是他对 Mariana 的全部告白
  - 直接问"您喜欢她，对吗" → Tomás 不答。**他离开**。Mariana 路线 *Sospecha* +10（Tomás 会告诉 Mariana）

### Acto V（金融议题侧的小事件）
- 一份金融条例 —— 涉及限制北方资本流入。Tomás 在某次正式会议上**罕见地表态**——他**反对**。
- 这是 Tomás 在剧情中的**唯一一次政治表态** —— 他做这件事是为了家族银行的稳定，不是为了寡头议程。

### Acto VI / 终章
- 视玩家路径：
  - 联姻成功 → Tomás 在终章里**不出席婚礼**。他离开维拉杜里亚。
  - 联姻失败 + Mariana 信任建立 → Tomás 与 Mariana 仍然不能在一起——但 Mariana 把 Tomás 留下。**这是 Mariana 的另一种"自由"**——她拒绝把 Tomás 当作"备胎"。

## 五、写作红线

1. **他不是浪漫线工具**——他与 Mariana 的暧昧**不是**为了 Saulito 的三角戏。**这是他们两个人的事**——Saulito 是局外人。
2. **他从不直接表白**——他**最多说"Cuídela"**——这是他全部的告白。**禁止**让他直白宣言。
3. **他的 Omega Seamaster 是祖父的**——这是他与真总统的 Seiko 形成的私人镜像。**首周目玩家不应明显注意**这种镜像，但**多周目玩家可能注意到**。
4. **他不识破替身**——首周目锁死。
5. **他的克制不是软弱**——任何让他变成"被动可怜的男二"的写法**禁止**。他**有选择**——他选择克制。
6. **他与 Joaquín 表弟的关系**：**他不挽救**他。**禁止**让他成为 Joaquín 丑闻路径的协助者。
