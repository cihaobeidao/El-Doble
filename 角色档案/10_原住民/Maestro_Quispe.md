# El Maestro Quispe

> 他在每场弥撒上都坐在 *Catedral de la Aurora* 最后一排。
> 但他从不是天主教徒。
> 他来这里是因为这是城里**唯一让他坐着不被赶出去**的公共空间。

## 一、身份信息

| | |
|---|---|
| 全名 | Antonio Quispe（**他自己更喜欢"Maestro Quispe"** —— "木匠师傅"——他从未要求别人这样叫他，但**所有人都这样叫**） |
| 年龄 | 55 |
| 出身 | Cordillera 山区——从小学木工 → 1990 年代下山到首都谋生 |
| 教育 | 民间手艺传承 |
| 现职 | 圣克里斯托瓦尔旧城的木匠 —— 修桌椅、做棺材、雕圣母像 |
| **失语症** | 不是先天——**是 1980 年代山区某次事件后**（**禁区**：他**不能讲话**——但他**听**） |
| 婚姻 | 未婚 |
| 标志 | 永远穿一件灰色工作服；**总是带着一把小折叠刀**（用来削木头）；左眼睛失明（白内障未治） |
| **他不是天主教徒** | 但他每周日去坐 Catedral de la Aurora 最后一排 —— 这是**城里唯一让他坐着不被赶出去**的公共空间 |

## 二、声音卡（**他不说话**——他的"声音"是他怎么被书写）

```
描写规则：
  - 他**不说话** —— 写他时**永远不要让他开口**——除了**点头/摇头**和**写字**
  - 他**写字** —— 用他随身的小铅笔 + 一本旧账本——他能写**短句**
  - 他**雕**——他对世界的回应是**雕刻**——他给孩子雕小动物，
     给死者雕棺材纹饰，给 Cardenal Soto 雕过一只小木鸟（**Soto 留着**）
  - 他**听** —— 他的"对话"主要通过他**在听什么**呈现
禁忌：
  - 不写"他想说但说不出"——这是廉价情节
  - 不让他突然恢复说话能力——他**不会**
  - 不暗示具体导致他失语的事件——**禁区**
他写过的字（玩家可见的几行）：
  - "*Estoy aquí.*"（"我在这里。"）
  - "*Gracias.*"（"谢谢。"——他对所有给他食物 / 工具的人）
  - "*El árbol también muere.*"（"树也死。"——他唯一一次写一个意见）
```

### 标志性动作
- **每场弥撒坐 Catedral 最后一排**——他**不接受 Cardenal 的特别照顾**——他**坚持**坐最后一排。
- **削木头**——他随身一块小木头，谈话时削。**这是他思考的方式**。
- **微笑** —— 他**笑**得多——**与他失语形成对比**——他**会用眼睛与表情**回应——他比许多有声角色"说"得多。

## 三、与替身位置的关系

### 他不直接接触替身
- 他**远远看到** Saulito 几次——一次是 Catedral 弥撒散场时的人群中。
- 他**点头一次** —— 隐性标记 `MAESTRO_QUISPE_NODDED`——**不直接改派系数值**——但**改变事件的语气与多周目的揭示方式**。

### 他知道什么
- 他**几乎不知道任何事** —— 他**远** —— 但他**听** —— 他听过的弥撒、听过的街头讨论、听过的祭典——这些堆积成他的判断。
- 他**对 Saulito 的判断**：玩家不被告知。但**他的点头**或**不点头**是设计的隐性信号。

## 四、重大事件

### Acto II 起（背景）
- 玩家在 Catedral 弥撒场景中**可能注意到**他——他坐最后一排——他**在削一块木头**。
- 这是**首次出现的视觉锚** —— 但叙述者**不点破他是谁**。

### Acto V（"点头"事件）
- 一次 Catedral 散场——Saulito 路过 Maestro Quispe 旁边。
- Maestro **抬头看 Saulito 一眼**——他**点头一次**或**不点头**。
- **点头条件**：玩家在 Acto III 山区 Aurora 那场选择"沉默"或"哭出来"——这是 Saulito 路线**最微妙的隐性轴对接**。
- 这一刻**只有玩家自己知道**它的意义——`MAESTRO_QUISPE_NODDED` 标记触发。

### Acto VI（如玩家深度路径）
- Maestro 在自己的小作坊给 Saulito **雕了一个小木雕** —— 不知名的鸟——通过 Cardenal Soto 转送。
- Saulito 收到——**纸条**："*Para el de la cara cansada.*"（"给那张疲惫的脸。"）—— 他**没写名字**。

### 终章
- True Ending Route：Saulito 在 Las Veladas 黎明前的灰光里——**Maestro Quispe 路过**——他**点头**——他**继续走**。
- 这是终章的一个**沉默见证**——他**承认**这个具体的人。

## 五、写作红线

1. **他不说话** —— **永远** —— 任何让他开口的台词违反角色一致性。
2. **不写他失语的具体原因** —— **禁区**——首周目玩家可以怀疑（一桩 1980 年代事件），**多周目深度路径**才间接揭示。
3. **他不被简化为"沉默象征"** —— 他**有具体的木工活、具体的小折叠刀、具体的微笑** —— **他是一个具体的人**。
4. **他与 Cardenal Soto 的关系**：相互尊敬——**但他不是天主教徒**——他**坚持**坐最后一排——**严守**。
5. **他的小木鸟给 Cardenal**：Cardenal 留着——这是他们**两个老人之间的私人语言**——**禁止**让其他角色知道这件事的"意义"。
6. **他写的字 —— 短句** —— 他**不写长句**——任何让他写长篇文字的设计违反角色一致性。
7. **`MAESTRO_QUISPE_NODDED` 是隐性标记** —— **不直接改数值**——**改语气**。**严守**。
