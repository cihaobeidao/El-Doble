## 一、整局节奏一览

```
                    替身在位 75 天
        ─────────────────────────────────────────
Acto    │ 0 │  I  │  II   │  III  │   IV   │   V   │  VI   │ End │
天数    │ 0 │1-7  │ 8-18  │19-32  │33-47   │48-60  │61-73  │74-75│
玩家时长│15m│30m  │35m    │40m    │40m     │40m    │45m    │25m  │
强制    │序幕│就职 │国宴   │抗议   │圣灵接洽 │选边站 │被识破 │收场 │
        │  │    │       │       │ + 联姻 │       │       │     │
派系焦点│Vega│老派│十二姓氏│街头   │圣灵集团│双使馆 │多方反扑│结局│
─────────────────────────────────────────────────────────────────
```

**叙事弧线设计**：

- **Acto 0 序幕**：危机萌发 — 玩家被强制接受身份。**没有真选择**。
- **Acto I-III**：建立 — 玩家熟悉系统、开始与各派系建立关系、第一次"伪装度"危机。
- **Acto IV**：拐点 — 圣灵集团送来魔鬼合同，寡头联姻线推开。**所有派系开始"真实地"压榨玩家**。
- **Acto V**：选边 — 双国博弈到达顶峰，玩家无法再做中间派。
- **Acto VI**：解构 — 至少一方已经识破，所有伏笔开始爆。
- **终章**：结算 — 75 天倒计时到。玩家面对国家与自己的双重命运。

## 三、主线 vs 暗线

### 主线（玩家直接面对的）

A. **"撑下去 75 天"** — 国家不能崩、你不能被识破、Cordura 不能归零
B. **派系周旋** — 五大派系 + 双使馆双子轴 + 街头街区
C. **资源管理** — 时辰 / Tesoro / 派系态度 / Sospecha

### 暗线（多周目逐步浮现）

A. **真总统的下落** — 4 张卷宗（A/B/C/D）
B. **Vega 的真实计划** — 他在用替身做什么？
C. **Mariana 为什么不揭穿** — 她有什么自己的目的？
D. **Susana 与你的过去** — 她为什么对你"过分熟悉"？
E. **Saulito 父亲是谁** — 多周目挖出的家族秘密
F. **第五真相 *La Verdad Compuesta*** — 集齐 4 卷宗后浮现的整体真相（详见 99_秘密文档.md）

## 五、阅读顺序建议

### 给设计师 / 主笔

按 0 → 9 顺序读。每个文件独立可读，但相互之间有 cross-ref，建议第一遍按顺序，第二遍跳读关心的章节。

### 给写作者（具体写事件文本）

1. 先读 [01_关键标记索引.md](01_关键标记索引.md) — 你写的事件可能要消费或新增标记
2. 读你负责的 Acto 文件
3. 读前一个 Acto 文件（理解上文）
4. 读 06_主题与调性.md + 07_风格样本.md
5. 写

### 给制作人 / 测试

读 [00_总览索引.md](00_总览索引.md)（本文件）+ [09_终章与结局.md](09_终章与结局.md) — 这两份足够掌握全局走向。
