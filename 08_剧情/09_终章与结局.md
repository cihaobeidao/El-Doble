# 08·09 · 终章 *El Pueblo Mira* / *El Telón Cae* + 全部结局

> **戏剧问题**：你是替身、是总统、是逃犯、还是死人？
> **剧情天数**：第 74–75 天
> **玩家时长**：约 25 分钟（剧情过场为主，主要选项已在第六幕选定）
> **核心**：本作所有结局的进入条件 + 一周目 15 种 + 多周目 4 种 + True Ending *Las Veladas*。

---

## 一、终章节奏

终章不再有"时辰系统"——剧情按 cutscene 模式推进。玩家可触发的最后选择仅在两个关键节点：

```
第 74 天 ─ 早上 ─ 最后一次内阁会议（强制 cutscene）
                │
                ↓
            晚上 ─ 私人时间（视前置选项的小决定）
                │
                ↓
第 75 天 ─ 黎明 ─ 命运决定时刻 ★（最后一次选择）
                │
                ↓
              结局 cutscene
```

## 二、终章关键事件

### END_P_001 · 第 74 天 — 最后一次内阁会议

**时刻**：第 74 天 09:00
**类型**：Público / 强制（cutscene）
**时辰**：n/a

**剧情骨架**：

- 内阁会议表面议程是"季度回顾"。Saulito 坐在主席位置——但他的"真实身份"已经在房间里几乎所有人之间被默契地知道。
- Gabriela 第一次回到内阁会议（自序幕以来）。她坐在 Vega 旁边。**她是真正的主席**——这一刻玩家应该开始隐隐感到。
- 各派系代表轮流发言。每段发言都是对玩家累积选择的回响：
  - Doña Isabel 谈"国家长期繁荣"——视联姻路线变化
  - Cardenal Soto 谈"良心"——视街头标记
  - Don Federico 谈"经济信心"——视铜矿议案
  - Whitfield / Aldana 各谈一段（视选边路线）
- 玩家此时不发言。**他在听。他知道他即将谢幕。**

**衔接**：会议结束 → 进入第 74 天晚上。

### END_C_001 · 第 75 天黎明 — 命运决定时刻

**时刻**：第 75 天 06:00
**类型**：Crisis / 强制
**时辰**：n/a

**剧情骨架**：

- Saulito 在卧室醒来。他没睡好。他走到窗边——天还没亮，但 La Casa Cobre 外面已经有人在等。
- Susana 在化妆室门口等他。她没拿化妆笔——她拿着 Saulito 的旧外套（凌晨被带走那晚穿的那件）。
- "Patrón——Saúl——三个月到了。该走了。"

**最后选择**（基于第六幕末选定的路线 → 此处是路线内部的最后微调）：

| 路线 | 最后选项 |
|---|---|
| A 撤幕路线 | A1 跟 Susana 走 / A2 拒绝走，向 Vega 表态愿意"留作过渡" |
| B 广场路线 | B1 在直播上明白说出 / B2 用诗意暗示 / B3 让 Cardenal Soto 来揭露 |
| C 演出继续路线 | C1 与 Gabriela 达成"长期协议"继续做总统 / C2 接受被寡头家族正式收编 |
| D 半揭露路线 | D1 在演讲中"暗示"——具体词句视前置标记 |
| E True Ending 路线（多周目） | 详见 §四 |

## 三、一周目 15 种结局（详细判定）

按 **国家命运 × 替身命运** 二维分布。每个结局包含：进入条件、过场内容、配乐与画面提示。

### 国家稳定（Economía/Orden 都不太负 + Legitimidad 不崩）

#### 1. **"撤幕"** *Telón* — 替身存活离开

- **进入**：路线 A + 整局派系态度无任何"敌对"+ Cordura ≥ 30
- **过场**：第 76 天清晨。一辆没有车牌的雪佛兰把 Saulito、Filomena、Lucía 送到边境。Susana 没去。Saulito 一直没有回头。
- **配乐**：拉美 bolero，钢琴独奏
- **画面提示**：最后一幕 — Las Veladas 的清晨，Mariposa 还住在那条街，少年继续养着她

#### 2. **"替罪羊"** *Chivo Expiatorio* — 替身被识破处决

- **进入**：路线 A 失败（`BURNED_ESTELA_CHANNEL` 或派系强烈敌对）
- **过场**：Saulito 在边境被截下。他被带回。一周后他在监狱里"心脏病发作"。报纸上没登。
- **画面**：他母亲在 Veranillo 收到一个匿名包裹——里面是他那块 Seiko 表

#### 3. **"新总统先生"** *Presidente Nuevo* — 替身彻底变成总统

- **进入**：路线 C + Verosimilitud ≥ 70 + Doce 同盟 + 街头 ≥ 试探合作
- **过场**：Saulito 在第 75 天的全国讲话中**用 Saulito 自己的语气**讲话——但全国都以为这是"奇迹般康复的总统"。Gabriela 在他身后微笑。Vega 看着他——眼神是混合骄傲与悲哀的。
- **画面**：黑色乌托邦——国家短期"复兴"，但所有真相被永久封存

#### 4. **"酒馆"** *Cantina* — 替身死于私人原因

- **进入**：Cordura 长期低 + 没触发 *La Locura* + 路线 A 失败之外的其他不幸组合
- **过场**：Saulito 在卧室独自喝光一瓶 Don Sebastián。心脏停了。Susana 第二天发现他。葬礼办成"总统的葬礼"。
- **画面**：Filomena 在乡下听广播说"总统逝世"——她叹了口气，没说话。她不知道"总统"是儿子。

### 国家陷入新内战（Orden < −60 或 Vega 政变成功 + 老派反扑）

#### 5. **"逃出去"** *Fuga* — 替身存活离开

- **进入**：路线 A + 政变爆发 + `KEPT_SOUTHERN_INSURANCE`
- **过场**：内战爆发当晚 Vasconia 直升机降落 La Casa Cobre 屋顶。Saulito 与 Susana 一起被接走。Vega 在屋顶向他们挥手——他选择留下。
- **画面**：直升机起飞，下面是 La Casa Cobre 起火的画面

#### 6. **"公审"** *Tribunal* — 替身被识破处决

- **进入**：政变成功后 Vega 失败 + 老派军方接管 + 玩家被识破
- **过场**：Saulito 在临时军事法庭被审判。他没有辩护。他在最后陈述中说："我母亲不知道。请你们不要去找她。"

#### 7. **"战时强人"** *Caudillo* — 替身彻底变成总统

- **进入**：政变 + 玩家选 *La Locura* B（咽回去）+ Vega 隐性轴极高
- **过场**：Saulito 与 Vega 联手镇压老派。他在国家电视台宣布"过渡军事委员会"。他走上历史最不堪的位置。
- **画面**：几年后 — 报纸上他的照片越来越像真总统。**他的眼神是冷的**。

#### 8. **"尸体"** *El Cuerpo* — 替身死于私人原因

- **进入**：政变 + Cordura 0 + *La Locura* A 在不利场所触发
- **过场**：Saulito 在内阁会议上喊出真相——老派军官当场开枪。Susana 在他身边把他扶住。
- **画面**：Susana 坐在血泊里，不哭。她从口袋里拿出那张 1979 年的照片，放在他胸口

### 国家被外部接管（Legitimidad < −50 + 一国深度介入）

#### 9. **"吊船"** *El Telón Foráneo* — 替身存活离开

- **进入**：路线 A + `SIDED_NORTH` 或 `SIDED_SOUTH` + 国家被该国"保护"
- **过场**：Saulito 被对应大国接走。维拉杜里亚成为该国的"事实保护国"。
- **画面**：Saulito 在某个北方城市的小公寓里——Lucía 与 Filomena 在隔壁房间。他坐在窗边看雨。

#### 10. **"样板间"** *Vitrina* — 替身被识破处决

- **进入**：北方深度介入 + 玩家被使馆"主动揭穿"作为政治筹码
- **过场**：北方公开"维拉杜里亚替身丑闻"。Saulito 被"自杀"或"流亡"。
- **画面**：北方电视台用这个故事拍了一部纪录片

#### 11. **"傀儡"** *Marioneta* — 替身彻底变成总统

- **进入**：路线 C + `SIDED_NORTH` 或 `SIDED_SOUTH` 同盟 + 玩家持续做"听话总统"
- **过场**：Saulito 继续做总统。每个决定都先与某使馆电话沟通。
- **画面**：他在椭圆办公室里接电话——电话另一头说什么我们不知道。他点头。

#### 12. **"纪录片"** *Documental* — 替身死于私人原因

- **进入**：外部接管中 + Saulito 因压力突发死亡
- **过场**：多年后某外国导演拍了一部"维拉杜里亚替身"的纪录片。Saulito 被还原为一个名字、一段录音、一张照片。
- **画面**：纪录片片尾——"我们不知道他的母亲是否曾听过这段录音。"

### Cordura 0 → *La Locura* A 触发的特殊变体

#### 13. **"El Discurso Loco"** *疯子演说* — 在内阁中说出真相

- **进入**：*La Locura* 选 A + 在内阁
- **过场**：Saulito 站起来。他说完整段。所有人脸色变化。Gabriela 第一次失态。Vega 把手按在枪上。**Susana 是房间里唯一镇定的人**——她已经准备好这一刻。

#### 14. **"广场"变体** — 在街头说出真相

- **进入**：*La Locura* 选 A + 在公开活动
- **过场**：广场上的人群第一刻沉默。第二刻有人说"他是真的"。第三刻*Los Mocosos* 开始拍视频。**民意立刻分裂**——有人想保护他，有人想吊死他。结局取决于 Cardenal Soto 隐性轴。

#### 15. **"庇护"** *Asilo* — 在使馆茶会说出真相

- **进入**：*La Locura* 选 A + 在使馆
- **过场**：Whitfield 或 Aldana（视场所）面无表情地把电话拿起来。三十分钟后专机起飞。Saulito 被带走 — 但他不知道是被保护还是被囚禁。

## 四、多周目 4 种最终结局

只有玩家在多周目中达成特定条件后才能解锁。

#### MC1. **"Plaza"** *广场*

- **进入**：找到真总统下落（任一卷宗完整）+ 公开真相（路线 B）+ 街头与教会 ≥ "信任" + 第六幕末玩家亲赴广场
- **过场**：第 75 天清晨。Saulito 在 Plaza Bolívar 中央。他没穿西装——他穿了序幕那晚的旧夹克。他对人群说话——**用 Saulito 自己的语气**。Cardenal Soto 在他身边。Joel 在他身边。Mariana 远远地站在屋檐下看。
- **画面**：他说完之后人群的反应是混合的——有人哭、有人怒、有人困惑。**这不是胜利。这是开始**。

#### MC2. **"Espejo"** *镜中镜*

- **进入**：解锁替身 V（真总统本人回归）+ 真总统选择"不夺权"
- **过场**：真总统 Augusto Báez Marín 出现在某次内阁。他与 Saulito 对视。**两人都不说话**。最后真总统说："我需要您再演一段时间。"
- **画面**：真总统消失。Saulito 继续演。这次他不是被推上去的——这次是他选择留下。

#### MC3. **"Vela Apagada"** *蜡烛熄灭*

- **进入**：所有真相全部揭露 + Cordura 在多次危机后归零 + 路线 A 失败
- **过场**：Saulito 在第 75 天的某个偏僻角落自杀。Vega 找到他。Susana 找到他。Gabriela 没有去。
- **画面**：那支蜡烛——维拉杜里亚——熄灭了。**不是被风吹熄，是被烧到底**。

#### MC4. **"La Función Continúa"** *演出继续*

- **进入**：玩家在多周目知道完整真相后 + 选择路线 C + 接受寡头收编 + 维持谎言
- **过场**：Saulito 知道 1998 年发生过什么。他知道 Eladio 是他父亲。他知道 Gabriela 是真主谋。**他选择什么都不做**。
- **画面**：第 75 天晚上他坐在 La Casa Cobre 的卧室。他给母亲打电话。他说"妈，我下个月回来看您。"他知道他不会回去。**最讽刺意味的"happy ending"**。

## 五、True Ending — *Las Veladas*

> **进入条件**（极度严格，本作的真结局）：

1. 多周目集齐 A/B/C/D 四张下落卷宗 → `DOSSIER_FIFTH_AVAILABLE`
2. 已通过 Susana 路线获得 `KNOWS_SAUL_FATHER_WAS_DOUBLE`
3. 已通过深度路线获得 `KNOWS_GABRIELA_PUPPETMASTER`
4. 在第六幕某场关键事件中**主动说出**真相的某个核心片段（标记 `LA_VERDAD_SPOKEN`）
5. 第六幕末选项 E（"找到真相"路线）

**过场内容**：

- 第 75 天黎明。Saulito 知道一切。他不再是"替身"——他知道自己是替身的儿子；他知道整个国家从 1998 年起就没有真总统；他知道 Gabriela 与 Susana 是真正的玩家。
- 他不去广场，不接受庇护，不留在 La Casa Cobre。
- 他穿上序幕那晚的旧夹克。他走出 La Casa Cobre 的后门——没有人拦他。
- 他穿过夜色，走回 Las Veladas。
- 他在自己的旧公寓楼下站了一会儿。少年牵着 Mariposa 出门倒垃圾——Mariposa 立刻跑过来，趴在他脚边。
- 他蹲下，摸她的头。
- 镜头拉远——他蹲在那条街上，那条他长大的街上。Doña Pilar 的小卖部还亮着，电视开着、没声音。
- **他不是替身，不是总统，不是逃犯，不是死人——他是一个回到家的中年男人**。

**最后画面**：

- 黎明的光在 Las Veladas 的屋顶上升起。
- 文字浮现：

> *La República Federal de Veladuria 在 X 月 Y 日继续。*
> *没有人知道真总统的下落。没有人知道替身去了哪里。*
> *Las Veladas 的人开始上班。一个孩子在街角踢球。一只狗追着他跑。*
>
> *这只狗的左耳缺了一块。*

**配乐**：拉美 bolero，加一段无歌词的女声哼唱。

> **设计意图**：True Ending 不是"胜利"，是 **"接纳"**。Saulito 没有改变国家，没有揭露真相，没有救任何人——但他**走回了自己的家**。这是本作对所有政治剧的最终回答：在一个体制不让你做"对的事"的世界里，"做回你自己"已经是奇迹。

## 七、结局后的"档案室"流程

详见 [05_多周目机制.md](../05_多周目机制.md) §一。结局后玩家自动进入 *El Archivo*：

- 看本周目的关键事件回顾
- 看本周目获得的关键标记
- 看本周目错过 / 错答的事项
- 看下落卷宗进度
- 解锁的新替身
- 解锁的"金色档案选项"

然后玩家可以选择"开始新周目"——这次起始数值会被前周目轻微调整，金色选项可用，新替身可选。
