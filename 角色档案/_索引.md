# 角色档案 · 总索引

> 这个文件夹是**所有角色的工作卡**——为写事件、写对白、做派系反应时**先翻这里**而存在。
> 真相依据顺序：`99_秘密文档.md` > `01_*–05_*` 与 `08_剧情/*` > 本文件夹。
> 本文件夹是**汇编与提示**，不是设定的源头。如发现冲突，**以原始设定文档为准**，并修正本文件夹。

---

## 一、阅读规则

1. **每张角色卡都有四块**：身份信息 / 声音卡 / 重大事件 / 写作红线。
2. **声音卡是关键**——写对白前先读对应人物的声音卡。三种声音（场景叙述者、Saulito 内心独白、新闻广播）的总规则在 `06_主题与调性.md`，本文件夹只补"这个角色具体怎么说"。
3. **重大事件按 Acto 排列**。具体场景骨架仍以 `08_剧情/*` 为准。
4. **写作红线是为了不写错**——凡是出现"易被 AI 误写"的位置，就在红线里点名。
5. **首周目隐藏的剧情不放进非 99 子目录**。任何只有多周目玩家应当看到的真相，请只在 `99_秘密角色/` 下出现。

## 四、跨角色提示（写作时常用）

- **三种声音的角色归属**：场景叙述者覆盖大部分角色；**Saulito 内心独白**只属于 Saulito；新闻广播覆盖 Catalina 的电台、《太阳日报》、Radio del Espíritu。其余角色出现时一律是场景叙述者笔下的对白。
- **不同人称**：Susana、Aurora、Mariana 三人**会在某个时刻喊 Saulito 的真名**——这是触发点，不是日常用法。其余角色**永远只叫他"Patrón / Presidente / Augusto / 总统先生"**。
- **数字要有体感**：每张卡里的数字（年龄、薪水、汇款）都按 `01_世界观与派系小传.md` 物价表对照过；写新场景时把数字翻译成"几颗鸡蛋 / 几天工资"。
- **置信度标签** ✓ / ◇ / ✗（来自 *El Expediente Báez*）：本文件夹的"事实"原则上为 ✓；如标注 ◇ 表示有多周目分歧；标注 ✗ 表示首周目玩家不该看到。

## 五、维护纪律

- 任何对设定文档（`01_*` ~ `99_*`）的修改，**先去更新原文档**，**再回头把对应角色卡同步**，最后在对话里输出 changelog。
- 本文件夹**不保留版本历史**——历史用 git log 追。
- 新增角色 → 同时添加到本索引"角色总名单"段。
- 删除角色 → 不要直接 rm；先在卡顶部加一行"已废弃于 YYYY-MM-DD，原因：……"，等下一次清理时统一移除。
