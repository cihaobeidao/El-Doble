# 美术风格指南 · *La Estética del Cobre*（铜的美学）

> **这是所有 prompt 的根**。任何写出来的具体 prompt 都要在头部引用本指南的"风格区段"。
> 改风格 = 改这一份 = 影响所有图。**慎动**。

---

## 一、视觉调性（一句话）

**1970s–1980s 拉美政治纪实摄影的颗粒与重力 + 油画式的人脸厚度 + Marquez 式的金色蒙尘**。

不是干净。不是漂亮。**是让人在画面里闻到柴火、烟草、汗、铜锈的潮气**。

---

## 二、参考片单 / 摄影集（写 prompt 时可直接引用）

### 电影
- **Roma (2018, Cuarón)** — 灰调、城市生活的厚度、白制服与黑佣人、墨西哥 70 年代质感。**色彩与构图首要参考**。
- **The Death of Stalin (2017)** — 室内政治会议的压迫感、宫廷的腐朽、黑色幽默的视觉重量。**国宴 / 内阁会议参考**。
- **La Llorona (2019, Bustamante)** — 危地马拉政治创伤片，热带光、湿气、独裁者家族。**Doña Isabel 家族场景参考**。
- **Brazil (1985, Gilliam)** — 官僚的荒诞 + 城市迷宫。**走廊 / 公文场景参考**。
- **Tropical Malady (2004)** + **Memoria (2021)** — 热带的边缘感、潮湿的存在主义。**山区 / 雨林场景参考**。
- **Narcos** — 贩毒集团的物质细节（白衣、十字、金链）。**El Padrino 场景参考**。

### 摄影师
- **Sebastião Salgado** — 黑白颗粒、人的尊严与重量。**Las Veladas 街区与原住民人物的参考**。
- **Graciela Iturbide** — 墨西哥黑白人像，神秘 + 在场。**Aurora、Maestro Quispe 参考**。
- **Manuel Álvarez Bravo** — 墨西哥日常的诗意荒诞。**街头镜头参考**。
- **Susan Meiselas** — *Nicaragua* 系列，桑迪诺时期纪实彩色。**Mocosos 抗议、青年场景参考**。

### 绘画 / 书封
- **Marquez 1980 年代西语原版书封**（钢笔 + 水彩晕染） — **片头 / 片尾静帧美学参考**。
- **Fernando Botero**（**仅参考其讽刺、不参考其圆胖造型**）—— Joaquín 车祸丑闻的剪报美术可借鉴。
- **Diego Rivera 壁画**（仅参考其原住民题材的色彩与构图，**不**参考其英雄主义化倾向）。

## 五、镜头语言（写 prompt 时的"摄影术语区"）

```
媒介:    shot on 35mm film, Kodak Portra 400 (warm) / Kodak Tri-X 400 (B&W)
         / Fuji 400H (cool soft) — 按场景选
质感:    medium grain, slight halation, natural skin texture,
         visible pores, no plastic skin
镜头:    50mm prime (人像 default) / 35mm（中景 + 环境）/ 85mm（特写）
焦深:    shallow but not bokeh-extreme — face sharp, background organically soft
        (NOT cinematic-blur look from cheap AI)
角度:    eye level for dignity / slight low for power figures /
         slight high for vulnerability — 按角色当下处境选
后期:    minimal grading, slightly desaturated reds and greens,
         lifted shadows with green-magenta tint,
         film grain over digital cleanliness
```

## 九、角色一致性工作流（**写到痛处**）

> 这是整个项目美术能不能落地的关键。AI 生图最大的坑是"同一个角色，每张脸不一样"。下面这套流程是**强制执行**。

### Step 1 · 生成 canonical 头像
- 用 `01_角色/_角色生成清单.md` 里每个角色的"canonical headshot prompt"。
- 出 4 张候选，**Keyan 选一张**作为 canonical（标记为 `<角色ID>_bust_v01_CANONICAL.png`）。
- 这张图就是这个角色之后所有图的"锚"。

### Step 2 · 用 canonical 作为 cref 跑变体
- 任何"该角色在其他姿态 / 场景"的图——**必须**以 canonical 作为 cref / IPAdapter / FaceID 输入。
- prompt 头部仍引用 `_风格指南.md` 的风格区段。

### Step 3 · 跑场景 + 角色合成
- 复杂场景（多人 + 环境）建议**分两步**:
  1. 先跑空场景（参 `02_场景/_场景生成清单.md` 的 `empty` 版本）。
  2. 再用 inpaint / regional prompt 把角色"放进"场景，用 canonical 作 cref。
- **不要一次生成全场景 + 多人**——AI 会把脸混。

### Step 4 · 命名 + 入库
- 按 `01_角色/images/README.md` / `02_场景/images/README.md` 的命名规则保存。
- 在 `_总表.md` 把状态从"未生成"挪到"已 v01"。
