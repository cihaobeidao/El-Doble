# Susana Quintero

> 她不让你笑也不让你哭。她让你"是他"。
> 然后她在你即将出错前轻轻按一下你的肩膀。

## 一、身份信息

| | |
|---|---|
| 全名 | Susana Mercedes Quintero Lobo |
| 年龄 | 44 |
| 出身 | 首都贫民窟 *Las Veladas* —— **与 Saulito 同一区**。她从未提过这件事。 |
| 入行 | 16 岁起在剧院化妆 → 18 岁入 La Casa Cobre 协助首席化妆师 → 27 岁起独自掌印 |
| 当前职务 | 总统首席化妆师（已 17 年） |
| 公开身份 | **化妆师 / 仪表顾问 / 公共形象团队主管** |
| 真实身份 | **Saulito 在 Casa Cobre 里唯一全程知道他是替身的人** |
| 家庭 | 一位女儿在北方留学（19 岁，主修生物医学）；丈夫早逝 |
| 标志 | 总穿不显眼的灰色外套；指甲剪得极短（化妆需要）；左手中指有一枚银戒（亡夫的） |
| 政治效忠 | 表面：青年军官派 Mateo Vega。**真相参 99**。 |

## 二、声音卡

### 总规则
- Susana 说话**极少**。她大部分时间在"操作"——化妆、调整、手势、轻按。
- 她**几乎从不在公开场合**与替身对话——她的工作是**让替身说话**。
- 私下，她的语气像一位**"职业到无情的老师"**——不冷漠，但**绝不浪费一句话**。

### 私下对话
```
体裁：短句。指令式与反问式。**几乎不用感叹号**。
节奏：快。她**每秒钟都在评估**。
词汇：化妆术语 + 表演术语 + 偶尔贫民窟口音漏出来（这是她身份的暗门）。
口头禅：
  - "Patrón。"（她叫所有替身这个——这是她保持距离的方式）
  - "*A respirar.*"（"呼吸。"——她在 Saulito 即将崩溃前会这样说）
  - "*Otra vez. Más despacio.*"（"再来一次。慢一点。"——化妆排练时）
  - "*Eso no.*"（"那不行。"——她否决某个表情时）
```

### 关键时刻：**她叫真名的那一刻**

**第六幕中段**，化妆室里——她**第一次叫 Saulito 的真名**。
```
情境设计：
  - 周遭无第三方（这是必须的）。
  - 她在贴假发，停下，看着他在镜子里的眼睛。
  - 她说："Saulito。"
  - 仅此一句。然后继续工作。
玩家此时的回应决定 Susana 后期是否帮你：
  - 玩家保持沉默 → 标记 SUSANA_HEARD_REAL_NAME_AND_DID_NOTHING
  - 玩家点头一次 → 标记 SUSANA_BOND_FORMED
  - 玩家反问"您在叫谁?" → 标记 SUSANA_REJECTED
  - 玩家说"你认识我父亲，对吗？" → 触发 *El Cuestionario Final*（True Ending Route）
```

> 注：这是**Saulito 路线唯一从设计上明确指向 True Ending 的事件**。**写作时不要在任何 NPC 的台词里提示玩家这条路径**——必须由玩家自己想到。

### 标志性动作

- **轻按 Saulito 肩膀**（即将出错前）—— 一次轻短的按压，无声讯号。
- **整理 Saulito 衣领**（公开场合 Saulito 路线）—— 她以"化妆师"身份做这件事，但她做的方式与 Gabi 整领带**视觉完全不同**：Susana 是**贴近镜子**的、**戴白手套**的；Gabi 是**面对面**的、**赤手**的。
- **不看镜子里的自己**。化妆室里只有一面镜子，她从不照自己。

### 不会说的话

- 任何关于自己女儿在北方的细节——**这是她可被买通的死穴**（Helena Whitfield 在 Acto III 末知道她有女儿）。
- 任何关于"1998 年"的事（这是她真相中的时间锚——参 99）。
- 任何对 Gabriela 的评价——她**从不评价她**。这本身就是**她效忠 Gabriela 的最强信号**。

## 三、与替身位置的关系

### 表面关系（首周目可见）
- 她负责化妆、贴假发、调整语调、检查表情。
- 职业、克制、不显示情感。
- 但她**会在 Saulito 即将出错前轻轻按一下他的肩膀**——这是她的私人语言。
- 她**偶尔在档案上加新批注**：✓ 白色升级新事实、◇ 黄色澄清旧矛盾——这是 Saulito 与她的隐性关系渠道。

### 危险性（首周目层面）
- **中等**。她效忠 Vega，但她也有自己的女儿（北方留学）——她可以被买通。
- 她对 Saulito 的态度从"职业操作"逐渐转向"看着一个孩子的母亲不会做的事"——但**这条转变 Saulito 不会被告知**。

### 她真正在意什么（多周目逐步揭示）
- 她**早就认识 Saulito**——她见过他作为婴儿（参 99）。
- 她不是被 Vega 招募的——**她是这件事的合谋者**（参 99）。
- 她对 Saulito 的"温柔"**不是后期产生**的——是从她见到他第一秒就有的。

## 四、重大事件

### 序幕
- **PROL_C_001**：黑色雪佛兰副驾。**她在车上不说话**。
- **PROL_AR_001**：地下化妆室——她递给 Saulito *Tarjetas Esenciales*；考核题是**她出的**。她**故意没在题里加任何"父亲"相关的题**——这是她的第一次"沉默地保护"。
- **PROL_P_001 之前**：她为 Saulito 第一次化妆。完成时她说："Patrón。准备好了。" 这是她对他的**第一次正式称呼**。

### Acto I
- 第一次内阁会议——她在屏风后听。Saulito 即将出错时她**不能从屏风后干预**——她只能祈祷他自己解。**这是她在剧情中第一次表现出真情绪**：她在屏风后摸着自己手腕的银戒。
- 国宴前她**给 Saulito 重做了三次妆**——理由是"光线不对"。**真实理由是她在调整一种让他看起来与他父亲更像、与 Báez 更不像的微妙比例**——这条**首周目玩家完全感知不到**。

### Acto II
- 她**第一次在档案上加批注**：黄色 ◇——澄清"总统的祈祷习惯：每天 06:50 私人祈祷 5 分钟（来源：私人秘书证词）"。
- 国宴前 Mariana 第一次出场——Susana 给 Saulito 化妆时停了 1 秒。**她在这一秒里第一次担心了**——Mariana 的眼睛太尖。

### Acto III
- **Whitfield 在某次茶会发现 Susana 有一个在北方的女儿**——茶会后 Whitfield 派人把一封私人信件转交给那位女儿。**Susana 路线的 Cordura 危机第一次触发**。
- Saulito 在山区拜访 Aurora——Susana**没去**（化妆师不被允许进山区祭典）。她在 La Casa Cobre 等他回来时**她第一次坐在 Augusto 平时坐的椅子上**——一秒。然后站起来。

### Acto IV
- 联姻提议出来——Susana 在化妆室对 Saulito 说："Patrón，今晚的领结需要您自己决定。" 这是她**第一次把决策权交回给 Saulito**——她在用一个表面琐碎的细节给他暗示"这件事您必须自己想清楚"。

### Acto V
- 两国大使同时露面那场国宴——**Susana 在场**（以化妆师身份）。她**第一次让 Saulito 喝了一小杯酒**（违反所有她平时的规矩）。她在事后说："今晚不一样。"

### Acto VI（决定性时刻）
- **她叫真名**（详见声音卡"关键时刻"）。
- 这之后剧情进入 *El Cuestionario Final* 的引导轨——参 `99_秘密角色/_阅读警告.md`。

## 五、写作红线

1. **她极少说话**——写她的场景时**默认状态是无对白**。她有对白时**每一句都要有重量**。
2. **她从不喊"Saúl" / "Saulito"**——除了第六幕那一次。**全游戏只允许一次**。任何在这之前出现这个发音的位置必须改写。
3. **她不评价 Gabriela**——任何让 Susana 评价 Gabi 的台词都写错了。
4. **她的女儿在北方** ◇ 严格保密：首周目玩家**只能在 Whitfield 茶会后才能间接得知**。**禁止**在更早事件中提及她有女儿。
5. **她不流泪**。**全游戏她最多湿眼眶一次**——在第六幕"叫真名"那场。
6. **她的来自 Las Veladas 出身**：**整个首周目不揭示**。**只有多周目玩家**通过特定支线（Eladio Tobar 档案、街区考古纪录片支线等）才能拼出。
7. **化妆术语要真**：写她工作时要查实际化妆师术语（hot-set powder / lip-line / contour / cake foundation 等）。**这是她身份的真实质感来源**，不能糊弄。
8. **她整理 Saulito 衣领的方式 ≠ Gabriela 整领带**：参声音卡"标志性动作"——这两个动作必须视觉上明确区分。
